按**知识点**分题：上方是知识点名称与思想，下方是题目链接（可点击直达）及该题核心做法、差异点备注。

## 思维转化 / 问题转化

**思想**：不直接做原问题，先把题意或模型转化成另一类已会问题。常见手段：点权化边权、断环成链、凑子集、区间覆盖、传送/图论抽象等。

- [[算法学习/最短路算法/出差-真题]]  
	**点权化边权**：城市隔离时间合并到「指向该城市的边权」上，终点 C_N=0，然后跑 Dijkstra。 差异：带点权最短路→纯边权最短路。
- [[算法学习/动态规划/区间DP/课上小游戏]]  
	**断环成链**：环形数组复制一份接在后面变成长度 2n 的线性序列，在 2n 上做区间 DP，答案取所有长度为 n 的区间的最优值。 差异：环形→线性区间 DP。
- [[算法学习/动态规划/背包问题/01背包/盗墓分赃2]]  
	**分赃→能否凑出 Sum/2**：两人均分等价于从物品中选若干件，重量和是否恰好为总重一半，即 01 背包可行性。 差异：最优分配→定容量可行性。
- [[算法学习/动态规划/背包问题/01背包/1049. 最后一块石头的重量 II]]  
	**碰撞→差值**：每次撞相当于两堆做差，等价于把石头分成两堆使差最小，即 01 背包凑不超过 Sum/2 的最大和。 差异：过程模拟→子集和。
- [[算法学习/图论入门专练/建造国家-真题]]  
	**思维陷阱**：树和 k 个警察局，不建图不 BFS。结论：最多删 k-1 条边。 差异：答案只与 k 有关。
- [[算法学习/最小生成树算法/吊坠-真题]]  
	**环形 LCS→线性**：串复制成 s+s，滑动窗口取长度 L 子串与给定串 LCS 匹配；再 Kruskal。 差异：环形旋转→定长窗口 + MST。
- [[算法学习/动态规划/动态规划的基础/电影放映计划]]  
	**时间约束→完全背包**：「时长+空档 K」当物品重量，容量 M+K，转化为完全背包。 差异：带空档排片→标准完全背包。
- [[算法学习/动态规划/动态规划的基础/最快洗车时间]]  
	**分工→选数**：两边同时洗等价于选一部分和 S1≤S/2 尽量大，答案 S-S1。 差异：时间分配→子集和。
- [[算法学习/最小生成树算法/点亮须弥]]  
	**传送点→MST**：传送点点亮后可瞬移抽象成选边建图使连通，代价最小即 MST。 差异：游戏规则→图论 MST。
- [[算法学习/最短路算法/染色时间-真题]]  
	**触发顺序→最短路**：每个格子首次被染到的时刻即到该格的最短路（边权为对方格染色时间）。 差异：扩散过程→带权最短路。
- [[算法学习/数据结构/树形数据结构/森林的最大美丽值]]  
	**最大美丽值→二分判定**：二分 mid，判定是否能在 m 天内让所有树高都≥mid。 差异：最值→可行性判定。
- [[算法学习/字符串/契合匹配]]  
	**环形匹配→线性**：模式串拼成 S'+S'，循环移位变为在长串里找子串。 差异：环上匹配→KMP/字符串查找。
- [[算法学习/gcd与常见组合问题、计算几何基础/黑客-真题]]  
	**分母变化→分子 O(1)**：维护全局基准，用分子乘法表示分母变动。 差异：动态比例→固定基准+乘法修正。
- [[算法学习/动态规划/区间DP/艰苦打工的小蓝2]]  
	**思维转化**：区间选点/覆盖类条件转化为区间 DP 状态与转移。 差异：题意需先抽象成区间结构再 DP。

## 逆向思维

**思想**：从终点、结果或「最后一步」倒推，或反向跑 BFS/DP，常能少写状态、一次得到多源信息。

- [[算法学习/暴力解题方法与搜索问题/迷宫-真题]]  
	**从终点出发 BFS**：从终点反着 BFS 一次得到每个起点到终点的最短步数，再算期望。 差异：多源到一点→单源（终点）到多点。
- [[算法学习/动态规划/期望DP/小蓝玩游戏]]  
	**逆向 DP**：从 (n,m) 倒推到 (0,0) 的期望步数。 差异：正向难以写状态时，倒着推更自然。

## 贪心

**思想**：每步做当前看起来最优的决策（局部最优），并利用题目性质证明这样能得到全局最优。常配合排序、堆、双指针、差分等。

- [[算法学习/贪心与思维专练/水质检测-真题]]  
	**2×n 网格连通**：找 L、R 后线性扫，f0/f1 表示当前列站在上/下的最少新增数。 差异：线性 DP 与贪心结合，状态只有两行。
- [[算法学习/贪心与思维专练/最短距离-真题]]  
	**贪心选点/排序**：按题意选点或排序后贪心选，使某种距离和最小。
- [[算法学习/贪心与思维专练/最大数字-真题]]  
	**数位贪心/自定义排序**：高位尽量大，或按「拼接后字典序」排序再贪心选。 差异：比较规则是拼接结果而非数值。
- [[算法学习/贪心与思维专练/拼好数-真题]]  
	**频次数组 + 贪心**：用桶统计数字出现次数，按「省着用」策略拼数。
- [[算法学习/贪心与思维专练/回文数组-真题]]  
	**差分数组 + 双重打击**：D_i=a_i-a_{n-1-i}，目标把 D 变为 0；相邻同向可「双重打击」一次改两个。 差异：回文→差值数组上的贪心。
- [[算法学习/贪心与思维专练/画展布置-真题]]  
	**排序 + 贪心 + 滑动窗口**：按某一维排序后，用窗口或双指针在满足约束下选区间。
- [[算法学习/贪心与思维专练/训练士兵-真题]]  
	**代价与收益的贪心**：每次选「性价比」或「边际收益」最优的决策，可能用堆维护。
- [[算法学习/常见库函数与工具/原料采购-真题]]  
	**贪心 + 大根堆**：按价格或单位代价入堆，批量取用、扣减数量。
- [[算法学习/动态规划/背包问题/01背包/蓝桥课程抢购]]  
	**贪心 + 01 背包**：先按性价比等排序或选物品，再在容量限制下做 01 背包。
- [[算法学习/动态规划/区间DP/制作回文串]]  
	**增删等价**：插入与删除代价可统一，贪心简化区间 DP 转移。
- [[算法学习/贪心与思维专练/合并果子]]  
	**堆 / 哈夫曼**：每次选两堆最小的合并，用小根堆维护。 差异：经典贪心。
- [[算法学习/贪心与思维专练/抽奖-真题]]  
	**贪心 / 模拟**：按规则抽奖或分配，贪心选当前最优。
- [[算法学习/贪心与思维专练/电池分组-真题]]  
	**贪心分组**：按某种代价或容量贪心分组。
- [[算法学习/贪心与思维专练/书架还原-真题]]  
	**贪心 / 模拟**：按题意还原书架顺序或最小操作次数。
- [[算法学习/贪心与思维专练/劲舞团-真题]]  
	**模拟 + 贪心**：按节奏或规则模拟，贪心得分或判定。
- [[算法学习/贪心与思维专练/食堂-真题]]  
	**贪心 / 排队**：排队或窗口选择类贪心。
- [[算法学习/贪心与思维专练/最多次数-真题]]  
	**贪心 / 计数**：在约束下贪心使某操作次数最多或最少。
- [[算法学习/贪心与思维专练/红黑树-真题]]  
	**位运算 / 构造**：与红黑树性质或二进制构造相关。
- [[算法学习/贪心与思维专练/召唤数学精灵-真题]]  
	**数论 / 构造**：数学条件或公式下的构造与判定。
- [[算法学习/贪心与思维专练/2025图形-真题]]  
	**图形 / 模拟**：按规则画图或数图形，模拟或找规律。
- [[算法学习/贪心与思维专练/可分解的正整数-真题]]  
	**等差数列**：连续正整数和为定值，等差数列求和公式。 差异：数论+贪心。

## 线性 DP / 一次扫描

**思想**：状态与「位置/时间」强相关，从左到右（或单次遍历）递推，状态数有限，做到 O(n) 或 O(n·常数)。

- [[算法学习/贪心与思维专练/水质检测-真题]]  
	**2×n 网格**：f0/f1 到当前列上/下的最少新增，从左到右 O(n)。
- [[算法学习/动态规划/线性动态规划（Linear DP）]]  
	**线性 DP 总纲**：一维序列上的 DP、状态只依赖前若干位置。
- [[算法学习/动态规划/动态规划的基础/破损的楼梯]]  
	**楼梯跳格**：按台阶位置线性递推，能否到达或最少步数。
- [[算法学习/动态规划/动态规划的基础/电影放映计划]]  
	**时间轴选择**：转化后为完全背包，也可视为线性容量上的 DP。
- [[算法学习/动态规划/动态规划的基础/安全序列]]  
	**线性约束计数**：序列上满足「安全」条件，线性 DP 递推。
- [[算法学习/动态规划/动态规划的基础/蓝桥勇士]]  
	**LIS / 线性**：最长上升子序列或线性选数。
- [[算法学习/动态规划/最长上升子序列]]  
	**LIS 模板**：一维序列最长严格上升子序列，O(n log n) 二分。

## 二分答案 / 二分查找

**思想**：答案或某一维具有单调性时，二分该值，把「求最值」变成「判定」，写 check 函数即可。

- [[算法学习/贪心与思维专练/01串-真题]]  
	**数位统计 + 二分**：在满足条件的 01 串中二分第 k 小或统计数量。
- [[算法学习/二分解题法/管道-真题]]  
	**二分时间 + 区间合并**：二分 T，每个阀门覆盖一段区间，合并看是否覆盖 [1,len]。 差异：大范围时间→O(n) 判定。
- [[算法学习/前缀和与差分一日通/成绩统计-真题]]  
	**二分答案 + 排序滑动窗口**：二分最小方差等，check 时排序后取连续一段。
- [[算法学习/字符串/小蓝的神秘日记]]  
	**二分长度 + 字符串哈希**：二分最长重复子串长度，check 用滚动哈希。
- [[算法学习/字符串/幸运字符串]]  
	**二分前缀长度**：前缀长度有单调性，二分最大满足条件的长度。
- [[算法学习/二分解题法/大衣的旅行]]  
	**二分距离 d**：二分最小距离，check 某区域在 d 内能否容纳至少 K+1 人。
- [[算法学习/二分解题法/寻找黄金宝藏]]  
	**二分答案 + 判定转化**：二分价值/数量，序列转化为 0/1 或权重再判定。
- [[算法学习/数据结构/树形数据结构/森林的最大美丽值]]  
	**二分美丽值**：二分 mid，判定是否能在限定天数内让所有树高≥mid。
- [[算法学习/二分解题法/冶炼金属-真题]]  
	**二分 / 不等式**：根据冶炼记录二分转化率 V 的范围。
- [[算法学习/二分解题法/卡牌-真题]]  
	**二分 + 贪心**：二分答案，check 时贪心用牌。
- [[算法学习/二分解题法/最少刷题数-真题]]  
	**二分 / 排序**：二分刷题数或中位数，满足排名等约束。
- [[算法学习/二分解题法/肖恩的乘法表]]  
	**二分第 k 小**：在乘法表上二分第 k 小的数。
- [[算法学习/二分解题法/肖恩的n次根]]  
	**二分实数**：二分求 n 次根或满足条件的实数。
- [[算法学习/二分解题法/倒水-真题]]  
	**二分 / 模拟**：二分容量或次数，check 模拟倒水。

## DFS / 回溯 / 剪枝

**思想**：枚举决策树，用 DFS 递归，在「恢复现场」下回溯；通过约束剪枝或记忆化减少搜索量。

- [[算法学习/暴力解题方法与搜索问题/黑白棋填充-真题]]  
	**6×6 填 0/1**：每行每列 3 个 0、3 个 1，无连续三同色，行/列互不相同；DFS+剪枝。 差异：CSP 约束多，剪枝要细。
- [[算法学习/暴力解题方法与搜索问题/N皇后]]  
	**每行放一个皇后**：DFS 按行枚举列，用数组记列、两条斜线是否被占。
- [[算法学习/暴力解题方法与搜索问题/排列问题]]  
	**字典序排列**：DFS 枚举每一位填谁，注意恢复现场。
- [[算法学习/暴力解题方法与搜索问题/组合问题]]  
	**组合枚举**：控制 for 的起点，只枚举下标递增的组合。
- [[算法学习/暴力解题方法与搜索问题/最大数字-真题]]  
	**数位记忆化**：每位选 0~9 消耗 A/B，记忆化 (位置, 剩余 A, 剩余 B)。
- [[算法学习/搜索/串变换]]  
	**操作序列的 DFS**：每一步选一种操作执行，DFS 枚举操作排列。
- [[算法学习/搜索/黄金树]]  
	**树上的 DFS**：父节点传「黄金指数」给子节点，递归带父状态。
- [[算法学习/动态规划/数位DP/优雅的数字]]  
	**数位 DP**：DFS(i, 状态, limit) 填数位，[L,R] 用 count(R)-count(L-1)。
- [[算法学习/暴力解题方法与搜索问题/九宫幻方]]  
	**DFS 填幻方**：3×3 幻方填数，满足行列对角线和相等，判唯一解。
- [[算法学习/暴力解题方法与搜索问题/五子棋对弈]]  
	**DFS / 判定**：12 条胜线、黑白子数，DFS 或枚举判定。
- [[算法学习/暴力解题方法与搜索问题/扩散]]  
	**DFS/BFS 扩散**：格点扩散，确定边界范围后统计。
- [[算法学习/暴力解题方法与搜索问题/像素放置]]  
	**DFS 填格**：按约束在网格上填 0/1，DFS 回溯。
- [[真题文件/4-差分-前缀和-数据结构/399. 除法评估]]  
	**建图+DFS**：见题面。

## BFS / 最短路 / 扩散

**思想**：边权为 1 时 BFS 按层扩散，第一次到达即最短路；多源则多点入队；有时状态带「步数模 2k」等。

- [[算法学习/暴力解题方法与搜索问题/走迷宫]]  
	**网格四向 BFS**：从起点 BFS，第一次到终点即最短步数。
- [[算法学习/暴力解题方法与搜索问题/迷宫-真题]]  
	**从终点反着 BFS**：以终点为源点 BFS，各点距离即到终点的最短步数。
- [[算法学习/暴力解题方法与搜索问题/AB路线-真题]]  
	**带周期状态 BFS**：状态 (r, c, cur_k)，cur_k 为 2k 周期中的第几拍。
- [[算法学习/暴力解题方法与搜索问题/岛屿个数]]  
	**双重 BFS**：外海 8 连通 BFS，陆地 4 连通 BFS 数岛屿。 差异：海水 8 连通、陆地 4 连通。
- [[算法学习/最短路算法/星际旅行-真题]]  
	**全源 BFS + 前缀和**：对每个起点 BFS 得到到各距离的点数，前缀和 O(1) 回答。
- [[算法学习/搜索/小怂爱水洼]]  
	**Flood Fill**：遇到未访问的水格就 BFS 一整片水洼，计数。
- [[算法学习/动态规划/树形DP/卖树]]  
	**三遍 BFS**：BFS(1) 找一端 a，BFS(a) 找直径另一端 b，再 BFS 找平衡点。
- [[算法学习/搜索/混境之地2]]  
	**BFS/最短路**：网格或图上的最短路、可达性。
- [[算法学习/搜索/混境之地5]]  
	**BFS/最短路**：同上，变体。
- [[算法学习/搜索/仙境诅咒]]  
	**BFS/扩散**：诅咒扩散或连通块。
- [[真题文件/4-差分-前缀和-数据结构/1631. 最小体力消耗路径]]  
	**二分+BFS/最短路**：见题面。
- [[真题文件/8-搜索-BFS-图搜索/瞬移]]  
	**BFS/搜索**：瞬移或图搜索。

## 前缀和 / 差分 / 区间转化

**思想**：区间信息用「前缀和/前缀异或」表示成两端点差；或区间修改打差分，单点查变前缀和。

- [[算法学习/前缀和与差分一日通/异或和之和-位运算-真题]]  
	**前缀异或 + 按位贡献**：S[R]⊕S[L-1] 为区间异或和；拆位 (cnt0×cnt1)×2^k。 差异：区间异或和→两点异或+计数。
- [[算法学习/前缀和与差分一日通/子串简写-真题]]  
	**长度≥K 转化为下标**：固定右端点 j，左端点 i≤j-K+1，线性扫。
- [[算法学习/前缀和与差分一日通/商品库存管理-真题]]  
	**贡献转化**：「不执行操作 i 后库存为 0」转化为每个商品的贡献、是否唯一。
- [[算法学习/动态规划/数位DP/幸运年]]  
	**区间 [L,R] 转化**：count(R)-count(L-1)，数位 DP 求 count(x)。
