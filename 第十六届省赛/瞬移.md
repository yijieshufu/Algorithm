# 瞬移

### 题目分析

1. **场景模拟**：
    
    - 起点位置是 $1$，终点位置是 $L$。
        
    - 每次移动，你需要从给定的集合 $\{a_1, a_2, \dots, a_n\}$ 中选出两个数 $x$ 和 $y$（$x$ 和 $y$ 可以相同）。
        
    - 移动的距离是 $x + y$。
        
    - **位置更新公式**：如果当前位置是 $p$，移动后的新位置 $p'$ 为：
        
        $$p' = (p + x + y - 1) \pmod L + 1$$
        
    - 这个公式实际上是在 $1$ 到 $L$ 的环状空间内移动。目标是用最少的步数到达 $L$。
2. **核心难点**：

- **组合数**：从 $n$ 个数中选两个，看似有 $n^2$ 种组合。但实际上，我们只需要关心 $x+y$ 的结果模 $L$ 的值。
    
- **状态空间**：位置 $p$ 的取值范围是 $[1, L]$。由于 $L \leq 2000$，状态数很少，非常适合 BFS。
    

### 解题思路

1. **预处理可能的位移**：
    
    计算所有可能的 $(a_i + a_j) \pmod L$。由于我们最终关心的是对位置的影响，预先算出所有唯一的“步长”可以显著减少 BFS 过程中的遍历次数。
    
2. **BFS 过程**：
    
    - 使用一个队列存储当前到达的位置和步数。
        
    - 使用一个 `visited` 数组记录已经到达过的位置，防止死循环。
        
    - 从起点 $1$ 开始，每次尝试所有预处理出的“步长”，计算新位置。
        
    - 如果新位置等于 $L$，则返回当前步数。
    

```python
from collections import deque  
  
  
def solve():  
    # ---------------------------------------------------------  
    # 1. 数据读取与预处理  
    # ---------------------------------------------------------  
    try:  
        # 读取第一行：n (数组长度), l (出口位置/环的周长)  
        line1 = input().split()  
        if not line1: return  
        n, l = map(int, line1)  
  
        # 读取第二行：包含 n 个整数的数组 a        a = list(map(int, input().split()))  
    except EOFError:  
        return  
  
    # 预处理：我们每次移动的距离是 x + y，其中 x, y 都在数组 a 中  
    # 因为位置计算是模 L 的，所以我们只需要关心 (x + y) % L 的结果  
    possible_moves = set()  
  
    # 优化点：  
    # 先对 a[i] 取模并去重。因为如果 a[i] = 1 和 a[i] = 1 + L，  
    # 它们对移动结果的影响是一模一样的。  
    a_mod_unique = list(set(x % l for x in a))  
  
    # 双重循环组合出所有可能的移动步长 (x + y) % L    # 这里的复杂度最高为 O(L^2)，对于 L=2000 是安全的  
    num_unique = len(a_mod_unique)  
  
    for i in range(num_unique):  
        for j in range(i, num_unique):  
  
            # 计算两数之和对 L 取模的结果  
            move = (a_mod_unique[i] + a_mod_unique[j]) % l  
            possible_moves.add(move)  
  
    # ---------------------------------------------------------  
    # 2. 广度优先搜索 (BFS) 寻找最短路径  
    # ---------------------------------------------------------  
    # BFS 的特性决定了它第一次到达目标点时，所经过的路径一定是最短的  
  
    # 队列存储元组：(当前位置, 当前已移动的步数)  
    # 起始位置是 1，步数为 0    queue = deque([(1, 0)])  
  
    # visited 数组记录哪些位置已经走过，避免重复计算导致死循环  
    # 长度为 l + 1 是为了直接通过索引访问 1 到 L 的位置  
    visited = [False] * (l + 1)  
    visited[1] = True  
  
    while queue:  
        curr_p, steps = queue.popleft() # 从队列的最左端（队首）取出一个元素  
  
        # 尝试每一种预处理出来的步长  
        for move in possible_moves:  
            # 核心位置更新公式解析：  
            # 题目公式：(p + x + y - 1) % L + 1  
            # 这里的 move 已经是 (x + y) % L 了  
            # 这种写法是为了处理“从1开始计数”的环：  
            # 先减1将范围变为 [0, L-1]，取模后再加1变回 [1, L]            
            next_p = (curr_p + move - 1) % l + 1  
  
            # 如果到达了出口 L，直接返回步数（当前步数 + 1）  
            if next_p == l:  
                print(steps + 1)  
                return  
  
            # 如果这个位置之前没来过，加入队列继续搜索  
            if not visited[next_p]:  
                visited[next_p] = True  
                queue.append((next_p, steps + 1))  
  
    # ---------------------------------------------------------  
    # 3. 结果输出  
    # ---------------------------------------------------------  
    # 如果队列空了还没找到 L，说明无法到达  
    print("-1")  
  
  
if __name__ == "__main__":  
    solve()
```
