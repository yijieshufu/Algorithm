### 1. 核心规律：[[找必败态]]

在 Nim 游戏中，两名玩家交替行动。规则是每次可以拿走 $1$ 到 $3$ 块石头，拿到最后一块石头的人获胜。我们通过分析石头数量 $n$ 来寻找规律：

- **$n = 1, 2, 3$**：你可以一次性全部拿走，**你赢**。
    
- **$n = 4$**：无论你拿走多少个（$1, 2$ 或 $3$ 个），剩下的数量（$3, 2$ 或 $1$ 个）都会让对手一次性拿完。**你必败**。
    
- **$n = 5, 6, 7$**：你可以分别拿走 $1, 2, 3$ 个，使剩下的石头数量变为 $4$。此时对手面对的是必败态（$n=4$），所以**你赢**。
    
- **$n = 8$**：无论你拿走几个，对手都可以通过调整，使得剩下的石头再次回到 $4$ 的倍数。**你必败**。
    

### 2. 结论

通过归纳法可以得出：如果当前石头的数量 $n$ 是 **$4$ 的倍数**，那么先手（你）必败；否则，先手必胜。

> **原理**：只要 $n$ 不是 $4$ 的倍数，你总可以拿走 $r = n \pmod 4$ 个石头，让对手面对 $4$ 的倍数。之后无论对手拿走 $k$ 个（$1 \le k \le 3$），你只需拿走 $4-k$ 个，保持剩下的石头始终是 $4$ 的倍数，直到最后剩下 $4$ 个给对手。

## 算法复杂度

- **时间复杂度**：$O(1)$。仅需一次取模运算。
    
- **空间复杂度**：$O(1)$。不需要额外的存储空间。


```Python
class Solution:
    def canWinNim(self, n: int) -> bool:
        return n % 4 != 0
```
