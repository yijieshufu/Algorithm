# 70.爬楼梯
## 方法一：递归 + 记忆化搜索

```Python
class Solution:
    def climbStairs(self, n: int) -> int:
        # 1. 备忘录，初始全是 0
        memo = [0] * (n + 1)
        
        def dfs(i):
            # 2. 边界条件：第 1 阶有 1 种，第 2 阶有 2 种
            if i <= 2: return i
            # 3. 查表：如果算过了，直接给结果
            if memo[i] != 0: return memo[i]
            
            # 4. 决策：爬 1 步的方法 + 爬 2 步的方法
            memo[i] = dfs(i - 1) + dfs(i - 2)
            return memo[i]
	    return dfs(n)

```

## 方法二：迭代递推（蓝桥杯最常用）
这种方法是“自底向上”。先算出 $1$ 和 $2$，然后算出 $3$，$4$……直到 $n$。

```Python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2: return n
        # dp[i] 表示到达第 i 阶的方法数
        dp = [0] * (n + 1)
        dp[1] = 1
        dp[2] = 2
        
        for i in range(3, n + 1):
            dp[i] = dp[i-1] + dp[i-2]
            
        return dp[n]
```
