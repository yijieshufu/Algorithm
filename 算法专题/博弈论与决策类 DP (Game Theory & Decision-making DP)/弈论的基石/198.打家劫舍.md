# 198.打家劫舍

### 核心限制

你不能打劫**相邻**的两家。

### 关键决策（Rob or Skip?）

当你走到第 $i$ 家门口时，你面临一个抉择：

1. **打劫这一家**：既然不能动邻居，那么你目前的总钱数 = **第 $i$ 家的钱 + 到第 $i-2$ 家为止能抢到的最大值**。
    
2. **不抢这一家**：那么你目前的总钱数 = **到第 $i-1$ 家为止能抢到的最大值**。
    

### 状态定义

设 $dp[i]$ 为：**打劫到第 $i$ 家时，能抢到的最高金额**。

### 状态转移方程

$$
dp[i] = \max(dp[i-1], \text{nums}[i] + dp[i-2])
$$

> [!TIP]
> 
> 这个 $\max$ 极其重要！它代表了你的“贪婪”属性——在两个选项中永远选钱最多的那个。这和《石子游戏 II》中 Alice 永远选得分最高的逻辑是一模一样的。


```Python
class Solution:
    def rob(self, nums: List[int]) -> int:
        if not nums: return 0
        n = len(nums)
        if n == 1: return nums[0]
        
        # 1. 初始化 dp 数组
        # dp[i] 表示前 i 个房子能偷到的最大金额
        dp = [0] * n
        dp[0] = nums[0]
        dp[1] = max(nums[0], nums[1])
        
        # 2. 状态转移
        for i in range(2, n):
            # 选“抢这家”或“不抢这家”中的最大值
            dp[i] = max(dp[i-1], nums[i] + dp[i-2])
            
        return dp[n-1]
```
