# 1406. 石子游戏 III (Stone Game III)

```Python
class Solution:
    def stoneGameIII(self, stoneValue: List[int]) -> str:
        n = len(stoneValue)
        # dp[i] 表示从第 i 堆开始，当前玩家能领先对手的最大分差
        # 多开 3 个空间防止 i+1, i+2, i+3 越界
        # dp 初始化：[负无穷, 负无穷, 负无穷, 0, 0, 0]
        dp = [float('-inf')] * n + [0] * 3
        
        # 从后往前计算
        for i in range(n - 1, -1, -1):
            # 选项 1: 拿走 1 堆
            dp[i] = max(dp[i], stoneValue[i] - dp[i+1])
            
            # 选项 2: 拿走 2 堆（前提是有 2 堆可拿）
            if i + 1 < n:
                dp[i] = max(dp[i], stoneValue[i] + stoneValue[i+1] - dp[i+2])
                
            # 选项 3: 拿走 3 堆（前提是有 3 堆可拿）
            if i + 2 < n:
                dp[i] = max(dp[i], stoneValue[i] + stoneValue[i+1] + stoneValue[i+2] - dp[i+3])
        
        # 结果判定
        if dp[0] > 0: return "Alice"
        if dp[0] < 0: return "Bob"
        return "Tie"
```

| ------ | -------- | --------------------------------------------------------------- | ----------------------- |
| **初始** | -        | -                                                               | `[-inf, -inf, -inf]`    |
| **1**  | 2        | `3 - dp[3] = 3 - 0 = 3`                                         | `[-inf, -inf, 3]`       |
| **2**  | 1        | `max(2-dp[2], 2+3-dp[3]) = max(2-3, 5-0) = 5`                   | `[-inf, 5, 3]`          |
| **3**  | 0        | `max(1-dp[1], 1+2-dp[2], 1+2+3-dp[3]) = max(1-5, 3-3, 6-0) = 6` | `[6, 5, 3]`             |

### 1. 题目逻辑分析

- **核心变化**：不再有复杂的 $M$，你每次只能固定选择拿走 **1 堆、2 堆或 3 堆**。
    
- **胜负判定**：比较 Alice 和 Bob 最终拿到的石子总数。
    
- **最优策略思维**：
    
    由于总石子数固定，我们使用**[[相对分数（分差）]]** 来解题会非常优雅：
    
    > 设 $f[i]$ 为：从第 $i$ 堆石子开始到结束，当前玩家能领先对手的最大分差。
    

### 2. 状态转移方程

对于当前玩家，有三种选择：

1. 拿 1 堆：获得 $piles[i]$，对手面对剩余石子能领前的分差是 $f[i+1]$。
    
    分差 = $piles[i] - f[i+1]$
    
2. 拿 2 堆：获得 $piles[i] + piles[i+1]$，对手领前的分差是 $f[i+2]$。
    
    分差 = $(piles[i] + piles[i+1]) - f[i+2]$
    
3. 拿 3 堆：获得 $piles[i] + piles[i+1] + piles[i+2]$，对手领前的分差是 $f[i+3]$。
    
    分差 = $(piles[i] + piles[i+1] + piles[i+2]) - f[i+3]$
    

**当前玩家一定会选择这三者中的最大值。**
