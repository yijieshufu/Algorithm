这道题是算法竞赛中处理**大数幂模运算**的经典题型。在蓝桥杯这类比赛中，这类问题通常考察对幂运算性质的理解。

### 1. 考察算法类型

- **快速幂 (Fast Power / Modular Exponentiation)**
    
- **模运算性质**：$(a \times b) \pmod m = ((a \pmod m) \times (b \pmod m)) \pmod m$
    
- **数学归纳/递归思想**：将数组表示的大指数进行拆解。
    

---

### 2. 最优解思路

由于指数 $b$ 以数组形式给出（例如 $b = [1, 2, 3]$），直接拼接成整数会导致溢出。我们需要利用幂运算的性质进行拆解：

$$a^{[1, 2, 3]} = a^{12 \times 10 + 3} = (a^{[1, 2]})^{10} \times a^3 \pmod{1337}$$

**具体步骤：**

1. **从左向右遍历**数组 $b$ 的每一位数字 $d$。
    
2. 维护一个结果变量 `res`，初始为 1。
    
3. 对于每一个数字 $d$，更新 `res` 为：`(res**10 * a**d) % 1337`。
    
4. 利用 Python 内置的 `pow(base, exp, mod)` 函数，可以高效计算 $a^d \pmod{1337}$。
    

**复杂度分析：**

- **时间复杂度**：$O(n)$，其中 $n$ 是数组 $b$ 的长度。每次循环进行两次快速幂（一次 10 次方，一次 $d$ 次方），复杂度均为常数级。
    
- **空间复杂度**：$O(1)$。
    

---

### 3. Python 3.8.6 代码实现

```Python
class Solution:
    def superPow(self, a: int, b: List[int]) -> int:
        MOD = 1337
        res = 1
        # 预处理 a，缩小底数
        a %= MOD
        
        for d in b:
            # 核心逻辑：res = (res^10 * a^d) % MOD
            # 利用 Python 内置 pow(base, exp, mod) 的高效性
            res = (pow(res, 10, MOD) * pow(a, d, MOD)) % MOD
            
        return res
```
