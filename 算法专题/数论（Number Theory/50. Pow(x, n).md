### 算法类型

**快速幂 (Binary Exponentiation / Exponentiation by Squaring)**

### 最优解思路

1. **二分思想**：计算 $x^n$ 时，与其连乘 $n$ 次，不如利用性质 $x^n = (x^2)^{n/2}$。每次将指数减半，底数平方。
    
2. **位运算优化**：
    
    - 判断 $n$ 是否为奇数：使用 `n & 1`（比 `n % 2 == 1` 更快）。
        
    - 将 $n$ 折半：使用 `n >>= 1`（即整除 2）。
        
3. **处理负指数**：如果 $n < 0$，根据数学规则 $x^n = (\frac{1}{x})^{-n}$。先将底数取倒数，再将指数转为正数。
    
4. **复杂度**：
    
    - **时间复杂度**：$O(\log n)$，因为每次循环指数都会减少一半。
        
    - **空间复杂度**：$O(1)$，仅使用常数级变量。

### Python 3.8.6 提交代码

在 LeetCode 环境下需要保留类结构，但代码已按竞赛要求压缩到最精简：
```Python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        # 处理负指数情况
        if n < 0: 
            x, n = 1 / x, -n
        
        res = 1.0
        while n:
            # 如果当前指数是奇数，将当前的底数乘入结果
            if n & 1: 
                res *= x
            # 底数自乘（平方），指数右移（折半）
            x *= x
            n >>= 1
            
        return res
```

### 代码执行过程演示 (以 $2^{10}$ 为例)

- **初始**：`res=1`, `x=2`, `n=10(1010₂)`
    
- **第一轮**：`n` 是偶数，`res` 不变；`x` 变为 $2^2=4$，`n` 变为 $5(101₂)$
    
- **第二轮**：`n` 是奇数，`res = 1 * 4 = 4`；`x` 变为 $4^2=16$，`n` 变为 $2(10₂)$
    
- **第三轮**：`n` 是偶数，`res` 不变；`x` 变为 $16^2=256$，`n` 变为 $1(1₂)$
    
- **第四轮**：`n` 是奇数，`res = 4 * 256 = 1024`；`x` 变为 $256^2$，`n` 变为 $0$
    
- **结束**：返回 `1024`。
