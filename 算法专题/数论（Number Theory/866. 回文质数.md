这道题目在蓝桥杯或 LeetCode 中属于经典的**构造法**结合**数论性质**的题目。

## 考察算法类型

- **数论性质**：除 11 以外，所有偶数位长度的回文数都能被 11 整除（非质数）。
    
- **构造法**：与其遍历数字判断是否为回文，不如**直接构造回文数**。
    
- **质数判定**：使用试除法即可，效率足够。
    

---

## 最优解思路

1. **关键性质优化**：
    
    - 除了 11 以外，所有**偶数位数**的回文数（如 1221, 654456 等）都是 11 的倍数。
        
    - **证明简述**：回文数的奇偶位差之和为 0，根据 11 的整除特征，它必能被 11 整除。
        
    - **结论**：如果 $N$ 在 $10^7$ 到 $10^8$ 之间（8 位数），我们可以直接跳到 $10^8$ 之后寻找 9 位数的回文质数。
        
2. **构造回文序列**：
    
    - 回文数可以通过枚举“前半部分”来生成。
        
    - 例如：前半部分为 `123`，构造奇数长度回文为 `12321`。
        
    - 由于 $N \le 10^8$，我们只需要构造 1、3、5、7、9 位数的回文数（以及特殊处理 11）。
        
3. **流程**：
    
    - 特殊处理 $N \le 11$ 的情况（直接判断 2, 3, 5, 7, 11）。
        
    - 从前半部分（root）开始枚举，通过 `s + s[-2::-1]` 构造奇数长度回文数。
        
    - 对构造出的数进行 $N$ 范围判定和质数判定，找到第一个即可停止。

## LeetCode 提交代码
```Python
class Solution:
    def primePalindrome(self, n: int) -> int:
        # 质数判定函数
        def is_p(x):
            if x < 2 or x % 2 == 0: return x == 2
            for i in range(3, int(x**0.5) + 1, 2):
                if x % i == 0: return False
            return True
        
        # 处理小规模数据和偶数位特例 11
        if 8 <= n <= 11: return 11
        
        # 构造奇数长度回文数 (长度为 1, 3, 5, 7, 9)
        # 对应前半部分 root 的范围
        for length in range(1, 6):
            # 构造长度为 2*length - 1 的回文数
            for i in range(10**(length-1), 10**length):
                s = str(i)
                # 镜像构造：abc -> abcba
                v = int(s + s[-2::-1]) 
                # `s[start:stop:step]` 是一个功能强大的工具
                if v >= n and is_p(v):
                    return v
```

## 复杂度分析

- **时间复杂度**：$O(\sqrt{M} \cdot \sqrt{P})$。其中 $\sqrt{M}$ 是构造回文数的数量（约 $10^4$ 级别），$\sqrt{P}$ 是质数判定的开销（最大约 $\sqrt{10^9} \approx 31622$）。实际运行远远小于此上限，能秒过所有测试点。
    
- **空间复杂度**：$O(1)$，仅使用了常数空间。
    

## 教练提醒

> 在蓝桥杯中，遇到涉及 $10^8$ 甚至更大的回文数搜索时，**永远不要从 $N$ 开始自增遍历**。必须利用“偶数位回文数必能被 11 整除”这一性质直接跳过大量区间，或直接采用**构造法**生成回文数，这是通关此类题目的唯一最优解。
