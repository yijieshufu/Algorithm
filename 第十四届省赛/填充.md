# 填充


```python
import os  
import sys  
# 读取输入的一行字符串  
s = input()  
# 获取字符串长度  
n = len(s)  
# 定义所有“合法的”或“潜在合法”的长度为2的子串模式  
# 只要遇到这些组合，就可以通过填充 '?' 变成 '00' 或 '11'# 00: 本身就是  
# 11: 本身就是  
# 0?: ?填0 -> 00  
# 1?: ?填1 -> 11  
# ?0: ?填0 -> 00  
# ?1: ?填1 -> 11  
# ??: ?填00或11都可以  
ans = 0  # 用于记录找到的子串数量  
i = 1    # 指针，从第2个字符开始（索引为1），为了方便向回看 i-1# 遍历字符串  
while i < n:  
    current_pair = s[i - 1:i + 1]  
    # 如果不是 '01' 且 不是 '10'，那就是合法的  
    if current_pair != '01' and current_pair != '10':  
        ans += 1  
        i += 2  
    else:  
        i += 1    # 如果当前这两个凑不成一对（比如是 '01' 或 '10'），  
                  # 那么前一个字符 s[i-1] 肯定没用了，指针向后移1位，  
                  # 尝试看 s[i] 和 s[i+1] 能否组合。  
print(ans) # 输出最终结果
```
