

### 第一部分：题目深度剖析

#### 1. 核心操作的秘密：`s ^ rev(s)`

题目的关键在于理解操作 $s' = s \oplus rev(s)$ 会生成什么样的字符串。

假设 $s$ 的长度为 $L$，第 $i$ 位字符是 $s[i]$（$0 \le i < L$）。

$rev(s)$ 的第 $i$ 位就是 $s[L-1-i]$。

异或操作后，$s'[i] = s[i] \oplus s[L-1-i]$。

根据异或的交换律，我们可以发现：

$$
s'[L-1-i] = s[L-1-i] \oplus s[i] = s'[i]
$$

**结论 1：** 这个操作生成的一定是一个 **[回文串](回文串.md)**。

[回文串的定义](回文串的定义.md)
#### 2. 中心的特殊限制

对于回文串，我们需要区分奇数长度和偶数长度：

- **偶数长度**：对称中心在两个字符之间，不需要考虑中心字符的值。
    
- **奇数长度**：存在一个中心字符。假设中心位置为 $mid$，根据异或性质：

    $$


s'[mid] = s[mid] \oplus s[mid] = 0

$$
    

**结论 2：** 如果生成的串是奇数长度，**它的中心字符必须是 '0'**。如果是偶数长度，则无限制。

#### 3. 贪心策略（如何最少添加）

我们的目标是构造出 $T$。

- 添加 `0`：免费（或者说不影响我们计算 `1` 的数量）。
    
- 添加 `1`：需要花费 1 个代价。
    
- 使用异或操作：可以一次性生成一对 `1`（例如 $s=10, rev=01 \to 11$）。生成这一对 `1`，我们在 $s$ 中只需要放入一个 `1`。
    

**[最省钱的策略](最省钱的策略.md)：**

我们希望 $T$ 中尽可能多的一段连续子串（Palindrome）是由异或操作生成的。

因为在这个回文子串中，**每有两个对称的 '1'，我们实际只需要花费 1 个代价**。

换句话说，如果在 $T$ 中找到一个符合条件的回文子串，其中包含 $K$ 个 '1'，那么我们通过操作生成它只需要 $K/2$ 个 '1'。

我们可以节省的代价 = $K/2$（向下取整，因为奇数个1中间那个如果是1是不能被生成的，但本题中心限制为0，所以回文串里的1总是成对出现，肯定是偶数个）。

**最终算法思路：**

1. 统计 $T$ 中所有的 '1' 的个数，记为 `total_ones`。
    
2. 在 $T$ 中找到一个**最长回文子串**，要求该回文子串如果长度为奇数，中心必须是 '0'。
    
3. 计算这个最佳回文子串里包含的 '1' 的个数，记为 `max_sub_ones`。
    
4. 答案 = `total_ones - (max_sub_ones / 2)`。

### 第二部分： 代码与详解

```python
import sys
from itertools import accumulate

raw_s = sys.stdin.read().strip()
if not raw_s:
    print(0)
    exit()
t = '#'.join('^{}$'.format(raw_s))
n = len(t)
p = [0] * n
c = r = 0
pre = [0] + list(accumulate(1 if c == '1' else 0 for c in t))
mx = 0
for i in range(1, n - 1):
    p[i] = min(r - i, p[2 * c - i]) if r > i else 1
    while t[i + p[i]] == t[i - p[i]]: p[i] += 1
    if i + p[i] > r: c, r = i, i + p[i]
    if t[i] != '1':
        cnt = pre[i + p[i]] - pre[i - p[i] + 1]
        mx = max(mx, cnt)
print(raw_s.count('1') - mx // 2)
```

### 复杂度分析

- **时间复杂度**：$O(N)$。Manacher 算法是线性的，前缀和预处理也是线性的。这对于 $N \le 10^6$ 的数据量是必须的。
    
- **空间复杂度**：$O(N)$。用于存储变换后的字符串、半径数组 $P$ 和前缀和数组。
    

### 总结

这道题其实是披着“字符串操作”外衣的数学题。

- 看到 **异或+反转** $\rightarrow$ 想到 **回文**。
    
- 看到 **回文中心** $\rightarrow$ 想到 **$x \oplus x = 0$**。
    
- 看到 **最少添加** $\rightarrow$ 想到 **最长合法回文子串覆盖**。
    
- 看到 **$10^6$ 数据量** $\rightarrow$ 想到 **Manacher $O(N)$**。
