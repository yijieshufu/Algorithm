```Python
import sys
import heapq
from collections import defaultdict

# 快速读取
data = sys.stdin.read().split()
if not data: exit()
it = iter(data)

n = int(next(it))
m = int(next(it))
x = int(next(it))

# 初始库存
a = [int(next(it)) for _ in range(n)] #每种书 当前还有多少本


book_reqs = defaultdict(list) 
# 这里的 list 是“工厂函数”，代表默认值是一个空列表 []
# 如果钥匙（Key）不存在，不要报错，直接给我造一个空的。
# 按书本类型分组存储请求
for _ in range(m):
    b, l, r = int(next(it)), int(next(it)), int(next(it))
    book_reqs[b].append((l, r))

total_satisfied = 0
pool = []

for i in range(1, n + 1):
    reqs = book_reqs[i]
    if not reqs: continue
    
    # 按开始时间排序
    reqs.sort()
    
    # h 存储 (归还时间, 该副本已满足的请求数)
    h = []
    for l, r in reqs:
        # 如果当前最早归还的书在 l 之前已经归还 (l > h[0][0])
        if h and l > h[0][0]:
            _, count = heapq.heappop(h)
            heapq.heappush(h, (r, count + 1))
        else:
            heapq.heappush(h, (r, 1))
    
    # 提取所有层的收益
    counts = sorted([x[1] for x in h], reverse=True)
    
    # 初始库存 a[i-1] 可以覆盖收益最高的几层
    stock = a[i-1]
    for j in range(len(counts)):
        if j < stock:
            total_satisfied += counts[j]
        else:
            # 超出库存的层放入全局竞争池
            pool.append(counts[j])

# 从池子中选出收益最大的 x 个
pool.sort(reverse=True)
total_satisfied += sum(pool[:x])

print(total_satisfied)
```

在蓝桥杯中，你通常会用到以下三种 `defaultdict`：

| **定义方式**            | **默认值** | **适用场景**               |
| ------------------- | ------- | ---------------------- |
| `defaultdict(list)` | `[]`    | **建图（邻接表）**、按类分组。      |
| `defaultdict(int)`  | `0`     | **计数器**（如统计每个字符出现的次数）。 |
| `defaultdict(set)`  | `set()` | 去重集合分组。                |
