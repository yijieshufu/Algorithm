```python
import sys
import collections

# 使用 split() 能够自动处理换行符和多余空格，避免读取错误
data = sys.stdin.read().split()
if data:
    n = int(data[0])
    
    # 核心优化：若 n 是 2 的倍数，其倍数中 1 的最少个数与 n//2 相同

    while n > 0 and n % 2 == 0:
        n //= 2 
    # 如果 n 被除到了 1，说明原数是 2 的幂，二进制中 1 的个数最少显然是 1
    if n == 1:
        print(1)
    else:
        # dist[i] 记录余数为 i 时二进制中 1 的最少个数
        # 使用 -1 初始化表示未访问，比 float('inf') 更省内存
        dist = [-1] * n
        start = 1 % n
        dist[start] = 1
        dq = collections.deque([start])
        
        while dq:
        
            u = dq.popleft()
            
            # 找到余数为 0 的路径
            if u == 0:
                print(dist[u])
                break
            
            # 0-1 BFS 转移：
            # 1. 后面加 0：新余数 v0 = (u * 2) % n，代价不变 (边权 0)
            v0 = (u << 1) % n
            if dist[v0] == -1 or dist[v0] > dist[u]:
                dist[v0] = dist[u]
                dq.appendleft(v0) # 权值为 0，插到队首优先处理
            
            # 2. 后面加 1：新余数 v1 = (u * 2 + 1) % n，代价 + 1 (边权 1)
            v1 = (v0 + 1) % n
            if dist[v1] == -1 or dist[v1] > dist[u] + 1:
                dist[v1] = dist[u] + 1
                dq.append(v1) # 权值为 1，插到队尾
```


**核心规律**：一个数 $x$ 和它的 $2$ 倍 $2x$（二进制下左移一位），它们包含的 **1 的个数是完全相同的**。
