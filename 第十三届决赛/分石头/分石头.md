这道题目是一个典型的**公平组合游戏**，可以通过 **[[SG 定理]] (Sprague-Grundy Theorem)** 来解决。

### 题目分析

1. **游戏类型**：这是一个不公平（Impartial）的游戏，即两人的合法操作集合仅取决于游戏状态，与轮到谁无关。可以用 SG 函数求解。
    
2. **操作逻辑**：
    
    - **直接消除**：可以将一堆石子变为 0。
        
    - **分成分数**：选择一个质数 $p$（必须是当前堆数量 $x$ 的因数），将 $x$ 分成 $p$ 份，每份大小为 $x/p$。
        
3. **[[SG 值计算]]**：
    
    - 从状态 $x$ 出发，可以直接到达状态 0。
        
    - 分成的 $p$ 份石子相当于 $p$ 个独立的游戏，根据 SG 定理，该操作到达的状态 SG 值为：$SG(x/p) \oplus SG(x/p) \oplus \dots \oplus SG(x/p)$ （共 $p$ 个）。
        
        - 若 $p = 2$，则 $SG(x/2) \oplus SG(x/2) = 0$。
            
        - 若 $p > 2$（奇质数），则有奇数个相同的数异或，结果为 $SG(x/p)$。
            
    - 因此，$SG(x) = \text{mex}(\{0\} \cup \{ SG(x/p) \mid p \text{ 是 } x \text{ 的奇质因数} \})$。

### 最优解思路

- **规律推导**：
    
    - 令 $f(x)$ 为 $x$ 的**奇质因子总个数**（含重数）。
        
    - $f(x) = 0$（例如 $x=1, 2, 4$）时，$SG(x) = \text{mex}\{0\} = 1$。
        
    - $f(x) = 1$（例如 $x=3, 6, 12$）时，$SG(x) = \text{mex}\{0, SG(f=0)\} = \text{mex}\{0, 1\} = 2$。
        
    - $f(x) = 2$（例如 $x=9, 15, 45$）时，$SG(x) = \text{mex}\{0, SG(f=1)\} = \text{mex}\{0, 2\} = 1$。
        
    - 结论：**$SG(x) = 2$ 若 $x$ 的奇质因子总数为奇数，否则 $SG(x) = 1$**。
        
- **优化**：
    
    - 石子数量最高 $10^6$，先用线性筛或埃氏筛预处理每个数的最小质因子。
        
    - 递推计算每个数的奇质因子**个数奇偶性**。
        
    - 对每组询问，将 $n$ 堆石子的 SG 值进行异或，若结果不为 0 则输出 1，否则输出 0。

### Python 代码 (3.8.6)
```python
import sys
M = 1000001
min_p = list(range(M))
for i in range(2, 1001):
    if min_p[i] == i:
        for j in range(i * i, M, i):
            if min_p[j] == j:
                min_p[j] = i
sg = [0] * M
sg[1] = 1
for i in range(2, M):
    p = min_p[i]
    sg[i] = sg[i >> 1] if p == 2 else 3 - sg[i // p]
it = iter(sys.stdin.read().split())
T = int(next(it))
res = []
for _ in range(T):
    xor_sum = 0
    for _ in range(int(next(it))):
        xor_sum ^= sg[int(next(it))]
    res.append('1' if xor_sum else '0')
print('\n'.join(res))
```
