# **试题A: 劲舞团**

```python
# 1. 初始化变量
max_combo = 0      # 对应原代码的 ans
current_combo = 0  # 对应原代码的 count
last_correct_time = None # 记录上一次“正确”敲击的时间

# 2. 读取文件
with open("log.txt", "r") as file:
    lines = file.readlines()

# 3. 遍历每一行数据
for line in lines:
    # 拆分数据
    parts = line.split()
    if len(parts) < 3:
        continue
    
    target_char = parts[0]   # 应该打的
    user_char = parts[1]     # 实际打的
    current_time = int(parts[2]) # 当前时间戳

    # --- 逻辑核心开始 ---
    
    # 情况 A：打错了
    if target_char != user_char:
        current_combo = 1  # 连击重置为 1（注意：这里不更新时间，也不更新最大值）
        continue           # 直接跳过，看下一行

    # 情况 B：打对了
    # 如果是第一次打对
    if last_correct_time is None:
        current_combo = 1
    else:
        # 判断跟上一次“正确”敲击的时间差
        if current_time - last_correct_time <= 1000:
            current_combo += 1
        else:
            current_combo = 1
    
    # 只要打对了，就更新“最后一次正确时间”并挑战最大连击纪录
    last_correct_time = current_time
    if current_combo > max_combo:
        max_combo = current_combo

# 4. 输出最终结果
print(max_combo)
```
