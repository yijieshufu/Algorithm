### 1. 什么是质数？

**质数**是指一个大于 1 的自然数，**除了 1 和它自己以外，没法被其他任何数整除。**

**0 和 1** 既不是质数，也不是合数（它们是数字界的特殊分子）。

**2** 是唯一的偶数质数，也是最小的质数。

#### 埃氏筛法
如果我们知道 **2** 是质数，那么 2 的所有倍数（4, 6, 8, 10...）**绝对不可能是质数**

```python
import sys 
# 导入系统库。Python 默认的 input() 很慢， 
# 在这种处理上万个数据的问题里，用 sys 库能快几十倍。

def solve():
    # 因为题目给出的 n 最大是 100,000， 
    # 我们多预留一点空间（100005），防止数组越界。
    MAXN = 100005
    
    # 1. 埃拉托斯特尼筛法预处理质数
    is_prime = [True] * MAXN # 先假设所有数都是质数 (True)
    is_prime[0] = is_prime[1] = False # 0 和 1 既不是质数也不是合数，直接设为 False
    # 从 2 开始往后筛选，直到这个数的平方超过最大值（数学上的优化）
    for p in range(2, int(MAXN**0.5) + 1):
        if is_prime[p]: #如果 p 还是 True，说明它没被之前的数筛掉，它就是质数
            for i in range(p * p, MAXN, p): # 那么 p 的所有倍数（如 2p, 3p, 4p...）一定不是质数
	            # 我们从 p*p 开始标记（比从 2*p 开始效率更高）
                is_prime[i] = False
    
    # 2. 动态规划预处理胜负态
    # win[i] 表示长度为 i 时是否为必胜态
    win = [False] * MAXN # 胜负表
    loss_states = [0, 1] # 初始必败态 
    
    for i in range(2, MAXN):
        # 优化：只需检查当前的 i 减去之前的必败态是否等于某个质数
        # 如果能到达任何一个必败态，当前 i 就是必胜态
        can_win = False # 先默认小蓝会输
        for loss in loss_states:
            diff = i - loss # 我现在手里有长度 `i`，我能不能减去一个质数，刚好让剩下的长度变成名单里的某个‘必败态’（`loss`）？
            if diff < 2: 
                continue
            if is_prime[diff]:
                can_win = True 
                break
        
        if can_win:
            win[i] = True
        else:
            win[i] = False
            loss_states.append(i) # 记录新的必败态
            
    # 3. 处理输入输出
    # 使用 sys.stdin.read 一次性读取提高效率
    input_data = sys.stdin.read().split()
    if not input_data:
        return
        
    T = int(input_data[0])
    results = []
    for k in range(1, T + 1):
        n = int(input_data[k])
        results.append("1" if win[n] else "0")
    
    # 批量输出
    sys.stdout.write("\n".join(results) + "\n")

if __name__ == "__main__":
    solve()
```
