# 题目
**题目名称**：子段异或和之和

**核心任务**：给定一个长度为 $n$ 的数组 $A$。你需要计算所有可能的子段 $[L, R]$（其中 $1 \le L \le R \le n$）的异或和，最后把这些异或和全部加起来。

**约束条件**：

- $n \le 10^5$（必须在线性或 $O(n \log n)$ 级别解决）。
    
- $A_i \le 2^{20}$（提示我们要从“位”的角度思考）。

# 代码
```python
import os
import sys
it = iter(sys.stdin.read().split())
n = int(next(it))
a = [int(next(it)) for _ in range(n)]
s = [0] * (n + 1)
for i in range(n):
  s[i+1]=s[i] ^ a[i] # 异或前缀
ans = 0
for k in range(21):
  cnt0 = 0
  cnt1 = 0
  for i in range(n + 1):
    if (s[i] >> k) & 1:
      cnt1+=1
    else:
      cnt0+=1
  ans += (cnt0 * cnt1) * (1 << k)
print(ans)
```
