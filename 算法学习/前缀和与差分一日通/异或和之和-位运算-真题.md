# 题目
- **核心动作**：给定数组 $A$，求所有子段 $[L, R]$ 的异或和之总和，即 $\sum_{1 \le L \le R \le n} (A_L \oplus \dots \oplus A_R)$。
- **数据边界**：$n \le 10^5$，$A_i < 2^{20}$。$O(n^2)$ 暴力必炸，需 $O(n \log(\max A_i))$ 级别算法。
# 分析
- **算法/数据结构**：**按位贡献 + 前缀异或和**。
- **为何适用**：异或运算位与位之间互不干扰（无进位），且“求和”操作支持拆分计算。
- **通用骨架**：
```python
    # 拆位模板
    for k in range(21): # 遍历每一位
        # 统计该位对答案的贡献
        ans += (该位为1的子段数量) * (1 << k)
```
- **定级**：中等题（蓝桥杯省赛常见考点）。
- **核心难点**：理解如何利用前缀异或和 $S[R] \oplus S[L-1]$ 将区间问题转化为两个点位的位差，并利用计数原理避开嵌套循环。

- **[块 1：前缀异或转化]**  
    设 $S[i] = A[1] \oplus \dots \oplus A[i]$，则子段 $[L, R]$ 的异或和等于 $S[R] \oplus S[L-1]$。  
    题目转化为求 $\sum_{0 \le i < j \le n} (S[j] \oplus S[i])$。
- **[块 2：按位贡献与计数]**  
    对于第 $k$ 位，若 $S[j] \oplus S[i]$ 的第 $k$ 位为 $1$，当且仅当 $S[j]$ 与 $S[i]$ 的第 $k$ 位不同（一个是 $0$，一个是 $1$）。
- **[块 3：组合贡献计算]**  
    统计所有前缀和 $S[0 \dots n]$ 中，第 $k$ 位为 $0$ 的个数 $cnt0$ 和为 $1$ 的个数 $cnt1$。  
    该位贡献的“1”的总数为 $cnt0 \times cnt1$。  
	$$\underbrace{\text{Ans}}_{\text{最终答案}} = \sum_{k=0}^{20} (\underbrace{cnt0 \times cnt1}_{\text{第 } k \text{ 位是 1 的子段总数}} \times \underbrace{2^k}_{\text{该位的数值权值}})$$
