# 题目描述
- **核心目标**：在序列前 $x$ 个数中选出 $k$ 个数，使其方差 $\sigma^2 < T$，求最小的 $x$。
- **数据边界**：$n$ 通常为 $10^5$ 级别；方差公式需变形以规避浮点精度误差。
- **关键结论**：若要在 $x$ 个数中选 $k$ 个数使方差最小，**这 $k$ 个数在排序后必然是连续的**。
# 分析
- **算法/数据结构**：**二分答案 + 排序滑动窗口**。
- **为何适用**：$x$ 越大，可选范围越广，满足条件的概率越高，具有**单调性**。
- **通用骨架**：  
```python
    def check(mid):
        # 提取前 mid 个并排序
        # 滑动窗口计算 k 长度区间方差
        return True / False
    
    l, r, ans = k, n, -1
    while l <= r:
        mid = (l + r) // 2
        if check(mid): ans = mid; r = mid - 1
        else: l = mid + 1
```

- **定级**：中等题。
- **核心难点**：推导方差变形公式 $k \cdot \sum v_i^2 - (\sum v_i)^2 < k^2 T$ 以避免**浮点数精度**丢失，并理解排序后连续选取的贪心策略。

- **[块 1：方差公式降维]**  
    根据 $\sigma^2 = \frac{\sum v_i^2}{k} - (\frac{\sum v_i}{k})^2 < T$，两边同乘 $k^2$ 得：$k \cdot \sum v_i^2 - (\sum v_i)^2 < k^2 T$。全整数运算，精度无忧。
- **[块 2：Check 函数逻辑]**  
    对于给定的 $x$，取出 $a[0:x]$ 并排序。使用长度为 $k$ 的滑动窗口，计算窗口内的 $\sum v_i$ 和 $\sum v_i^2$。只需存在一个窗口满足块 1 的不等式，该 $x$ 即合法。
- **[块 3：二分搜索区间]**  
    左边界为 $k$（至少选 $k$ 个人），右边界为 $n$。若最终 `ans` 仍为初始值，则输出 -1。

## 代码

```python
import os
import sys

it = iter(sys.stdin.read().split())
n = int(next(it))
k = int(next(it))
t = int(next(it))
a = [int(next(it)) for _ in range(n)]
def check(x):
  sub = sorted(a[:x]) # 只排序前x个
  curr_sum = 0;curr_sq_sum=0
  for i in range(k):
    curr_sum +=sub[i] # 和
    curr_sq_sum +=sub[i]*sub[i] # 平方
  target = t*k*k
  if k * curr_sq_sum - curr_sum * curr_sum < target: # 方差公式
    return True
  for i in range(x-k): # 区间移动
    left_val = sub[i] #左边的值
    right_val = sub[i + k] #右边的值
    curr_sum += right_val - left_val # 计算当前的区间
    curr_sq_sum+= right_val*right_val-left_val*left_val
    # 实时检查当前窗口是否达标
    if k * curr_sq_sum - curr_sum * curr_sum < target:
        return True
  return False
# 二分法
ans = -1
low = k 
high = n 
while low<=high:
  mid = (low+high)//2
  if check(mid):
    ans = mid # 要检查的人数
    high=mid-1
  else:
    low=mid+1
print(ans)
```
