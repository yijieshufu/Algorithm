# 题目描述

### 题目转述：【真题】棋盘

- **初始状态**：你有一个 $n \times n$ 的大棋盘，最开始所有格子的棋子都是**白色**（用 $0$ 表示）。
    
- **操作任务**：给你 $m$ 组坐标 $(x_1, y_1, x_2, y_2)$，代表一个矩形区域。你需要把这个区域内所有的棋子**颜色取反**（白变黑，黑变白）。
    
- **最终目标**：执行完这 $m$ 次操作后，输出棋盘上每个格子最终的颜色（白色输出 $0$，黑色输出 $1$）。
    
- **数据规模**：棋盘大小 $n$ 和操作数 $m$ 最大都是 $2000$。

# 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
n = int(next(it))
m = int(next(it))
diff = [[0]*(n+2) for _ in range(n+2)]
for _ in range(m):
  x1 = int(next(it))
  y1 = int(next(it))
  x2 = int(next(it))
  y2 = int(next(it))
  diff[x1][y1]+=1
  diff[x1][y2+1]-=1
  diff[x2+1][y1]-=1
  diff[x2+1][y2+1]+=1  # 先记录好
for i in range(1,n+1):
  row_res = []
  for j in range(1,n+1):
    diff[i][j]+=diff[i-1][j]+diff[i][j-1]-diff[i-1][j-1] # 通过二维前最后求值
    if diff[i][j]%2==1:
      row_res.append('1')
    else:
      row_res.append('0')
  sys.stdout.write("".join(row_res) + "\n")
```
