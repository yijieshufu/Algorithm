# 题目描述
- **初始状态**：你有一个 $n \times n$ 的大棋盘，最开始所有格子的棋子都是**白色**（用 $0$ 表示）。
- **操作任务**：给你 $m$ 组坐标 $(x_1, y_1, x_2, y_2)$，代表一个矩形区域。你需要把这个区域内所有的棋子**颜色取反**（白变黑，黑变白）。
- **最终目标**：执行完这 $m$ 次操作后，输出棋盘上每个格子最终的颜色（白色输出 $0$，黑色输出 $1$）。
- **数据规模**：棋盘大小 $n$ 和操作数 $m$ 最大都是 $2000$。
# 分析
- **算法/数据结构**：二维差分。
- **为何适用**：题目要求进行 $m$ 次区域翻转（Range Update），最后一次性查询全图（Point Query）。这正是差分阵列的“主场”。
- **本题特殊性**：题目是“翻转”，即 $0 \to 1, 1 \to 0$。在数学上等同于对该区域进行 **$+1$ 操作**，最后通过 **`sum % 2`**（奇偶性）来判断颜色。

**[块 1：差分标记（修改）]**  
对于每一次操作 $(x_1, y_1, x_2, y_2)$，我们不改动原数组，只在 `diff` 数组的四个点上做标记。复杂度 $O(1)$。
```python
diff[x1][y1] += 1
diff[x1][y2+1] -= 1
diff[x2+1][y1] -= 1
diff[x2+1][y2+1] += 1
```

**[块 2：前缀和还原（计算）]**

操作全部完成后，通过二维前缀和公式还原出每个点被“翻转”的总次数。  
公式：`dp[i][j] = diff[i][j] + dp[i-1][j] + dp[i][j-1] - dp[i-1][j-1]`  
代码中直接原位更新：`diff[i][j] += diff[i-1][j] + diff[i][j-1] - diff[i-1][j-1]`。  
**[块 3：结果判定]**  
由于初始是白色（0），翻转奇数次变黑（1），翻转偶数次变白（0）。
```python
if diff[i][j] % 2 == 1: 
    ans = '1'
```
### 4. 复杂度心算
- **时间复杂度**：$O(m + n^2)$。$m$ 次标记是 $O(m)$，最后一次前缀和遍历是 $O(n^2)$。
    - 本题 $n, m = 2000$，计算量约为 $4 \times 10^6 + 2000 \approx 4 \times 10^6$，在 Python 1秒运行限制内非常安全。
- **空间复杂度**：$O(n^2)$。需要一个 $2000 \times 2000$ 的二维数组。

## 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
n = int(next(it))
m = int(next(it))
diff = [[0]*(n+2) for _ in range(n+2)]
for _ in range(m):
  x1 = int(next(it))
  y1 = int(next(it))
  x2 = int(next(it))
  y2 = int(next(it))
  diff[x1][y1]+=1
  diff[x1][y2+1]-=1
  diff[x2+1][y1]-=1
  diff[x2+1][y2+1]+=1  # 先记录好
for i in range(1,n+1):
  row_res = []
  for j in range(1,n+1):
    # 通过二维前最后求值
    diff[i][j]+=diff[i-1][j]+diff[i][j-1]-diff[i-1][j-1] 
    if diff[i][j]%2==1:
      row_res.append('1')
    else:
      row_res.append('0')
  sys.stdout.write("".join(row_res) + "\n")
```
