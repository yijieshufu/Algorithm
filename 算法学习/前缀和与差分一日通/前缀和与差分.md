## 蓝桥杯算法笔记：前缀和与差分

**核心思想**：

- **前缀和**是“从点到面/体”的积分过程，主要用于**快速求区间和**。
    
- **差分**是“从面/体还原点”的微分过程，主要用于**频繁的区间加减修改**。
    
- **复杂度**：预处理 $O(n)$，查询 $O(1)$。非常适合处理**静态数据**的多次区间询问。
    

---

### 一、 一维前缀和与差分

- **预处理公式**：$sum_i = sum_{i-1} + a_i$
    
- **区间 $[l, r]$ 求和公式**：$\sum_{i=l}^r a_i = sum_r - sum_{l-1}$
    
- **⚠️ 避坑技巧**：数组下标强烈建议**从 1 开始**，预留 $sum_0 = 0$ 作为哨兵，这样求 $[1, x]$ 的和时直接使用 $sum_x - sum_0$ 即可避免数组越界。
    

---

### 二、 二维前缀和

利用容斥原理，将图形的面积加减转化为公式计算。

- **预处理公式（维护前缀和）**：
    
    $sum_{i,j} = sum_{i-1,j} + sum_{i,j-1} - sum_{i-1,j-1} + a_{i,j}$
    
- **子矩阵 $(x_1, y_1)$ 到 $(x_2, y_2)$ 求和公式**：
    
    $Sum = sum_{x_2,y_2} - sum_{x_1-1,y_2} - sum_{x_2,y_1-1} + sum_{x_1-1,y_1-1}$
    
    _(注意：减去的部分下标是 $x_1-1$ 和 $y_1-1$)_

### 三、 三维前缀和与差分

#### 1. 三维前缀和（推荐降维迭代法）

```python
# 假设 a[x][y][z] 存储了原始数据
# 第一步：沿 X 轴累加
for i in range(1, X+1):
    for j in range(1, Y+1):
        for k in range(1, Z+1):
            a[i][j][k] += a[i-1][j][k]

# 第二步：沿 Y 轴累加
for i in range(1, X+1):
    for j in range(1, Y+1):
        for k in range(1, Z+1):
            a[i][j][k] += a[i][j-1][k]

# 第三步：沿 Z 轴累加
for i in range(1, X+1):
    for j in range(1, Y+1):
        for k in range(1, Z+1):
            a[i][j][k] += a[i][j][k-1]
            
# 此时 a[x][y][z] 即为三维前缀和
```

#### 2. 三维差分
```python
def update(d, x1, y1, z1, x2, y2, z2, c):
    d[x1][y1][z1] += c
    
    d[x2 + 1][y1][z1] -= c
    d[x1][y2 + 1][z1] -= c
    d[x1][y1][z2 + 1] -= c
    
    d[x2 + 1][y2 + 1][z1] += c
    d[x2 + 1][y1][z2 + 1] += c
    d[x1][y2 + 1][z2 + 1] += c
    
    d[x2 + 1][y2 + 1][z2 + 1] -= c
```

### 四、 运算扩展形式

前缀和不仅局限于加法，还支持其他具备**可逆性质**的运算：

- **异或和**：
    
    - 定义：$sum_n = a_1 \oplus a_2 \oplus \dots \oplus a_n$
        
    - 区间查询：$sum_{l,r} = sum_r \oplus sum_{l-1}$
        
- **乘法前缀和（积）**：
    
    - 定义：$P_n = a_1 \times a_2 \times \dots \times a_n$
        
    - 区间查询：$P_{l,r} = \frac{P_r}{P_{l-1}}$ _(注意防除零错或取模求逆元)_
