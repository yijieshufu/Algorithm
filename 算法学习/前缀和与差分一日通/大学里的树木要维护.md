### 1. 题目信息压缩（目标与边界）
- **目标**：在 $N$ 棵线性排列的树中，给定每棵树的维护费用 $A_i$，快速查询 $M$ 个闭区间 $[L, R]$ 内的费用总和。
- **数据边界**：$N, M$ 通常可达 $10^5$ 级。单次查询若采用暴力遍历 $O(N)$，总复杂度 $O(N \cdot M)$ 会超时（TLE），必须优化至 $O(1)$ 查询。

### 2. 算法识别与通用模板（模型映射）
- **算法/数据结构**：**前缀和 (Prefix Sum)**。
- **为何适用**：题目属于典型的“静态数组区间求和”问题，无修改操作，前缀和是此类需求的最优解。
- **通用骨架**：
    1. 预处理：$S[i] = S[i-1] + A[i]$
    2. 查询：$Sum(L, R) = S[R] - S[L-1]$

### 3. 难度评估
- **定级**：基础题。
- **核心难点**：处理 $1$-indexed（从 1 开始编号）的索引偏移，确保 $L-1$ 不越界。

### 4. 认知块解题推导（逻辑降维）
- **[块 1：输入流与空间预分配]**：利用迭代器快速读入 $N, M$，并初始化长度为 $N+1$ 的前缀和数组 `s`，索引 $0$ 位置留空作为哨兵。
- **[块 2：前缀和累加]**：在读入费用的同时，直接计算 $S[i] = S[i-1] + \text{current\_val}$，将空间复杂度控制在 $O(N)$。
- **[块 3：区间求和输出]**：对 $M$ 次查询执行 $S[R] - S[L-1]$，结果即为区间和，直接打印。

## 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
N = int(next(it))
M = int(next(it))
a = [0]+[int(next(it)) for _ in range(N)]
sum = [0]*(N+1) 
for i in range(1,N+1):
  sum[i]=sum[i-1]+a[i] # 计算区间的前缀和
for _ in range(M):
  L=int(next(it))
  R=int(next(it))
  print(sum[R]-sum[L-1])
```
