# 题目

### 题目背景

校园里有一排线性排列的树木，总共有 $N$ 棵，编号从 $1$ 到 $N$。每棵树都有一个固定的前期维护费用。由于维护费用是按区域统计的，学校的园艺人员需要快速计算出某一段连续区间内所有树木的维护费用总和。
# 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
N = int(next(it))
M = int(next(it))
a = [0]+[int(next(it)) for _ in range(N)]
sum = [0]*(N+1) 
for i in range(1,N+1):
  sum[i]=sum[i-1]+a[i]
for _ in range(M):
  L=int(next(it))
  R=int(next(it))
  print(sum[R]-sum[L-1])
```
