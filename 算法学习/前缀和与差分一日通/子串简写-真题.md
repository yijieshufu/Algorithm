# 题目
### 题目名：新式字符串缩写计数

**1. 规则：**

- **缩写逻辑：** 只保留首尾字符，中间用长度代替。例如 `internationalization` → `i18n`。
    
- **门槛：** 只有长度 **$\ge K$** 的字符串才配被缩写。
    
- **任务：** 给定大字符串 $S$ 和两个字符 $c_1, c_2$。求 $S$ 中有多少个子串满足：以 $c_1$ 开头、以 $c_2$ 结尾，且长度至少为 $K$。

**2. 约束：**

- $K \le |S| \le 5 \times 10^5$（这意味着 $O(N^2)$ 的暴力枚举子串必死，必须用 $O(N)$ 的算法）。
    
- 所有字符均为小写字母。
# 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
# 2. 获取参数
K = int(next(it))
S = next(it)
c1 = next(it)
c2 = next(it)
n = len(S)
ans = 0
cnt_c1= 0 
for j in range(n):
  i = j-K+1
  if i >=0 and S[i]==c1:
    cnt_c1+=1
  if S[j]==c2:
    ans+=cnt_c1
print(ans)
```
