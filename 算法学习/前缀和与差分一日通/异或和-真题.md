# 题目
- **核心动作**：给定序列 $a$，求所有数对 $(a_i, a_j)$ 的**异或值**与其**下标差**乘积的总和。
- **目标公式**：$\sum_{i=1}^{n} \sum_{j=i+1}^{n} (a_i \oplus a_j) \times (j - i)$
- **数据边界**：$n$ 通常为 $10^5$ 级，$a_i$ 约 $10^9$。直接 $O(n^2)$ 暴力必 TLE，需优化至 $O(n \log A)$
```python
# 暴力 只能过 8/12
import os
import sys
it = iter(sys.stdin.read().split())
n = int(next(it))
a = [0]+[int(next(it)) for _ in range(n)]
res =0
for i in range(1,n+1):
  for j in range(i+1,n+1):
    res+=(a[i]^a[j])*(j-i)
print(res)
```
# 分析过程
- **算法/数据结构**：**按位贡献法**。
- **为何适用**：异或运算（$\oplus$）在每一位上是独立的。只要算出第 $k$ 位对最终答案贡献了多少个 $2^k$，最后求和即可。
- **通用骨架**：
```python
  for k in range(31): # 遍历每一位
        for x in a:
            bit = (x >> k) & 1
            # 统计当前位为 0 或 1 时的某种累加状态
```

- **定级**：中等题（省赛高频考点）。
- **核心难点**：如何处理乘法项 $(j - i)$。如果只是求 $\sum (a_i \oplus a_j)$，只需数 0 和 1 的个数；多了下标差，需要利用求和号的**线性性质**拆项。
#### [块 1：公式拆解]
异或的本质是二进制位不相同则为 1。我们将原式拆解到每一位 $k$ 上：  
$$Ans = \sum_{k=0}^{30} 2^k \times \left( \sum_{i < j, \text{bit}_k(a_i) \neq \text{bit}_k(a_j)} (j - i) \right)$$  
这意味着对于每一位，我们只需要关注那些“第 $k$ 位不同”的下标对 $(i, j)$，并累加它们的距离 $(j - i)$。
#### [块 2：动态维护贡献]
对于固定的位 $k$，当我们遍历到下标 $j$ 时，如果 $a_j$ 的第 $k$ 位是 $1$，那么所有之前出现的第 $k$ 位为 $0$ 的下标 $i$ 都会产生贡献：  
$$\text{Contribution} = \sum_{i < j, \text{bit}_k(a_i)=0} (j - i) = j \times (\text{0 的个数}) - (\text{0 的下标之和})$$  
反之，如果 $a_j$ 的第 $k$ 位是 $0$，则贡献为：  
$$\text{Contribution} = j \times (\text{1 的个数}) - (\text{1 的下标之和})$$
#### [块 3：状态变量]
为了实现 $O(n)$ 遍历，我们需要实时维护四个变量：
- `c0`, `s0`: 当前位置之前，第 $k$ 位为 $0$ 的元素的**个数**与**下标和**。
- `c1`, `s1`: 当前位置之前，第 $k$ 位为 $1$ 的元素的**个数**与**下标和**。
