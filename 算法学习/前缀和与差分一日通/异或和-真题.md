# 题目
### 1. 核心任务

你需要计算序列中所有“不重复下标对”的特定运算结果之和。公式表达为：

$$\sum_{i=1}^{n} \sum_{j=i+1}^{n} (a_i \oplus a_j) \times (j - i)$$

### 2. 条件拆解

- **输入数据**：一个长度为 $n$ 的正整数数组 $a$。
    
- **计算对象**：对于数组中任意两个位置 $i$ 和 $j$（假设 $j > i$）：
    
    1. 计算两数之值的**按位异或**：$(a_i \oplus a_j)$。
        
    2. 计算两数之位置的**下标差**：$(j - i)$。
        
    3. 将上述两个结果**相乘**。
        
- **最终结果**：将所有可能的下标对 $(i, j)$ 的乘积结果全部累加。
# 代码
```python
# 暴力 只能过 8/12
import os
import sys
it = iter(sys.stdin.read().split())
n = int(next(it))
a = [0]+[int(next(it)) for _ in range(n)]
res =0
for i in range(1,n+1):
  for j in range(i+1,n+1):
    res+=(a[i]^a[j])*(j-i)
print(res)
```

```python
import sys
# 1. 快速读入
it = iter(sys.stdin.read().split())
n = int(next(it))
a = [0] + [int(next(it)) for _ in range(n)]
ans = 0
# 2. 遍历每一位（通常 30 位足够）
for k in range(31):
  c0, s0 = 0, 0  # 第 k 位为 0 的：个数、下标和
  c1, s1 = 0, 0  # 第 k 位为 1 的：个数、下标和
  bit_total = 0
  for j in range(1,n+1):
    if(a[j]>>k)&1:
      bit_total += (j * c0 - s0)
      c1+=1
      s1+=j
    else:
      bit_total += (j * c1 - s1)
      c0+=1
      s0+=j
  ans += bit_total * (1<<k)
print(ans)
```
