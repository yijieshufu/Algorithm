# 题目描述
**题目描述**：

小蓝在数轴上的 $0$ 点出发，要在总移动距离不超过 $m$ 的前提下，尽可能多地收集矿石。数轴上有 $n$ 个矿石，位置在 $a_i$。经过矿石点即收集，每个矿石只能收集一次。

**约束条件**：

- $n \leq 10^5$（需要 $O(N)$ 或 $O(N \log N)$ 的算法）。
    
- $m \leq 2 \times 10^6$。
    
- $a_i \in [-10^6, 10^6]$。
    
- **关键点**：小蓝可以向左走再掉头向右，或者向右走再掉头向左。
# 代码
```python
import os
import sys
# 请在此输入您的代码
it = iter(sys.stdin.read().split())
n = int(next(it))
m = int(next(it))
a = [int(next(it)) for _ in range(n)]
neg = sorted([-x for x in a if x<0])
pos = sorted([x for x in a if x>=0])
ans = 0 
j = len(pos)
for i in range(len(neg)+1):
  L = neg[i-1] if i>0 else 0 
  if L>m:
    break
  while j >0:
    R = pos[j-1]
    dist = min(2*L+R, 2*R+L)
    if dist<=m:
      break
    j-=1
  ans =max(ans,i+j)
print(ans)
```
