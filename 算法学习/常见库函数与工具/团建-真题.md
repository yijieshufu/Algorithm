# 题目描述

**1. 问题核心 (Core Problem)**  
给定两棵带权树（根节点均为 1），每人从各自树的根节点出发走向某个叶子节点，路径上经过的权值构成一个序列。求两人序列的**最长公共前缀（LCP）** 的最大长度。  
**2. 关键输入 (Key Inputs)**
- **规模**：树 A 大小 $n$，树 B 大小 $m$。
- **权值**：$c_i$ 为树 A 节点 $i$ 的权值，$d_i$ 为树 B 节点 $i$ 的权值。
- **结构**：分别给出两棵树的 $n-1$ 和 $m-1$ 条无向边。

**3. 核心约束 (Crucial Constraint)**
- **唯一性**：**任意节点的子节点权值互不相同**。这意味着只要确定了下一个权值，在树上的走向就是唯一的，无需回溯或复杂的 DP。
# 代码
```python
import os
import sys

sys.setrecursionlimit(1000000)

it = iter(sys.stdin.read().split())
n=int(next(it))
m=int(next(it))
# 权值
w1 =[0]+[int(next(it)) for _ in range(n)]
w2 =[0]+[int(next(it)) for _ in range(m)]
# 建立邻接表
def build_t(n,w):
  g=[[] for _ in range(n+1)]
  for _ in range(n-1):
    u=int(next(it));v=int(next(it))
    g[u].append(v);g[v].append(u)
  
  t = [{}for _ in range(n+1)]
  def pre(u,p):
    for v in g[u]:
      if v!=p:
        t[u][w[v]]=v # 列表为u的节点里面出入了 key:权重 val:儿子节点编号
        pre(v,u)
  pre(1,-1)
  return t
t1 = build_t(n,w1)
t2 = build_t(m,w2)
def dfs(u,v):
  mx=0
  for val,nxt_u in t1[u].items():
    if val in t2[v]:
      nxt_v =t2[v][val]
      res = dfs(nxt_u,nxt_v)
      if res>mx:mx=res
  return 1+mx
if w1[1]!=w2[1]:
  print(0)
else:
  print(dfs(1,1))
```
