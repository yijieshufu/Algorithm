# 题目描述
- **核心动作**：在 $N \le 100$ 的极小规模下，实现**插入排序**算法对序列进行升序排列。
- **数据边界**：$N \in [2, 100]$，由于 $N$ 极小，$O(N^2)$ 的暴力实现完全可行。
# 分析
- **算法/数据结构**：**插入排序 (Insertion Sort)**。
- **为何适用**：题目明确要求实现该算法，且 $N$ 的范围极小，不要求 $O(N \log N)$ 的高级排序。
```python
for i in range(1, n):
    key = a[i]
    j = i - 1 
    while j >= 0 and a[j] > key:
        a[j + 1] = a[j] # 后移
        j -= 1
    a[j + 1] = key # 插入
```
- **定级**：基础题。
- **核心难点**：在于**手动实现**排序过程而非直接调用 `sort()`，需处理好内部元素的“后移”逻辑。

- **[块 1：建立有序区]**：默认第一个元素是有序的，从第二个元素（下标 1）开始向后遍历。
- **[块 2：空位后移]**：将当前待插入值 `v` 与左侧有序区比较，若左侧值大，则将其向右挪一位，腾出空间。
- **[块 3：精准插入]**：当找到不再大于 `v` 的位置或到达边界时，将 `v` 填入该空位。
