# 题目描述
- **核心动作**：在由大写字母组成的字符串 $S$ 中，找出**出现次数最多**的字符；若有多个，按**字典序**依次输出。
- **数据边界**：$|S| \le 10^6$。必须实现 $O(N)$ 时间复杂度以避免超时（TLE）。
# 分析
- **算法/数据结构**：**频率统计（Hash/计数数组）**。
- **为何适用**：字符集极小（仅 26 个大写字母），使用长度为 26 的数组记录频率是时空最优解。
- **通用骨架**：  
```python
cnt = [0] * 26
for x in data:
    cnt[映射函数(x)] += 1
max_val = max(cnt)
```
- **定级**：**基础题**。
- **核心难点**：大数据量（$10^6$）下的快速输入读取，以及多解时的字典序处理。

- **[块 1：高效读取与映射]**：利用 `sys.stdin.read()` 一次性读入长字符串，通过 `ord(c) - 65` 将 'A'-'Z' 映射到索引 $0-25$。
- **[块 2：全量计数]**：单次遍历字符串，完成 26 个桶的累加。
- **[块 3：极值检索与提取]**：先求出 `max(cnt)`，再按 $0 \rightarrow 25$ 的顺序检查每个桶，确保输出天然有序。
