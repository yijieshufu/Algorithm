# 题目描述
在 $N \times M$ 的房间矩阵中，每个房间有固定的容纳人数。我们需要选定一个房间给老师，并选定若干房间给 $K$ 个学生。定义距离为**切比雪夫距离**：$d = \max(|i_1 - i_2|, |j_1 - j_2|)$。目标是：**找到一个老师的住处，使得覆盖至少 $K+1$ 人（含老师本人）所需的最小正方形半径 $d$ 达到全局最小。**
- **输入：** $T$ 组数据，$N, M$（矩阵规模），$K$（学生数），$A_{i,j}$（容量）。
- **输出：** 最小的最大距离。若总人数不足，输出 -1。
# 分析
1. **切比雪夫距离的性质**：给定中心点 $(i, j)$ 和距离 $d$，所有满足距离 $\leq d$ 的点构成一个**以 $(i, j)$ 为中心、边长为 $2d+1$ 的正方形区域**。
2. **区域求和优化**：频繁查询矩形区域内的总人数，是典型的**二维前缀和**应用场景，可将查询复杂度降至 $O(1)$。
3. **最优化策略**：由于距离 $d$ 具有单调性（$d$ 越大，容纳人数越多），我们可以对每个老师的位置进行**二分查找**距离 $d$，或者直接从小到大枚举 $d$。

- **前缀和公式**：$S[i][j] = A[i][j] + S[i-1][j] + S[i][j-1] - S[i-1][j-1]$。
- **矩形查询**：以 $(i,j)$ 为中心、半径为 $d$ 的正方形，左上角为 $(x1, y1) = (\max(1, i-d), \max(1, j-d))$，右下角为 $(x2, y2) = (\min(N, i+d), \min(M, j+d))$。
- **判定逻辑**：二分搜索 $d$。在当前的 $d$ 下，是否存在至少一个位置 $(i,j)$ 满足 $A[i][j] > 0$（老师得能住进去）且该区域总容量 $\ge K+1$。

# 代码
```python
import sys
# 优化 I/O
it = iter(sys.stdin.read().split())
T_cases = int(next(it))
for _ in range(T_cases):
    N = int(next(it))
    M = int(next(it))
    K = int(next(it))

    grid = []
    total_sum = 0
    for i in range(N):
        row = [int(next(it)) for _ in range(M)]
        grid.append(row) # 一行行存储
        total_sum += sum(row)

    # 基础可行性判定
    if total_sum < K + 1: #total_sum:总的容量
        print("-1")
        continue

    # 预处理二维前缀和
    S = [[0] * (M + 1) for _ in range(N + 1)]
    for i in range(1, N + 1):
        for j in range(1, M + 1):
            S[i][j] = grid[i - 1][j - 1] + S[i - 1][j] + S[i][j - 1] - S[i - 1][j - 1] #注意grid的下标

    def check(d):
        # 检查半径 d 是否能容纳 K+1 人
        for i in range(1, N + 1):
            for j in range(1, M + 1):
                if grid[i - 1][j - 1] > 0:  # 老师入住的房间必须有容量
                    r1, c1 = max(1, i - d), max(1, j - d)
                    r2, c2 = min(N, i + d), min(M, j + d) # 矩形的区域
                    if S[r2][c2] - S[r1 - 1][c2] - S[r2][c1 - 1] + S[r1 - 1][c1 - 1] >= K + 1:
                        return True
        return False
    # 二分查找最小距离 d
    low, high = 0, max(N, M)
    ans = high
    while low <= high:
        mid = (low + high) // 2
        if check(mid):
            ans = mid
            high = mid - 1
        else:
            low = mid + 1
    print(ans)


```
