# 题目描述
在 $N \times M$ 的房间矩阵中，每个房间有固定的容纳人数。我们需要选定一个房间给老师，并选定若干房间给 $K$ 个学生。定义距离为**切比雪夫距离**：$d = \max(|i_1 - i_2|, |j_1 - j_2|)$。目标是：**找到一个老师的住处，使得覆盖至少 $K+1$ 人（含老师本人）所需的最小正方形半径 $d$ 达到全局最小。**
- **输入：** $T$ 组数据，$N, M$（矩阵规模），$K$（学生数），$A_{i,j}$（容量）。
- **输出：** 最小的最大距离。若总人数不足，输出 -1。
# 分析
1. **切比雪夫距离的性质**：给定中心点 $(i, j)$ 和距离 $d$，所有满足距离 $\leq d$ 的点构成一个**以 $(i, j)$ 为中心、边长为 $2d+1$ 的正方形区域**。
2. **区域求和优化**：频繁查询矩形区域内的总人数，是典型的**二维前缀和**应用场景，可将查询复杂度降至 $O(1)$。
3. **最优化策略**：由于距离 $d$ 具有单调性（$d$ 越大，容纳人数越多），我们可以对每个老师的位置进行**二分查找**距离 $d$，或者直接从小到大枚举 $d$。

- **前缀和公式**：$S[i][j] = A[i][j] + S[i-1][j] + S[i][j-1] - S[i-1][j-1]$。
- **矩形查询**：以 $(i,j)$ 为中心、半径为 $d$ 的正方形，左上角为 $(x1, y1) = (\max(1, i-d), \max(1, j-d))$，右下角为 $(x2, y2) = (\min(N, i+d), \min(M, j+d))$。
- **判定逻辑**：二分搜索 $d$。在当前的 $d$ 下，是否存在至少一个位置 $(i,j)$ 满足 $A[i][j] > 0$（老师得能住进去）且该区域总容量 $\ge K+1$。
