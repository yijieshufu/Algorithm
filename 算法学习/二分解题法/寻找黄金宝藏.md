
# 题目描述
给定一个长度为 $n$ 的数字序列，求所有长度不小于 $k$ 的连续子序列中，**中位数**的最大值。 （注：中位数定义为升序排列后，奇数长度 $L$ 取第 $(L/2)+1$ 个，偶数长度 $L$ 取第 $L/2$ 个）。
# 分析
**二分答案 + 前缀和优化检查**
- **为什么使用二分**：中位数的取值具有单调性。
	- **如果我告诉你中位数是 $10$，你能否快速判断原序列里是否存在一个子序列，它的中位数大于等于 $10$？**
		- 如果存在，说明真正的最大中位数可能比 $10$ 还大（往右找）。
		- 如果不存在，说明 $10$ 定高了（往左找）。  
	    这种“判断是否存在”的问题（判定性问题）通常比直接“求解最大值”要容易得多

要判定一个猜测的数字 $mid$ 能否成为中位数，核心在于 **“数人头”**：
1. **判定转化**：
    - 如果一个子序列中，大于等于 $mid$ 的元素个数比小于 $mid$ 的元素个数多，那么该子序列的中位数一定 $\ge mid$。
    - **及格标记法**：令 $a_i \ge mid$ 的数为 $+1$（及格），$a_i < mid$ 的数为 $-1$（不及格）。
    - **核心结论**：只要存在一个长度 $\ge k$ 的区间权重和 $S \ge 1$，则 $mid$ 就是一个可行的中位数候选。
2. **高效查找区间**：
    - 利用前缀和数组 $s$，区间权重和为 $s[j] - s[i]$。
    - 我们需要寻找一对 $i, j$ 满足 $j - i \ge k$ 且 $s[j] - s[i] \ge 1$。
    - **优化技巧**：固定右端点 $j$，记录左侧合法的最小前缀和 `min_pre`。

## 二分中位数的“奇偶归一化”
### 1. 核心背景与判定转化
在二分答案 $x$ 的过程中，我们将原序列转化为权重序列：
- **及格 ($+1$)**：若 $a_i \ge x$
- **不及格 ($-1$)**：若 $a_i < x$  
判定目标：是否存在一个长度 $m \ge k$ 的区间，其权重和 $S$ 满足中位数条件。
### 2. 奇偶定义的统一推导
假设子序列长度为 $m$，及格人数为 $c$，权重总和 $S = c - (m - c) = 2c - m$。
- **情况 A：奇数长度 (以 $m=5$ 为例)**
    - **题目定义**：中位数为第 $\lfloor 5/2 \rfloor + 1 = 3$ 个数。
    - **达标要求**：排序后第 3 位及以后必须及格，即及格人数 $c \ge 3$。
    - **计算总和**：当 $c=3$ 时，$S = 3 - 2 = \mathbf{1}$。
    - **结论**：只要 $S \ge 1$，中位数即达标。
- **情况 B：偶数长度 (以 $m=4$ 为例)**
    - **题目定义**：中位数为第 $4/2 = 2$ 个数。
    - **达标要求**：排序后第 2 位及以后必须及格，即及格人数 $c \ge 3$ (第 2, 3, 4 位)。
    - **计算总和**：当 $c=3$ 时，$S = 3 - 1 = \mathbf{2}$。
    - **结论**：只要 $S \ge 2$，中位数即达标。
### 3. 为什么代码不需要判断奇偶？
观察推导结果：
- **奇数满足时**：$S \in \{1, 3, 5, \dots\}$
- **偶数满足时**：$S \in \{2, 4, \dots\}$

**核心结论**：无论长度 $m$ 是奇是偶，判定条件均可统一为 **$S \ge 1$**。
```Python
# 统一判定代码片段
if s[j] - min_pre >= 1: 
    return True # 只要权重和大于 0，及格人数就足以支撑中位数达标
```
# 代码
```python
import os
import sys
# 读入数据
it = iter(sys.stdin.read().split())
n = int(next(it))
k = int(next(it))
a = [int(next(it)) for _ in range(n)]
# 二分判断函数
def check(mid):
  # 计算前缀和：定义前缀和列表
  s = [0]*(n+1)
  for i in range(n):
    val = 1 if a[i]>=mid else -1
    s[i+1] = s[i]+val
  # 长度不小于k的连续子系列
  min_pre = 0
  for j in range(k,n+1):
    if s[j-k]< min_pre:
      min_pre=s[j-k]
    if s[j]-min_pre>=1:
      return True
  return False
# 二分模板
l,r =1,max(a)
ans =1 
while l<=r:
  mid = (l+r)//2
  if check(mid):
    ans = mid
    l = mid+1
  else:
    r = mid-1
print(ans)
```
