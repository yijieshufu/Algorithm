# 题目描述
用一句话总结题目：在一个 $n \times m$ 的矩阵中，每个位置 $(i, j)$ 的值为 $i \times j$，求所有元素从**小到大**排序后的第 $k$ 个数。
**输入：** 三个整数 $n, m, k$。
**输出：** 第 $k$ 小的数值。
**数据范围：** $n, m \le 5 \times 10^5$，$1 \le k \le n \times m$。
**评估：** 难度：中等考频：高频必备（二分答案经典题）
# 分析
- **复杂度推导**：由于 $n \times m$ 极大，暴力 $O(nm)$ 必死。我们需要一个 $O(n \log(nm))$ 或更优的算法。
- **模型匹配**：**二分答案 (Binary Search on Answer)**。
    - 当我们很难直接求出“第 $k$ 个数是多少”时，可以转而思考：“小于等于 $x$ 的数有多少个？”
    - 如果小于等于 $x$ 的个数 $\ge k$，说明第 $k$ 小的数可能就是 $x$ 或者更小。这个判定过程具有明显的**单调性**。

我们需要写一个函数 `check(x)`，计算在乘法表中，有多少个格子满足 $i \times j \le x$。 对于每一行 $i$ ($1 \le i \le n$)，满足条件的 $j$ 必须满足 $j \le \lfloor x / i \rfloor$。同时由于每行只有 $m$ 列，所以第 $i$ 行满足条件的个数为 $\min(m, \lfloor x / i \rfloor)$。
# 代码
```python
import os
import sys
data = sys.stdin.read().split()
n,m,k=map(int,data)
# 确保n小
if n>m: n, m = m, n
# 统计第i行有多少个小于x的
def chack(x):
  cnt = 0
  for i in range(1,n+1):
    cnt +=min(m,x//i)
  return cnt
# 二分法
l,r=1,n*m
ans =r 
while l<=r:
  mid = (l+r)//2
  if chack(mid)>=k:
    ans = mid
    r =mid-1
  else:
    l =mid+1
print(ans)
```
