
# 题目描述
有 $n$ 个瓶子，第 $i$ 个瓶子装水 $a_i$。瓶子颜色以 $k$ 为周期循环（即 $i$ 与 $i+k, i+2k, \dots$ 同色）。同色瓶子间，水只能从左侧瓶子倒入右侧瓶子。求经过任意次操作后，所有瓶子水量最小值的最大可能值。 **数据范围**：$n, a_i \le 10^5$，$1 \le k \le n$。

# 分析
**二分答案 (Binary Search on Answer)**
- **原因**：题目要求“最小值的最大值”，这是典型的二分搜索标志。
- **单调性**：如果最小值 $X$ 可以达到，那么比 $X$ 小的值一定也能达到。
- **验证逻辑**：由于同色瓶子只能“从左往右”倒水，对于每一组同色瓶子，只要保证其前缀和始终能覆盖每个瓶子分配 $X$ 单位水的要求，该 $X$ 即为合法。

- **二分搜索范围**：最小值 $L=0$，最大值 $R=10^{10}$（或初始数组的最大值，保险起见取大值）。
- **Check 函数实现**：
    - 将瓶子按 $i \pmod k$ 分成 $k$ 组。
    - 对每一组进行遍历。维护一个变量 `surplus`（当前可用盈余）。
    - 遍历到瓶子 $a_i$ 时，`surplus += a[i] - X`。
    - 如果 `surplus < 0`，说明左侧没有足够的水填补当前瓶子达到 $X$ 的缺口，且水不能从右往左倒，因此 $X$ 不合法。
- **结果**：二分找到满足条件的最大 $X$。
