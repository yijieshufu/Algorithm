# 题目描述
给定 $N$ 组冶炼记录 $(A, B)$，满足 $B = \lfloor A / V \rfloor$，求正整数转化率 $V$ 的**最小值**和**最大值**。 数据范围：$N \le 10^4$，$1 \le B \le A \le 10^9$。
# 分析
**数学推导 / 枚举优化**
- **为什么：** 虽然这题可以用二分答案（Binary Search）来寻找边界，但通过简单的数学变形，我们可以直接推导出每一组 $(A, B)$ 对应的 $V$ 的取值范围，从而在 $O(N)$ 时间内解决。二分虽稳，但数学公式更快更直接。
### 解题思路
对于每一组记录 $(A, B)$，我们需要满足：$B \le \frac{A}{V} < B + 1$。  
由此可以推导出 $V$ 的范围：
1. **最大值 $V_{max}$：** 根据 $B \le \frac{A}{V}$，得出 $V \le \frac{A}{B}$。由于 $V$ 是整数，所以 $V_{max} = \lfloor \frac{A}{B} \rfloor$。
2. **最小值 $V_{min}$：** 根据 $\frac{A}{V} < B + 1$，得出 $V > \frac{A}{B + 1}$。由于 $V$ 是整数，所以 $V$ 必须大于 $\frac{A}{B+1}$ 的整数部分，即 $V_{min} = \lfloor \frac{A}{B + 1} \rfloor + 1$。

最后，对所有记录要求的 $V_{min}$ 取**最大值**（满足最严苛的下限），对 $V_{max}$ 取**最小值**（满足最严苛的上限）。

为了同时满足所有记录，转换率 $V$ 的取值必须是所有可行区间的**交集**，即：**下限取最大（最严门槛），上限取最小（最紧封顶）**。
