# 题目描述
小明有 $n$ 种卡牌，每种已有 $a_i$ 张。他有 $m$ 张空白卡，可涂成第 $i$ 种牌（每种最多涂 $b_i$ 张）。问最多能凑出多少套完整的卡牌（每套含全部 $n$ 种牌各一张）。
# 分析
**二分答案**
- **特征**：题目要求“**最大化最小值**”（最多能凑出多少套），且答案具有单调性——如果能凑出 $x$ 套，必然也能凑出 $x-1$ 套。
- **理由**：直接计算最优解较难，但给定一个目标套数 $x$，验证其是否可行（Check函数）仅需 $O(n)$ 时间。

- **确定二分范围**：答案最小为 $0$，最大不会超过 $\max(a_i + b_i)$。根据数据范围，设上限为 $2 \times 2 \cdot 10^5 = 4 \cdot 10^5$（或直接取 $2n+n$）。
- **编写 Check(mid) 函数**：
    - 遍历每种牌 $i$，若当前 $a_i < mid$，则需要补 $mid - a_i$ 张。
    - **限制 A**：补的张数不能超过 $b_i$，否则该 $mid$ 不可行。
    - **限制 B**：所有牌补的总张数之和不能超过 $m$。
- **执行二分**：不断收缩区间，找到满足条件的最大 $mid$。

r 的范围确定：对于任意一种牌，其最大总数 $a_i + b_i \le 2n + 2n = 4n$

## 代码

```python
import sys
# 输入数据
it= iter(sys.stdin.read().split())
n = int(next(it))
m = int(next(it))
a = [int(next(it)) for _ in range(n)]
b = [int(next(it)) for _ in range(n)]
# 二分判断函数
def check(mid):
  # 判断小于套牌mid的
  needed = 0
  for i in range(n):
    if a[i] <mid: 
      need = mid -a[i]
      if need > b[i]:#限制条件 手写牌最多b[i]张
        return False
      needed+=need
      if needed>m:# 限制条件：只有m张空白牌
        return False
  return True
# 二分法
l,r= 1,2*n*2
while l<=r:
  mid = (l+r)//2
  if check(mid):
    ans = mid 
    l = mid+1
  else:
    r = mid-1
print(ans)
```
