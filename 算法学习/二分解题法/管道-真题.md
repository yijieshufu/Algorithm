# 题目描述
在长度为 `len` 的管道上分布着 $n$ 个阀门，每个阀门在 $S_i$ 时刻开启并向两端扩散水流。求最早在哪个时刻 $T$，管道的每一段（从 $1$ 到 `len`）都检测到水流。
# 分析
原因在于**数据范围极大**（$len, S_i$ 均可达 $10^9$），直接模拟每一秒的状态会超时且空间溢出，必须使用 $O(n \log n)$ 级别的算法配合二分。

**二分答案 + 区间合并**
- **二分答案**：时间 $T$ 具有单调性。如果 $T$ 时刻管道已满，那么比 $T$ 大的时刻一定也满；反之亦然。
- **区间合并**：对于给定的时刻 $T$，每个开启的阀门覆盖的范围是一个**闭区间** $[L_i - (T - S_i), L_i + (T - S_i)]$。判断管道是否被填满，等价于判断这些区间的**并集**是否完全覆盖 $[1, len]$。

## 解题思路
### 1. 二分策略 (Binary Search)
- **目标**：寻找满足“全覆盖”条件的**最小时间 $T$**。
- **范围**：$L = 0, R = 2 \times 10^9$（覆盖最晚开启时刻 $10^9$ 与最长扩散长度 $10^9$ 的极端情况）。
### 2. 校验逻辑 `check(T)`
对于指定时刻 $T$，判断管道是否被填满：
1. **区间生成**：遍历阀门，若 $T \ge S_i$，计算覆盖范围 $[L_i - (T - S_i), L_i + (T - S_i)]$，并裁剪至 $[1, len]$ 范围内。
2. **排序**：将所有有效区间按**左端点**升序排列。
3. **合并判定**：
    - 设当前连续覆盖的最右端为 $curr\_r = 0$。
    - 遍历排序后的区间，若**新起点 $L \le curr\_r + 1$**（离散段无缝衔接），则更新 $curr\_r = \max(curr\_r, R)$。
    - **判定标准**：若最终 $curr\_r \ge len$，说明 $T$ 时刻全覆盖成功。




# 笔记

## 代码

```python
import os
import sys

# 读入数据
it = iter(sys.stdin.read().split())
n = int(next(it))
m = int(next(it)) 
a = [(int(next(it)),int(next(it))) for _ in range(n)]
# check 函数
def check(t):
  # 计算区间
  q =[]
  for i in range(n):
    l,s=a[i]
    if t>=s:
      d = t-s
      q.append((max(1,l-d),min(m,l+d)))
  if not q:return False
  q.sort()
  if q[0][0]>1:return False
  curr = q[0][1]
  # 裁剪区间
  for i in range(len(q)):
    if q[i][0]<curr+1:
      curr = max(curr,q[i][1])
    else:
      break
  return curr>=m
# 二分法
l, r, ans = 0, 2000000000, 0
while l<=r:
  mid = (l+r)//2
  if check(mid):
    ans = mid
    r = mid-1
  else:
    l = mid+1
print(ans)
```
