# 题目描述
给定一个包含 $n$ 个点、$m$ 条边的带权有向图（存在自环和重边），计算从 **1 号点**到 **$n$ 号点**的最短路径长度。若不可达则输出 -1。 **数据范围**：$n \le 500$，$m \le 10^5$，边权 $c \le 10^4$。
# 分析
**Dijkstra 算法（堆优化版）**
- **原因**：题目要求单源最短路，且所有边权均为正数（$c \ge 1$）。虽然 $n=500$ 较小，可以使用 $O(n^3)$ 的 Floyd 算法，但考虑到 $m$ 较大且 Dijkstra 是图论中的万能起手式，配合优先队列（堆）可以稳过。

### 解题思路
1. **图的存储**：由于存在重边，如果使用邻接矩阵需保留权值最小的那条；如果使用邻接表（如模板所示），直接追加即可，Dijkstra 会自动选择最优路径。
2. **初始化**：设置 `dist` 数组，除起点 `dist[1] = 0` 外，其余初始化为无穷大。
3. **核心迭代**：
    - 将 `(0, 1)` 放入小根堆。
    - 每次弹出堆顶距离最小的点 `u`。
    - 遍历 `u` 的所有邻接点 `v`，若 `dist[u] + w < dist[v]`，则更新 `dist[v]` 并将其入堆。
4. **判空**：最后检查 `dist[n]` 是否仍为无穷大。
