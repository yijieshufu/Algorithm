
## 1. 题目描述

该题目要求我们在二维平面内，给定 $N$ 个点的坐标 $(x, y)$ 以及一条直线的一般式方程 $Ax + By + C = 0$。我们需要统计这 $N$ 个点中，有多少个点“在直线上”。

**判定标准：**

由于计算机处理浮点数存在误差，题目给出了一个精度阈值：如果一个点到该直线的**距离 $d$ 小于 $10^{-3}$**，则认为该点在直线上。

## 2. 核心考点：点到直线的距离公式

点 $(x_0, y_0)$ 到直线 $Ax + By + C = 0$ 的距离公式为：

$$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$

## 3. 题型判断

- **基本题型：** 计算几何 - 点线位置关系。
    
- **算法复杂度：** $O(N)$。遍历每个点并应用公式计算即可。
    
- **难点：** 主要是对浮点数精度的处理和公式的正确应用。
