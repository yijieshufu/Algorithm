
## 一、 题目背景与模板库

- **题目描述**：给定一棵 $n$ 个结点的有根树（根为 1），支持两种操作：1. 修改某个结点的权值；2. 查询某个结点为根的子树内所有点的权值异或和。
    
- **输入条件**：$n, m \le 100,000$；时限 3s（Python 需注意常数优化）。
    
- **所需积木**：【DFS 序】、【树状数组】。

### 通用积木代码
```Python
# 【模板积木 1】DFS 序 (Manual Stack version for Python)
# 功能：建立映射，使子树节点在 DFS 序列中连续。L[u] 为入戳，R[u] 为出戳。
def get_dfs_order(n, g, root):
    L, R = [0] * (n + 1), [0] * (n + 1)
    timer, stack = 0, [(root, 0)]
    while stack:
        u, p = stack[-1]
        if L[u] == 0:
            timer += 1
            L[u] = timer
            for v in g[u]:
                if v != p: stack.append((v, u))
        else:
            R[u] = timer
            stack.pop()
    return L, R

# 【模板积木 2】树状数组 (XOR version)
# 功能：维护前缀异或和，支持单点修改和区间查询。
def update(tr, i, val, n):
    while i <= n:
        tr[i] ^= val
        i += i & (-i)

def query(tr, i):
    res = 0
    while i > 0:
        res ^= tr[i]
        i -= i & (-i)
    return res
```

## 二、 积木说明书与拼装逻辑

- **积木说明书**：
    
    - **DFS 序**：入参为邻接表和根节点；出参为 `L` 和 `R` 数组。它维护的是树到线性的**映射关系**。
        
    - **树状数组**：入参为下标和权值；维护的是线性数组上的**动态前缀异或和**。
        
- **决策依据**：
    
    - 看到“子树查询”，第一反应是**“化树为线”**。通过 DFS 序，子树被压缩成了一个连续区间 $[L[x], R[x]]$。
        
    - 看到“单点修改 + 区间查询”，且操作是“异或”，符合**结合律**和**逆运算（$a \oplus a = 0$）**，果断引入树状数组。
        
- **拼装过程**：
    
    - 先用 DFS 序拿到每个点的“新位置” $L[x]$。
        
    - 初始化时，将原权值 $a[x]$ 插入到树状数组的 $L[x]$ 位置。
        
    - 查询子树 $x$ 时，转化为查询树状数组区间 $[L[x], R[x]]$。
        
- **复杂度**：时间复杂度 $O((n+m) \log n)$，空间复杂度 $O(n)$。

---

## 三、 我的盲点记录

1. **异或修改逻辑**：不同于加法修改（`val - old`），异或修改目标值 $y$ 需要通过 `old ^ y` 算出增量，利用 `old ^ (old ^ y) = y` 达到修改目的。
    
2. **递归深度**：Python 默认递归深度极小，对于 $10^5$ 规模的树，必须手动设置 `sys.setrecursionlimit` 或使用**手动栈 DFS**（推荐后者，更稳）。
    
3. **映射偏移**：原数组下标是 $x$，但在树状数组中操作的下标必须是 $L[x]$。

## 代码

```python
import sys

sys.setrecursionlimit(200000)

input = sys.stdin.read().split()
it = iter(input)

def get_dfs_order(n, g, root):
    L, R = [0] * (n + 1), [0] * (n + 1)
    timer, stack = 0, [(root, 0)]
    while stack:
        u, p = stack[-1]
        if L[u] == 0:
            timer += 1
            L[u] = timer
            for v in g[u]:
                if v != p: stack.append((v, u))
        else:
            R[u] = timer
            stack.pop()
    return L, R

def update(tr, i, val, n):
    while i <= n:
        tr[i] ^= val
        i += i & (-i)

def query(tr, i):
    res = 0
    while i > 0:
        res ^= tr[i]
        i -= i & (-i)
    return res

def solve():
    n, m = int(next(it)), int(next(it))

    a = [0] * (n + 1)
    for i in range(1, n + 1):
        a[i] = int(next(it))

    g = [[] for _ in range(n + 1)]
    for _ in range(n - 1):
        u, v = int(next(it)), int(next(it))
        g[u].append(v)
        g[v].append(u)

    L, R = get_dfs_order(n, g, 1)

    tr = [0] * (n + 1)
    for i in range(1, n + 1):
        update(tr, L[i], a[i], n)

    res = []
    for _ in range(m):
        op = next(it)
        if op == '1':
            x, y = int(next(it)), int(next(it))
            delta = a[x] ^ y
            update(tr, L[x], delta, n)
            a[x] = y
        else:
            x = int(next(it))
            ans = query(tr, R[x]) ^ query(tr, L[x] - 1)
            res.append(str(ans))

    sys.stdout.write("\n".join(res) + "\n")

solve()
```
