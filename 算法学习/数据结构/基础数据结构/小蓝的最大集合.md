

### 题目背景与模板

**题目描述**：维护一个集合，支持插入、全体加 $x$、全体减 $x$、弹出并输出最大值。

**输入条件**：$n \le 2 \times 10^5$，初始含一个数 $k$。

**题型**：**优先队列（大根堆） + 全局偏移量（Lazy Tag）**。

# 变量说明：h 为堆，d 为全局偏移量 delta
def solve():
    # 示例结构，具体逻辑在范本中体现
    h = []
    d = 0
    # 插入值 v 时：heapq.heappush(h, -(v - d))
    # 弹出最大值时：max_val = -heapq.heappop(h) + d
    pass

# 主流程：处理 I/O
input = sys.stdin.read().split()
it = iter(input)
if it:

---

### 题目与核心思路

1. **逻辑总结**：
    
    - **核心矛盾**：全体修改（操作 2, 3）如果 $O(n)$ 遍历堆会超时。
        
    - **解决方案**：维护一个全局变量 $d$ 记录“水位高度”。
        
    - **插入同步**：新元素 $x$ 进堆前，先减去当前水位 $d$，即存入 $x-d$。这样当 $d$ 变化时，它与老元素同步受影响。
        
    - **还原输出**：堆顶弹出的值是“相对高度”，输出时需加上当前水位 $d$ 还原为“绝对高度”。
        
2. **复杂度**：
    
    - 时间复杂度：$O(n \log n)$，每次堆操作均为对数级。
        
    - 空间复杂度：$O(n)$，存储所有插入的元素。
        

---

### 我的盲点记录

- **Python 堆特性**：`heapq` 默认是小根堆。为了实现大根堆，存储时必须取负号 `-val`，弹出时再取负还原。
    
- **初始值同步**：初始给定的 $k$ 也要视作一次插入，如果此时 $d \neq 0$，同样需要执行 $k-d$（本题初始 $d=0$ 可简化，但逻辑上需统一）。

### 最终范本代码

```Python
import sys
import heapq

# 严格执行 Python 3.7.3 规范与极简命名
input = sys.stdin.read().split()
it = iter(input)

def solve():
    n = int(next(it))
    k = int(next(it))
    
    h = []
    d = 0 # 全局偏移量 delta
    
    # 初始元素处理
    heapq.heappush(h, -k) # -> 初始 d=0，简写为 -k
    
    for _ in range(n):
        op = next(it)
        if op == '1':
            x = int(next(it))
            heapq.heappush(h, -(x - d)) # -> 逆向抵消当前偏移量，存入相对值
        elif op == '2':
            d += int(next(it)) # -> 直接修改全局偏移量
        elif op == '3':
            d -= int(next(it)) # -> 直接修改全局偏移量
        elif op == '4':
            top = -heapq.heappop(h) # -> 弹出相对最大值
            print(top + d) # -> 加上当前偏移量还原为真实值

if it:
    solve()
```
