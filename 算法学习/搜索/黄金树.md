
## 题目与核心思路 🧠

**题目目标**：计算二叉树中“黄金指数”为 $0$ 的所有节点的权重之和。

**核心逻辑**：

1. **定义规则**：
    
    - 根节点指数 $= 0$。
        
    - 左儿子指数 $=$ 父亲指数 $+ 1$。
        
    - 右儿子指数 $=$ 父亲指数 $- 1$。
        
2. **算法选择**：使用 **深度优先搜索 (DFS)**。因为每个节点的黄金指数严格依赖于其父节点，DFS 可以通过递归参数非常自然地完成“父传子”的状态传递。
    
3. **统计方式**：在遍历每个节点时，实时判断其指数是否为 $0$，若是则累加权重。

## 我的盲点记录 💡

这是你在互动中重点突破的地方：

- **遍历方式的选择**：起初对 BFS 和 DFS 的适用场景有模糊。通过讨论明确了：当需要将父节点的状态（如这里的黄金指数）传递给子节点时，**DFS 递归** 更加直观简洁。
    
- **判断时机**：曾考虑在判断子节点时进行指数检查。经过推演发现，为了不漏掉**根节点**，最稳妥的做法是进入 `dfs` 函数后立即检查当前节点的 `current_index`。
    
- **右儿子的计算规则**：右儿子的指数应该是父节点指数 **$- 1$**。在推导过程中纠正了误用 $+1$ 的情况。

## 最终范本代码 🐍

这份代码完全符合你的编码习惯：Python 3.7.3、简洁变量名、1-indexed 数组、使用 `sys.stdin.read`。


```Python
import sys

# 设置递归深度防止深树溢出
sys.setrecursionlimit(200000)

def solve():
    # 使用 iter 处理所有输入
    it = iter(sys.stdin.read().split())
    
    try:
        line1 = next(it)
    except StopIteration:
        return
        
    n = int(line1)
    
    # 权重 w (1-indexed)
    w = [0] + [int(next(it)) for i in range(n)]
    
    # 左右孩子数组 (1-indexed)
    ls = [0] * (n + 1)
    rs = [0] * (n + 1)
    for i in range(1, n + 1):
        ls[i] = int(next(it))
        rs[i] = int(next(it))
        
    ans = 0

    def dfs(u, idx):
        nonlocal ans
        # 1. 检查当前节点
        if idx == 0:
            ans += w[u]
        
        # 2. 递归左儿子 (index + 1)
        if ls[u] != -1:
            dfs(ls[u], idx + 1)
            
        # 3. 递归右儿子 (index - 1)
        if rs[u] != -1:
            dfs(rs[u], idx - 1)

    # 从根节点 1 开始遍历，初始指数为 0
    dfs(1, 0)
    print(ans)

solve()
```

## 边界与坑点 ⚠️

1. **根节点的处理**：必须确保根节点（1号点）也被纳入黄金指数的判断中。
    
2. **递归深度**：对于 $n$ 较大的树（如退化成链表），Python 默认递归深度可能不足，建议使用 `sys.setrecursionlimit`。
    
3. **右减左加**：规则容易记反，右儿子对应的是 $-1$。
    
4. **输入空值**：在某些测评环境下，读取结尾可能存在空行，使用 `try-except` 或判断 `it` 是否耗尽可以增强鲁棒性。
