
### 📝 1. 题目背景与模板

- **题目描述**：图书馆有 $N$ 本书，给出它们的名称。读者进行 $M$ 次询问，每次给出一个字符串 $T$，求出有多少本书的名字是以 $T$ 为前缀开头的。（输入字符串总长度 $\le 2 \times 10^5$）
    
- **输入条件**：第一行 $N$ 和 $M$。接下来 $N$ 行输入书名。再接下来 $M$ 行输入查询前缀 $T$。
    
- **所选题型**：Trie (字典树 / 前缀树)

**Sigma通用模板代码**：
```Python
import sys
input = sys.stdin.read().split()
it = iter(input)

# [Sigma Trie Template]
# m_sz: 题目给定的总字符长度限制 + 5
m_sz = 200005 
tr = [[0] * 26 for _ in range(m_sz)]
cnt = [0] * m_sz # 记录经过该节点的字符串数量（前缀统计）
tot = 1 # 当前已使用的节点编号，根节点为 1

def ins(s):
    global tot
    u = 1
    for char in s:
        c = ord(char) - ord('a')
        if not tr[u][c]:
            tot += 1
            tr[u][c] = tot
        u = tr[u][c]
        cnt[u] += 1 # 维护前缀数量

def qry(s):
    u = 1
    for char in s:
        c = ord(char) - ord('a')
        if not tr[u][c]: return 0 # 路径断了，说明没有以此为前缀的词
        u = tr[u][c]
    return cnt[u] # 返回走到终点节点时的计数值
```

### 🧠 2. 题目与核心思路

- **逻辑总结**：这道题是字典树最经典的应用场景“前缀频次统计”。直接两步走：第一步，连续 $N$ 次调用 `ins()` 将所有书名入树，树节点中的 `cnt` 数组会自动记录每个前缀被覆盖的次数；第二步，连续 $M$ 次调用 `qry()` 搜索目标前缀，直接返回该前缀末尾节点对应的 `cnt` 值。
    
- **时空复杂度**：时间复杂度 $O(\sum |S_i| + \sum |T_j|)$，插入和查询的时间仅与字符串长度相关；空间复杂度 $O(26 \times L)$，其中 $L$ 为所有插入字符串的总长度上限。

### 🔍 3. 我的盲点记录

- **盲点 1：模板与主逻辑未分离**。之前习惯在主函数的循环里把字典树的插入和查询逻辑重新默写一遍（比如写两层 `for` 循环）。这不仅增加了记忆和打字的工作量，还容易把变量名搞混。
    
- **纠正方案**：现在明确了**“黑盒调用”**思想。牢记并固化 `ins(s)` 和 `qry(s)` 模板，在主函数中直接作为工具函数调用，代码瞬间变得极简。

### 💻 4. 最终范本代码

```Python
import sys
input = sys.stdin.read().split()
it = iter(input)

# [Sigma Trie Template] 放在最外层
m_sz = 200005 
tr = [[0] * 26 for _ in range(m_sz)]
cnt = [0] * m_sz 
tot = 1 

def ins(s):
    global tot
    u = 1
    for char in s:
        c = ord(char) - ord('a')
        if not tr[u][c]:
            tot += 1
            tr[u][c] = tot
        u = tr[u][c]
        cnt[u] += 1 

def qry(s):
    u = 1
    for char in s:
        c = ord(char) - ord('a')
        if not tr[u][c]: return 0 
        u = tr[u][c]
    return cnt[u] 

def solve():
    n = int(next(it))
    m = int(next(it))
    
    for _ in range(n):
        ins(next(it)) # [MOD] 直接调用模板完成书名插入
        
    for _ in range(m):
        res = qry(next(it)) # [MOD] 直接调用模板查询前缀频次
        sys.stdout.write(str(res) + '\n')

solve()
```

### 🔄 5. 变体题型简述

- **最长公共前缀 (LCP)**：也就是我们上一次做的那道题。插入逻辑不变，查询时改为顺着树往下走，当 `cnt[u] == 1` 时提前截断并返回深度。
    
- **全字匹配 (精确查找)**：如果题目不是问“前缀”，而是问“有没有哪本书的名字**完全等于**查询词”？
