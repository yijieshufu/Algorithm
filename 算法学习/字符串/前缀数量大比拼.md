
## 1. 题目背景与模板题目描述

- **题目要求**：小秋和小雨各有字符串 $s$ 和 $t$。定义得分为：自己的字符串中包含对方字符串**所有前缀**的数量。得分多者获胜。
    
- **输入条件**：字符串长度 $n \le 10^5$，仅包含小写字母。
    
- **所选题型**：扩展 KMP (ExKMP / Z-algorithm)。
    
- **引用模板**：
    
    > 基础模板用于求模式串 $v$ 的 Z 数组 $a$，以及目标串 $u$ 与 $v$ 的匹配数组 $b$。其中 $b[i]$ 表示 $u[i..n]$ 与 $v$ 的最长公共前缀 (LCP) 长度。
```python
import sys; input=sys.stdin.read().split(); it=iter(input)

# 1. 初始化数据 (根据题目修改 n, m)
# u: 目标串 (Target), v: 模式串 (Pattern)
# 习惯性补空格实现 1-indexed
n = int(next(it))
u = " " + next(it)
v = " " + next(it)
m = len(v) - 1

# 2. 计算模式串 v 的 Z 数组 (a 数组)
a = [0] * (m + 1)
q, w = 0, 0
a[1] = m
for i in range(2, m + 1):
    if i <= w: a[i] = min(a[i - q + 1], w - i + 1)
    while i + a[i] <= m and v[i + a[i]] == v[1 + a[i]]: a[i] += 1
    if i + a[i] - 1 > w: q, w = i, i + a[i] - 1

# 3. 计算目标串 u 与模式串 v 的 LCP (b 数组)
b = [0] * (n + 1)
q, w = 0, 0
for i in range(1, n + 1):
    if i <= w: b[i] = min(a[i - q + 1], w - i + 1)
    while i + b[i] <= n and 1 + b[i] <= m and u[i + b[i]] == v[1 + b[i]]: b[i] += 1
    if i + b[i] - 1 > w: q, w = i, i + b[i] - 1
```

## 2. 题目与核心思路

- **核心逻辑**：
    
    “包含对方前缀的数量”可以转化数学表达。对于字符串 $u$ 中的位置 $i$，如果它与 $v$ 的 LCP 长度为 $L$，则它贡献了 $v$ 的长度为 $1, 2, \dots, L$ 的共 $L$ 个前缀。
    
    因此，总得分即为匹配数组 $b$ 的**所有元素之和**：

    $$


Score = \sum_{i=1}^{n} b[i]

$$
    
- **复杂度分析**：
    
    - **时间复杂度**：$O(n)$。Z 算法通过维护“匹配框” $[q, w]$，保证右边界 $w$ 不回退。
        
    - **空间复杂度**：$O(n)$。需要存储字符串及对应的 $a, b$ 数组。
        

---

## 3. 我的盲点记录

- **匹配框的镜像原理** 🪞：在框内 ($i \le w$)，利用对称性 `a[i] = min(a[i-q+1], w-i+1)`。这里的 `min` 是为了防止越出当前已知的匹配边界 $w$。
    
- **角色对调** 🔄：小秋的得分是 $s$ 匹配 $t$ 的前缀；小雨是 $t$ 匹配 $s$ 的前缀。需要调用两次算法，并正确交换 $u, v$ 的角色。
    
- **1-indexed 偏移** 📍：在 Python 中处理 1-indexed 数组时，需注意 `range` 范围和字符串填充空格。

## 4. 最终范本代码

```Python
import sys; input=sys.stdin.read().split(); it=iter(input)

# [MOD] 初始化数据
n = int(next(it))
u = " " + next(it)
v = " " + next(it)

def get_ans(u, v, n):
    # a: 模式串 v 的 Z 数组; b: 目标串 u 的 LCP 数组
    a = [0] * (n + 1)
    q, w = 0, 0
    a[1] = n
    for i in range(2, n + 1):
        if i <= w: a[i] = min(a[i - q + 1], w - i + 1)
        while i + a[i] <= n and v[i + a[i]] == v[1 + a[i]]: a[i] += 1
        if i + a[i] - 1 > w: q, w = i, i + a[i] - 1
    
    b = [0] * (n + 1)
    q, w = 0, 0
    res = 0
    for i in range(1, n + 1):
        if i <= w: b[i] = min(a[i - q + 1], w - i + 1)
        while i + b[i] <= n and 1 + b[i] <= n and u[i + b[i]] == v[1 + b[i]]: b[i] += 1
        if i + b[i] - 1 > w: q, w = i, i + b[i] - 1
        # [MOD] 累加得分：每一个 LCP 长度代表匹配到的前缀数量
        res += b[i]
    return res

ans_q = get_ans(u, v, n)
ans_y = get_ans(v, u, n)

if ans_q > ans_y: print("Xiaoqiu")
elif ans_y > ans_q: print("Xiaoyu")
else: print("Draw")
```

## 5. 变体题型简述

- **求字符串循环节**：利用 Z 数组判断 $i + a[i] == n + 1$ 且 $n \% (i-1) == 0$。
    
- **最长回文子串 (Manacher)**：Manacher 算法的“回文半径数组”更新逻辑与 Z 算法的匹配框逻辑高度相似。
    
- **子串出现次数统计**：给定模式串 $P$ 和文本串 $T$，求 $P$ 在 $T$ 中作为子串（而非仅前缀）出现的总次数。
