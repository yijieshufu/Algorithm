
### 1. 题目背景与模板

**题目描述**：给定长度为 $n$ 的字符串 $S$，通过无限次循环移位操作，求能产生的不同字符串的数量。

**输入条件**：$n \le 10^6$，字符串仅含小写字母。

**所选题型**：字符串最小正周期 (KMP)

**Sigma 通用模板代码**：

```Python
import sys; input=sys.stdin.read().split(); it=iter(input)
# Minimal Vars: n, s, nxt, i, j, res
n = int(next(it))
s = " " + next(it) # 1-indexed padding
nxt = [0] * (n + 1)
# [TEMPLATE LOGIC: KMP Prefixes]
i = 2; j = 0
while i <= n:
    while j > 0 and s[i] != s[j + 1]: j = nxt[j]
    if s[i] == s[j + 1]: j += 1
    nxt[i] = j
    i += 1
```

### 2. 题目与核心思路

- **逻辑总结**：不同循环位移的数量等价于字符串的**最小正周期**长度。利用 KMP 算法求出整个字符串的最长相等前后缀 $nxt[n]$，则 $k = n - nxt[n]$ 为最小潜在周期。若 $n$ 能被 $k$ 整除，则 $k$ 为答案；否则说明没有更短的完整循环节，答案为 $n$。
    
- **复杂度**：
    
    - **时间复杂度**：$O(n)$，KMP 线性扫描。
        
    - **空间复杂度**：$O(n)$，用于存储 $nxt$ 数组。

### 3. 我的盲点记录

- **KMP `nxt` 数组的理解**：起初不确定 $nxt[n]$ 如何手动计算（它是指最长的相等前后缀长度）。通过 `ababab` 的例子理清了 $nxt[6]=4$。
    
- **周期与整除的关系**：理解了为什么 $n - nxt[n]$ 是潜在周期，以及为什么当 $n$ 不能被其整除时（如 `abcab`），最小正周期只能是 $n$ 本身。

### 4. 最终范本代码

```Python
import sys; input=sys.stdin.read().split(); it=iter(input)
n = int(next(it))
s = " " + next(it) # [MOD] 1-indexed padding
nxt = [0] * (n + 1)
i = 2; j = 0
while i <= n:
    while j > 0 and s[i] != s[j + 1]: j = nxt[j]
    if s[i] == s[j + 1]: j += 1
    nxt[i] = j
    i += 1
# [MOD] Calculate minimal positive period
k = n - nxt[n]
if n % k == 0: # [MOD] Check divisibility
    res = k
else: # [MOD] No shorter period
    res = n
print(res)
```

### 5. 变体题型简述

- **最小表示法**：在所有循环位移中，求字典序最小的那个字符串。
    
- **周期填充**：给定周期 $k$，构造满足条件的特定字符串。
    
- **KMP 扩展**：利用 $nxt$ 数组求字符串中所有出现的循环节及其出现次数。
