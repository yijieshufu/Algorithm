
## 题目与核心思路：

- **目标**：给定字符串 $s$，求其最大循环次数 $res$（即最多由多少个相同子串拼接而成）。
    
- **模型转化**：利用 KMP 的 `next` 数组（前缀函数）。设字符串长度为 $n$，整个串的最长相等前后缀长度为 $m = a[n]$。
    
- **关键定理**：
    
    1. 若字符串存在长度为 $L$ 的循环元，则 $n - a[n]$ 是最小循环元的**候选长度**。
        
    2. 如果 $n$ 能被 $n - a[n]$ 整除，说明该候选长度确实是循环节，此时 $res = n / (n - a[n])$。
        
    3. 如果不能整除，说明没有比原串更小的循环节，此时 $res = 1$。
        
- **复杂度**：时间复杂度 $O(n)$，空间复杂度 $O(n)$。

## 我的盲点记录：

- **$j$ 的回退逻辑**：当 $s[i] \neq s[j+1]$ 时，`j = a[j]` 的含义是利用已经算好的前缀信息，直接跳到上一个“可能匹配的更短前后缀”的位置，避免了暴力重匹配。
    
- **1-indexed 的实现细节**：在 Python 中，通过 `s = ' ' + s` 将字符串变为从下标 1 开始，匹配逻辑中比较的是 $s[i]$ 和 $s[j+1]$。
    
- **整除判定的必要性**：最长相等前后缀 $m$ 存在并不代表一定有循环节。只有当剩余部分 $n-m$ 能被 $n$ 整除时，循环才具备完整性。

## 最终范本代码：

```Python
import sys
# 1. I/O 模式：快速读取
input = sys.stdin.read().split()
it = iter(input)

s = next(it)
n = len(s)
s = ' ' + s # 4. 索引：补位实现 1-indexed
a = [0] * (n + 1) # 2. 风格：极简变量名 a 充当 next 数组

# 构造 next 数组
j = 0
for i in range(2, n + 1):
    while j > 0 and s[i] != s[j + 1]:
        j = a[j] # 核心逻辑：回溯
    if s[i] == s[j + 1]:
        j += 1
    a[i] = j

# 周期性判定
m = a[n]
L = n - m
res = n // L if n % L == 0 else 1 # 5. 逻辑：追求最高效判定
print(res)
```

## 边界与坑点：

- **全等字符串**：如 `aaaaa`，$n=5, a[5]=4, L=1$，结果为 $5$，完全正确。
    
- **无重复单元**：如 `abcde`，$n=5, a[5]=0, L=5$，结果为 $1$，逻辑依然稳健。
    
- **下标越界**：在 `while` 循环和 `if` 比较中，务必注意 $j+1$ 的边界，这也是为什么数组要开到 `n+1` 且 $j$ 初始为 0。
    
- **时间限制**：$|s| \le 10^6$ 时，Python 的 `while` 嵌套在 `for` 中看似慢，但 $j$ 的总增加和回退次数是线性的，总复杂度仍是 $O(n)$，可以稳过。
