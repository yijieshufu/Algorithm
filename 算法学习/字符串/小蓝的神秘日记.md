### 1. 题目背景与模板

**题目描述：** 寻找字符串中最长的“秘密密码”。即在一个给定字符串中，找出两个完全相同且**互不重叠**的最长子串，求该最长子串的长度。

**输入条件：** 输入包含一个字符串 $s$。长度不超过 $10^5$，仅包含小写字母。最大运行时间 1s，最大运行内存 512M。

**所选题型：** 字符串哈希 (String Hashing) + 二分答案 (Binary Search on Answer)

**引用模板：**
```Python
import sys; input=sys.stdin.read().split(); it=iter(input)
s = next(it)
n = len(s)
base = 131
mod = 2**61 - 1
h = [0] * (n + 1)
p = [1] * (n + 1)
for i in range(1, n + 1):
    h[i] = (h[i-1] * base + ord(s[i-1])) % mod
    p[i] = (p[i-1] * base) % mod
```

### 2. 题目与核心思路

- **逻辑总结：** 最长子串长度具备单调性（长度为 $L$ 存在，则 $L-1$ 必然存在）。因此在 $[1, n//2]$ 范围内二分枚举长度 `mid`。在校验 `check(mid)` 时，从左到右滑动窗口，通过滚动哈希在 $O(1)$ 时间内计算子串哈希值 `val`。利用字典 `st` 记录 `val` **首次出现**的结束位置。若后续滑动中再次遇到 `val`，且当前起点满足不重叠数学关系（即 `i - mid >= st[val]`），则校验通过。
    
- **复杂度分析：**
    
    - 时间复杂度：$O(n \log n)$。二分区间大小为 $n/2$，需进行 $O(\log n)$ 次；单次 `check` 线性扫描全字符串需要 $O(n)$。
        
    - 空间复杂度：$O(n)$。需要长度为 $n+1$ 的哈希前缀数组 `h`、次幂数组 `p` 以及最坏情况下大小为 $O(n)$ 的哈希字典 `st`。

### 3. 我的盲点记录

- **哈希模数的选择**：在大数据量（$10^5$）下，不熟悉防碰撞的大质数选择，容易导致单哈希冲突。采用梅森素数 $2^{61}-1$ 是最优解。
    
- **空间换时间的结构选择**：在 Python 中寻找 $O(1)$ 映射关系时，未能第一时间反应出应当使用字典（`dict`）来存储 `哈希值 -> 首次出现位置` 的键值对。
    
- **不重叠的边界判定**：容易混淆当前子串与前一个子串的位置关系。核心推导应为：当前子串终点为 $i$，长度为 $mid$，则当前起点为 $i - mid + 1$。为了不与终点为 $st[val]$ 的前一子串重叠，需满足 $i - mid + 1 > st[val]$，化简即为 `i - mid >= st[val]`。
    
- **贪心保留最早位置**：在字典更新策略上，容易误以为要更新到最新位置。实际上为了给后续可能出现的重复子串留出最大的空间以满足不重叠条件，字典中的位置记录**绝不能更新**，必须只保留第一次（最早）的记录。

### 4. 最终范本代码

```Python
import sys
input = sys.stdin.read().split()
it = iter(input)

s = next(it)
n = len(s)
base = 131
mod = 2**61 - 1

h = [0] * (n + 1)
p = [1] * (n + 1)
for i in range(1, n + 1):
    h[i] = (h[i-1] * base + ord(s[i-1])) % mod
    p[i] = (p[i-1] * base) % mod

ans = 0
low = 1
high = n // 2

while low <= high:
    mid = (low + high) // 2
    ok = 0
    st = {}
    for i in range(mid, n + 1):
        # [MOD] 提取当前长度为 mid 的子串哈希值，去除了 get_hash 函数调用以降低开销
        val = (h[i] - h[i-mid] * p[mid]) % mod
        
        if val in st:
            # [MOD] 核心判定不重叠：当前起点 >= 首次出现的终点
            if i - mid >= st[val]:
                ok = 1
                break
        else:
            # [MOD] 贪心策略：只记录哈希值首次出现的结束位置
            st[val] = i
            
    if ok:
        ans = mid
        low = mid + 1
    else:
        high = mid - 1

# [MOD] 极简输出
print(ans)
```

### 5. 变体题型简述

- **允许重叠的最长重复子串**：代码逻辑基本一致，只需在 `check` 函数中移除 `i - mid >= st[val]` 的不重叠位置判定，只要 `val` 在字典中出现过即返回 `True`。
    
- **多串匹配（最长公共子串）**：给定两个不同的字符串，求它们的最长公共子串。可将两字符串拼接（中间插入未出现过的特殊字符，如 `#`），然后转化为求拼接字符串的长重复子串问题，或者对短串二分长度并记录哈希集，再在长串中校验是否存在相同哈希。
