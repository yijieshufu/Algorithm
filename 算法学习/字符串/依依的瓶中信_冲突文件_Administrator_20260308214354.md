

### 📝 1. 题目背景与模板
- **题目描述**：有 $N$ 个字符串（总长度 $\le 10^5$），求出每个字符串与其他所有字符串的最长公共前缀（LCP）长度。
- **输入条件**：第一行整数 $N$；接下来 $N$ 行每行一个全小写英文字符串。
- **所选题型**：Trie (字典树 / 前缀树)

**Sigma通用模板代码**：

```Python
import sys
input = sys.stdin.read().split()
it = iter(input)

# [Sigma Trie Template]
# m_sz: 题目给定的总字符长度限制 + 5
m_sz = 200005 
tr = [[0] * 26 for _ in range(m_sz)]
cnt = [0] * m_sz # 记录经过该节点的字符串数量（前缀统计）
tot = 1 # 当前已使用的节点编号，根节点为 1

def ins(s):
    global tot
    u = 1
    for char in s:
        c = ord(char) - ord('a')
        if not tr[u][c]:
            tot += 1
            tr[u][c] = tot
        u = tr[u][c]
        cnt[u] += 1 # 维护前缀数量

def qry(s):
    u = 1
    for char in s:
        c = ord(char) - ord('a')
        if not tr[u][c]: return 0 # 路径断了，说明没有以此为前缀的词
        u = tr[u][c]
    return cnt[u] # 返回走到终点节点时的计数值
```

### 🧠 2. 题目与核心思路

- **逻辑总结**：将原模板中的“前缀整体计数”逻辑转换为“最长公共前缀”逻辑。第一遍利用 `ins(s)` 将所有 $N$ 个字符串存入 Trie 树，并在经过的每个节点累加 `cnt`。第二遍改造 `qry(s)` 函数，在向下查找时，只要当前节点的 `cnt > 1` 就说明该前缀被共享，累加长度；一旦 `cnt == 1` 即进入独有分支，立刻 `break`。
    
- **时空复杂度**：时间复杂度 $O(\sum |S_i|)$，仅需对所有字符执行两次线性遍历；空间复杂度 $O(26 \times \sum |S_i|)$，核心开销在 `tr` 二维数组与 `cnt` 数组。
    

### 🔍 3. 我的盲点记录

- **盲点 1：Trie 的底层数组映射**。起初对字典树的基础结构不清晰。现已理解 `tr[u][c] = v` 本质是一个二维数组映射：`u` 代表当前节点号，`c` 是字符转化后的索引（如 'a' 为 0），`v` 是新建或指向的下一个节点编号。
    
- **盲点 2：查询时的截断判定**。之前未完全掌握判断“最长”公共前缀的边界。现明确 `cnt[u] > 1` 是累加前缀长度的充要条件，而一旦遇到 `cnt[u] == 1` 必须使用 `break` 及时跳出，停止无意义的向下遍历。
    

### 💻 4. 最终范本代码



```Python
import sys
input = sys.stdin.read().split()
it = iter(input)

m_sz = 200005 
tr = [[0] * 26 for _ in range(m_sz)]
cnt = [0] * m_sz 
tot = 1 

def ins(s):
    global tot
    u = 1
    for char in s:
        c = ord(char) - ord('a')
        if not tr[u][c]:
            tot += 1
            tr[u][c] = tot
        u = tr[u][c]
        cnt[u] += 1 

# [MOD] 改造查询函数：从“返回末尾计数值”改为“计算最长 cnt>1 的深度”
def qry(s):
    u = 1
    res = 0 # [MOD]
    for char in s:
        c = ord(char) - ord('a')
        u = tr[u][c]
        if cnt[u] > 1: # [MOD] 存在公共前缀
            res += 1
        else:          # [MOD] 进入独有分支，直接截断
            break
    return res         # [MOD]
# [MOD] 核心执行流程
n = int(next(it))
ss = []
for _ in range(n):
    ss.append(next(it))
for s in ss:
    ins(s) # [MOD] 第一遍：构建字典树并全局计数
for s in ss:
    sys.stdout.write(str(qry(s)) + '\n') # [MOD] 第二遍：输出各自的LCP长度
```

### 🔄 5. 变体题型简述

- **前缀频率统计**：即原版 `qry(s)` 模板的直接应用。给出多个查询词，要求快速输出每个查询词在库中作为前缀出现的总次数。
    
- **01-Trie 最大异或对**：不存 26 个小写字母，而是将数字转为二进制存入 `[0] * 2` 的数组。在一组数字中寻找异或值最大的两个数（查询时运用贪心思想，优先走与当前二进制位相反的路径）。
