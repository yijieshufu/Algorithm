
### 1. 题目背景与模板

- **题目描述**：给定一个长度为 $N$ 的字符串，求其中最长回文子串的长度。
    
- **输入条件**：$1 \le |S| \le 5 \times 10^5$，保证只包含大小写字母和数字。
    
- **所选题型**：字符串 / Manacher 算法 🏃
    
- **通用模板代码**：
    


```Python
import sys
input = sys.stdin.read().split()
it = iter(input)

def solve():
    s = next(it)
    # 1. 预处理字符串，填充特殊字符以处理偶数回文
    t = "#" + "#".join(s) + "#"
    t = "^" + t + "$"
    n = len(t) - 2
    p = [0] * (n + 2)
    c = r = 0
    res = 0
    
    for i in range(1, n + 1):
        if i < r:
            p[i] = min(r - i, p[2 * c - i])
        while t[i + p[i] + 1] == t[i - p[i] - 1]:
            p[i] += 1
        if i + p[i] > r:
            c, r = i, i + p[i]
        res = max(res, p[i])
    
    print(res)

solve()
```

### 2. 题目与核心思路

- **逻辑总结**：通过在字符间插入分隔符统一奇偶回文长度。维护已知最右回文边界 `r` 及其中心 `c`，利用对称点 $j = 2c - i$ 的已知半径 $p[j]$ 避免重复扫描，仅在突破边界 `r` 时进行暴力扩展。
    
- **时空复杂度**：
    
    - **时间复杂度**：$O(n)$。由于右边界 `r` 只增不减，`while` 循环中的字符比较总次数不超过处理后字符串的长度。
        
    - **空间复杂度**：$O(n)$。需要存储长度为 $2n+3$ 的处理串 $t$ 和对应的半径数组 `p`。
        

### 3. 我的盲点记录

- **对称中心公式**：$j = 2c - i$ 的本质是中点公式的推导，它使得当前点可以“白嫖”左侧对称点的计算结果。
    
- **边界更新时机**：只有当 $i + p[i] > r$（即探索到了未知领域）时，才需要更新 $c$ 和 $r$。被完全包裹在内的子串无需更新边界。
    
- **原串长度映射**：处理串中以某个字符为中心的回文半径 $p[i]$，其数值正好等同于该回文在原串中的总长度。
    

### 4. 最终范本代码

本题为标准裸题，直接套用模板即可，无需任何结构性修改。

```Python
import sys
input = sys.stdin.read().split()
it = iter(input)

def solve():
    s = next(it)
    
    t = "#" + "#".join(s) + "#"
    t = "^" + t + "$"
    n = len(t) - 2
    p = [0] * (n + 2) 
    c = r = 0
    res = 0
    
    for i in range(1, n + 1):
        if i < r:
            p[i] = min(r - i, p[2 * c - i])
        while t[i + p[i] + 1] == t[i - p[i] - 1]:
            p[i] += 1
        if i + p[i] > r:
            c, r = i, i + p[i]
        res = max(res, p[i])
    
    print(res)

solve()
```

### 5. 变体题型简述

- **提取具体回文串**：在更新 `res` 时，同步记录最大半径对应的中心索引 `max_i`，最后利用 `s[(max_i - res) // 2 : (max_i + res) // 2 - 1]` 截取。
    
- **回文子串计数**：数组 `p` 中每个位置的 $p[i]$ 代表以该点为中心的最大半径，累加 `(p[i] + 1) // 2` 即可得到原串所有的回文子串数量。
