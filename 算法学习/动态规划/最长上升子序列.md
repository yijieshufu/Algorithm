# 二分优化下的最长非严格上升与最长下降子序列模型

```Python
g = []
# 非严格上升：允许 a[j] <= a[i]
if x >= g[-1]: 
	g.append(x)
else:
	idx = bisect.bisect_right(g, x) # 找第一个 > x 的位置替换
	g[idx] = x
	
if not g or x > g[-1]:
		g.append(x)
	else:
		idx = bisect.bisect_left(g, x)
		g[idx] = x
```

|**特性**|**严格上升 (Strictly)**|**不下降 (Non-decreasing)**|
|---|---|---|
|**判定条件**|`x > g[-1]` (只有大于才接在后面)|`x >= g[-1]` (等于也可以接在后面)|
|**二分函数**|`bisect_left`|`bisect_right`|
|**替换逻辑**|找到第一个 **$\ge x$** 的位置并替换|找到第一个 **$> x$** 的位置并替换|
|**目的**|尽可能让 `g` 中的数字**变小**且**不重复**|尽可能让 `g` 中的数字**变小**且**允许重复**|
### 实现代码

```Python
import sys
import bisect

# 统一读取数据
it =iter(sys.stdin.read().split())

n = int(next(it))
a = [0] + [int(next(it)) for x in range(n)]

# --- 变体 1：最长非严格上升子序列 ---
def solve_non_strict():
    g = []
    for i in range(1, n + 1):
        x = a[i]
        if not g or x >= g[-1]: # 允许相等，直接延长
            g.append(x)
        else:
            # 找到第一个 > x 的位置进行替换
            # 这样可以保留前面的相等元素，仅更新比 x 大的最小值
            idx = bisect.bisect_right(g, x)
            g[idx] = x
    return len(g)

# --- 变体 2：最长严格下降子序列 ---
def solve_lds():
    # 技巧：取相反数转化为最长严格上升子序列
    b = [-x for x in a[1:]] 
    g = []
    for x in b:
        if not g or x > g[-1]:
            g.append(x)
        else:
            idx = bisect.bisect_left(g, x)
            g[idx] = x
    return len(g)

# 示例输出（以非严格上升为例）
print(solve_non_strict())
```
