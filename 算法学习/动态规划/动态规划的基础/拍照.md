# 题目分析

题目要求的“美学”标准是：存在一个中心点 $i$，使得左侧是不降序列（$\le$），右侧是不增序列（$\ge$）。

- **数学表达**：$a[1] \le ... \le a[i] \ge a[i+1] \ge ... \ge a[n]$。
    
- **注意点**：题目使用的是 $\le$ 和 $\ge$，这意味着序列中**允许出现身高相同**的同学（非严格递增/递减）
# 代码

```Python
import sys
import bisect

# 1. 快速读入
it = iter(sys.stdin.read().split()) 

n = int(next(it))

a = [0] + [int(next(it)) for _ in range(n)]


# 2. 定义不下降子序列长度数组函数（O(nlogn) 模板）
def get_f(arr):
    # arr 是纯身高数据列表
    g = []
    f = [0] * len(arr)
    for i in range(len(arr)):
        x = arr[i]
        if not g or x >= g[-1]: # 非严格不下降条件
            g.append(x)
            f[i] = len(g)
        else:
            idx = bisect.bisect_right(g, x) # 找第一个 > x 的位置替换
            g[idx] = x
            f[i] = idx + 1
    return f

# 3. 分别计算左右两侧

f1 = [0] + get_f(a[1:])

# 逻辑：将 a[1:] 反转 -> 求不下降子序列 -> 将结果反转回来 -> 补占位符
f2_raw = get_f(a[1:][::-1])[::-1]

f2 = [0] + f2_raw

# 4. 合并结果
ans = 0
for i in range(1, n + 1):
    # a[i] 作为山峰，左右两边的最长长度相加，减去重复计算的自己
    ans = max(ans, f1[i] + f2[i] - 1)

# 输出需要出队的最小人数
print(n - ans)
```
