# 题目分析
### 核心条件分析

- **资源限制：** 有 **2** 台洗车机，可以**同时**工作。
    
- **目标：** 最小化“关闭电源的时间”。因为电源必须在**所有**车洗完后才能关闭，所以时间取决于**耗时最长的那台洗车机**。
    
- **数学抽象：**
    
    设所有车洗车总时间为 $S = \sum t_i$。
    
    我们将这些车分成两组，第一组的总耗时为 $S_1$，第二组的总耗时为 $S_2$。
    
    显然有 $S_1 + S_2 = S$。
    
    我们要最小化的目标值是 $\max(S_1, S_2)$。
    

### 2. 逻辑转化

为了让 $\max(S_1, S_2)$ 最小，直观的感觉是让 $S_1$ 和 $S_2$ 尽量接近。

- 理想情况是 $S_1 = S_2 = S/2$。
    
- 如果不能平分，我们希望其中一组的和尽量接近 $S/2$ 但不超过它（假设是 $S_1 \le S/2$），那么另一组 $S_2 = S - S_1$ 就是我们要找的最小化的最大值。
    

**结论：** 题目转化为：**从 $n$ 个数中选出若干个数，使其总和 $S_1$ 在不超过 $S/2$ 的前提下尽可能大。** 最后的答案就是 $S - S_1$。

### 完整实现代码

```Python
import sys

it=iter(sys.stdin.read().split())
n = int(next(it))
a = [int(next(it)) for _ in range(n)]
s = sum(a)
m = s // 2
# dp[j] 表示容量为 j 的背包能装下的最大洗车时间
# 空间优化：滚动数组
dp = [0] * (m + 1)
for x in a:
    # 逆序遍历确保每个物品只用一次
    for j in range(m, x - 1, -1):
        if dp[j - x] + x > dp[j]:
            dp[j] = dp[j - x] + x
# 较长的一组时间即为答案
print(s - dp[m])
```
