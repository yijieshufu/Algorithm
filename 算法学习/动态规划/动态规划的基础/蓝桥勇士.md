# 题目条件

- **“按顺序阻挡”且“不走回头路”**：这说明我们选择的对手在原序列中的下标必须是**递增**的。也就是说，我们是在原数组中提取一个**子序列**。
    
- **“只愿意挑战战力值越来越高的对手”**：这意味着子序列中的元素必须是**严格单调递增**的（即 $a_{i} < a_{i+1}$，不能等于）。
    
- **“最多会挑战多少名对手”**：目标是求这个严格递增子序列的**最大长度**。

- **基础代码模板**：
    ```Python
    import bisect
    g = [] # g[i] 维护长度为 i+1 的上升子序列末尾元素的最小值
    for x in a:
        if not g or x > g[-1]:
            g.append(x)
        else:
            # 找到第一个大于等于 x 的位置并替换它
            idx = bisect.bisect_left(g, x)
            g[idx] = x
    return len(g)
    ```    

---

### 实现代码
```Python
import sys
import bisect

it=iter(sys.stdin.read().split())
n = int(next(it))
# 数组下标从 1 开始，a[0] 占位
a = [0] + [int(next(it)) for _ in range(n)]

# g 数组存储当前发现的各个长度上升子序列的最小结尾数值
g = []

for i in range(1, n + 1):
    x = a[i]
    if not g or x > g[-1]:
        # 战力值比当前所有记录的结尾都大，直接延长最长子序列
        g.append(x)
    else:
        # 二分查找：找到 g 中第一个大于等于 x 的元素下标
        # 使用 bisect_left 确保“严格上升”逻辑
        idx = bisect.bisect_left(g, x)
        g[idx] = x

# g 数组的长度即为所求的最长上升子序列长度
print(len(g))
```
