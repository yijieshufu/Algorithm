这是一个经典的**最长公共子序列（Longest Common Subsequence, LCS）**问题。这类问题通常使用**动态规划（Dynamic Programming, DP）**来解决。

---

## 什么是“子序列” (Subsequence)？

这是最关键的第一步。很多人会把**子序列**和**子串**混淆：

- **子串 (Substring)：** 必须是**连续**的。比如 `abcde` 的子串有 `abc`、`bcd`。
    
- **子序列 (Subsequence)：** 只要**相对顺序不变**即可，不需要连续。
    
    - 比如序列 `1, 2, 3, 4, 5`
        
    - `1, 3, 5` 是它的子序列（跳着选，但顺序没乱）。
        
    - `5, 1` **不是**子序列（顺序反了）。

**公共子序列**：就是两个序列里都存在的、顺序一致的数字组合。 **最长公共子序列**：就是这些组合里，长度最长的那一个。

## 一、 解题思路分析

### 1. 定义状态

我们使用一个二维数组 $dp[i][j]$，其含义为：**数组 $a$ 的前 $i$ 个元素和数组 $b$ 的前 $j$ 个元素的最长公共子序列的长度**。

### 2. 状态转移方程

对于 $dp[i][j]$，我们需要考虑 $a[i]$ 和 $b[j]$ 的关系：

- **情况一：$a[i] == b[j]$**
    
    说明当前这两个元素可以作为公共子序列的一部分。那么它的长度就是在去掉这两个元素后的基础上加 $1$。

    $$


dp[i][j] = dp[i-1][j-1] + 1

$$
    
- **情况二：$a[i] \neq b[j]$**
    
    说明这两个元素不能同时出现在以 $a[i]$ 和 $b[j]$ 结尾的公共子序列中。我们需要舍弃其中一个，取两种情况的最大值：
    
    1. 舍弃 $a[i]$，看 $a$ 的前 $i-1$ 个与 $b$ 的前 $j$ 个：$dp[i-1][j]$
        
    2. 舍弃 $b[j]$，看 $a$ 的前 $i$ 个与 $b$ 的前 $j-1$ 个：$dp[i][j-1]$
        
        
$$

dp[i][j] = \max(dp[i-1][j], dp[i][j-1])

$$

### 3. 初始化与边界

- $dp[0][j]$ 和 $dp[i][0]$ 均初始化为 $0$，表示其中一个数组为空时，公共子序列长度为 $0$。

## 二、 代码

```Python
# 1. 输入处理
n, m = map(int, input().split())
# 在列表前面加 [0] 是为了让下标从 1 开始，方便对应 dp[i][j]
a = [0] + list(map(int, input().split()))
b = [0] + list(map(int, input().split()))

# 2. 初始化 DP 表
# 创建一个 (n+1) x (m+1) 的二维矩阵，初始值全为 0
dp = [[0] * (m + 1) for i in range(n + 1)]

# 3. 动态规划填表 (计算最大长度)
for i in range(1, n + 1):
    for j in range(1, m + 1):
        if a[i] == b[j]:
            # 如果当前两个元素相等，长度等于左上方对角线的值 + 1
            dp[i][j] = dp[i - 1][j - 1] + 1
        else:
            # 如果不相等，取左方或上方单元格的最大值
            dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

# 输出计算出的最长长度
print(dp[n][m])
```

## 三、 复杂度分析

1. **时间复杂度：** $O(N \times M)$。
    
    我们需要填充一个 $N \times M$ 的矩阵，对于题目中 $10^3$ 的量级，运行速度非常快。
    
2. **空间复杂度：** $O(N \times M)$。
    
    主要消耗在存储 `dp` 矩阵上。
