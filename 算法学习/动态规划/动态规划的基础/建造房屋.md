
# 题目条件分析

1. **街道与位置**：共有 $N$ 条街道，每条街道有 $M$ 个可选位置。
    
2. **建造限制**：每个位置最多建一个房屋。这意味着每条街道上建造的房屋数量 $x_i$ 必须满足 $0 \le x_i \le M$。
    
3. **强制要求**：每条街道上**至少**建一个房屋，即 $1 \le x_i \le M$。
    
4. **预算限制**：总预算为 $K$，每个房屋 1 元。即所有街道房屋总数 $\sum x_i \le K$。
    
5. **方案定义**：根据样例分析（$N=2, M=3, K=5$ 输出 8），这里的“建造方案”是指**每条街道建造房屋的数量组合**，而不是具体位置的组合。
    
    - 若 $x_1=1$，则 $x_2$ 可选 $\{1, 2, 3\}$（3种）
        
    - 若 $x_1=2$，则 $x_2$ 可选 $\{1, 2, 3\}$（3种）
        
    - 若 $x_1=3$，则 $x_2$ 可选 $\{1, 2\}$（2种，因为 $3+3 > 5$）
        
    - 总计：$3 + 3 + 2 = 8$ 种。

### 竞赛级 Python 实现
```Python

import sys

# 竞赛快读
it = iter(sys.stdin.read().split())

n=int(next(it))
m=int(next(it))
k=int(next(it))

mod = 10**9 + 7

# dp[j] 表示当前总房屋数为 j 的方案数
# 初始：0 条街总数为 0 的方案为 1
f = [1] + [0] * k

for _ in range(n):
    # 构建前缀和数组 s，s[i] = sum(f[0...i-1])
    s = [0] * (k + 2)
    for i in range(k + 1):
        s[i+1] = (s[i] + f[i]) % mod
    
    nf = [0] * (k + 1)
    for j in range(1, k + 1):
        # 当前街道选 x \in [1, m], 则前一个状态为 [j-m, j-1]
        l = max(0, j - m)
        r = j - 1
        if l <= r:
            nf[j] = (s[r+1] - s[l] + mod) % mod
    f = nf

# 如果是求“恰好等于 k”：print(f[k]) 
# 如果是求“不超过 k”：print(sum(f) % mod) 
print(sum(f) % mod) 
```
