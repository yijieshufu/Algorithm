# 题目条件
### 1. 核心规则
- **起点与终点**：从第 0 级出发，目标是到达第 $N$ 级。
    
- **移动方式**：每次可以迈 **1 级** 或 **2 级**。
    
- **限制条件**：有 $M$ 个指定的台阶（$a_1, a_2, \dots, a_M$）是坏的，**绝对不能踩上去**。
    
- **结果要求**：求总方案数，并对 $10^9 + 7$ 取模。


### 2. 数学模型：动态规划 (DP)

设 $f[i]$ 为到达第 $i$ 级台阶的方案数。

- **状态转移方程**：
    
    - 如果第 $i$ 级台阶是**坏的**：$f[i] = 0$。
        
    - 如果第 $i$ 级台阶是**好的**：$f[i] = (f[i-1] + f[i-2]) \pmod{10^9+7}$。
        
- **边界条件**：
    
    - $f[0] = 1$（站在起点算 1 种方案）。
        
    - $f[1]$：如果第 1 级没坏，$f[1] = 1$；如果坏了，$f[1] = 0$。



- **基础模板**（滚动数组骨架）：

    ```Python
    f0, f1 = 0, 1 # f0 为 dp[i-2], f1 为 dp[i-1] (初始化 dp[0]=1)
    for i in range(1, n + 1):
        cur = (f0 + f1) % mod if not bad[i] else 0
        f0, f1 = f1, cur
    ```

# 代码

```Python
import sys

# 高效读取所有输入
it=iter(sys.stdin.read().split())

n = int(next(it))
m = int(next(it))


# 使用布尔数组记录损坏台阶，下标从 1 开始
bad = [False] * (n + 1)
for _ in range(m):
	bad[int(next(it))] = True
	
mod = 10**9 + 7

# f0 对应 dp[i-2], f1 对应 dp[i-1]
# 初始状态：dp[0] = 1, 假设存在 dp[-1] = 0
f0, f1 = 0, 1 

# 迭代计算从第 1 级到第 n 级
for i in range(1, n + 1):
	if bad[i]:
		cur = 0
	else:
		cur = (f0 + f1) % mod
	
	# 状态滚动更新
	f0 = f1
	f1 = cur
	
print(f1)

```

# 线性 DP 常见变体实现

在理解了上述核心逻辑后，针对蓝桥杯可能出现的变体，可以快速套用并修改如下：

#### 变体 1：带权重的最小代价（Min-Max DP）

**场景**：每级台阶有代价 $a[i]$，求到达顶端的最小总和。
```Python
# a 为代价数组，下标从 1 开始
# f0, f1 分别代表 dp[i-2], dp[i-1] 的最小代价
f0, f1 = 0, 0 # 初始站在 0 级和 1 级（假设 1 级无额外代价）
for i in range(2, n + 1):
    cur = min(f0, f1) + a[i]
    f0, f1 = f1, cur
```

#### 变体 2：状态限制约束（状态机 DP）

**场景**：不能连续迈两次 2 步。需要用元组记录“最后一步是怎么跳的”。

```Python
# f0, f1 存储 [上步跳1阶到达, 上步跳2阶到达]
f0 = [0, 0] # i-2 级
f1 = [1, 0] # i-1 级（第 1 级只能是 1 步上来的）
for i in range(2, n + 1):
    if bad[i]:
        cur = [0, 0]
    else:
        # 当前迈 1 步：前一级迈 1 或 2 均可
        c1 = (f1[0] + f1[1]) % mod
        # 当前迈 2 步：两级前那一跳必须是迈 1 步上来的
        c2 = f0[0]
        cur = [c1, c2]
    f0, f1 = f1, cur
print(sum(f1) % mod)
```

#### 变体 3：多步跳跃优化（前缀和优化 DP）

**场景**：每次可以跳 $1, 2, \dots, K$ 步。当 $K$ 很大时，使用前缀和将 $O(NK)$ 降为 $O(N)$。
```Python
# s[i] 为 dp[0...i] 的前缀和
dp = [0] * (n + 1)
s = [0] * (n + 1)
dp[0] = s[0] = 1
for i in range(1, n + 1):
    if not bad[i]:
        l = max(0, i - k)
        # 区间 [i-k, i-1] 的和
        dp[i] = (s[i-1] - (s[l-1] if l > 0 else 0)) % mod
    s[i] = (s[i-1] + dp[i]) % mod
```
