
## 算法类型

- **核心算法**：线性动态规划 (Dynamic Programming, DP)。
    
- **题目分类**：计数类 DP / 组合递推。
    
- **考点分析**：题目要求在 $n$ 个位置放油桶，且油桶间距至少为 $k$。这种“相邻元素有限制”的计数问题，通常可以通过考察最后一个位置的状态（放或不放）来推导状态转移方程。
    

---

## 最优解思路

1. **状态定义**：设 $f[i]$ 表示在前 $i$ 个位置放置油桶的总方案数。
    
2. **状态转移**：
    
    - **不放置**：若第 $i$ 个位置不放油桶，方案数等于前 $i-1$ 个位置的方案数，即 $f[i-1]$。
        
    - **放置**：若第 $i$ 个位置放油桶，由于要求至少间隔 $k$ 个空位，则前一个油桶最晚只能出现在 $i-k-1$ 的位置。
        
        - 当 $i > k$ 时：方案数为 $f[i-k-1]$。
            
        - 当 $1 \le i \le k$ 时：只能放这一个油桶，方案数为 $1$。
            
3. **[[递推公式]]**：

    $$


f[i] = \begin{cases} f[i-1] + 1 & 1 \le i \le k \\ f[i-1] + f[i-k-1] & i > k \end{cases}

$$
    
4. **初始化**：$f[0] = 1$（表示一个都不放的初始状态）。
    
5. **复杂度**：时间复杂度 $O(n)$，空间复杂度 $O(n)$，对于 $10^6$ 的数据量可在 1s 内通过。
    

---

## Python 3.8.6 代码

```Python
import sys

# 读入 n 和 k
input_data = sys.stdin.read().split()
n, k = int(input_data[0]), int(input_data[1])
MOD = 10**9 + 7

# f[i] 存储前 i 个位置的方案数
f = [1] * (n + 1)

for i in range(1, n + 1):
    if i <= k:
        # 此时只能放 0 个桶或 1 个桶，且 1 个桶可以放在 1~i 的任意位置
        # 根据递推：f[i] = f[i-1] (不放) + 1 (在当前位置 i 放且前面全空)
        f[i] = (f[i-1] + 1) % MOD
    else:
        # 根据递推：f[i] = f[i-1] (不放) + f[i-k-1] (放，且前面预留 k 个空位)
        f[i] = (f[i-1] + f[i-k-1]) % MOD

print(f[n])
```
