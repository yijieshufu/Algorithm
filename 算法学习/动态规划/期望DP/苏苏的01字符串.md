## 1. 题目描述
**输入**: 长度为 $n$ 的字符串 $S$，包含 `'0'`, `'1'`, `'?'`。  
**规则**:
1. `'?'` 有 $50\%$ 概率变成 `'0'`， $50\%$ 概率变成 `'1'`。
2. 定义字符串的**贡献值**为：所有连续 `'1'` 子串长度的**平方和**。
   > 例如 `1011` 的贡献为 $1^2 + 2^2 = 5$。  
**目标**: 求修改后字符串贡献值的**期望**，结果对 $10^9+7$ 取模。

---

## 2. 核心数学模型

这是一道典型的**增量法期望 DP**。我们不需要每次重新计算整个串的平方和，而是计算**每增加一个字符带来的贡献增量**。

### 增量公式推导
假设以位置 $i-1$ 结尾的连续 `'1'` 的长度为 $L$。  
若第 $i$ 位是 `'1'`，则长度变为 $L+1$。  
**贡献的变化量 ($\Delta$)** 为：

$$
\Delta = (L+1)^2 - L^2 = L^2 + 2L + 1 - L^2 = 2L + 1
$$

根据期望的线性性质 $E[A+B] = E[A] + E[B]$，我们只需要维护 $E[L]$ 即可计算 $E[\Delta]$：

$$
E[\Delta] = 2 \cdot E[L] + 1
$$

### 状态定义
我们需要维护两个变量：
1.  **`ans`**: 当前累计的总期望贡献。
2.  **`len`**: 以当前字符结尾的连续 `'1'` 的**期望长度** ($E[L]$)。

> [!NOTE] 为什么不需要 l2 ($E[L^2]$)?  
> 因为本题求的是**平方和**，其增量公式 $(2L+1)$ 只是一次多项式，仅依赖 $E[L]$。  
> 如果题目求**立方和**，增量为 $3L^2+3L+1$，那时才需要维护 $E[L^2]$。

---

## 3. 状态转移方程

设当前处理字符为 $c$：

1.  **$c = '0'$**
    *   连续段中断，期望长度归零。
    *   `len` $\leftarrow 0$
    *   `ans` 不变。

2.  **$c = '1'$**
    *   必然延长，贡献增加 $2 \cdot len + 1$。
    *   `ans` $\leftarrow$ `ans` $+ (2 \cdot len + 1)$
    *   `len` $\leftarrow$ `len` $+ 1$

3.  **$c = '?'$**
    *   $50\%$ 概率为 `'0'` (贡献 0，长度归 0)
    *   $50\%$ 概率为 `'1'` (贡献 $2 \cdot len + 1$，长度 $+1$)
    *   `ans` $\leftarrow$ `ans` $+ \frac{1}{2}(2 \cdot len + 1)$
    *   `len` $\leftarrow$ $\frac{1}{2}(len + 1)$

---

## 4. AC 代码 (Python)

使用 `iter` 和 `sys.stdin.read` 进行极速 I/O，并利用费马小定理求模逆元（除以 2 等同于乘以 `INV2`）。

```python
import sys

def solve():
    # 1. 竞赛级快速读取
    input_data = sys.stdin.read().split()
    if not input_data:
        return
    iterator = iter(input_data)
    
    n = int(next(iterator))
    s = next(iterator)
    
    # 2. 常量定义
    MOD = 10**9 + 7
    INV2 = 500000004 # (MOD + 1) // 2
    
    ans = 0 # 总期望贡献
    l = 0   # 当前结尾连续1的期望长度
    
    # 3. 线性扫描
    for c in s:
        if c == '1':
            # 增量: 2L + 1
            ans = (ans + 2 * l + 1) % MOD
            l = (l + 1) % MOD
            
        elif c == '0':
            l = 0
            # ans 不变
            
        else: # c == '?'
            # 期望增量: 0.5 * (2L + 1)
            term = (2 * l + 1) * INV2 % MOD
            ans = (ans + term) % MOD
            
            # 期望长度: 0.5 * (L + 1)
            l = (l + 1) * INV2 % MOD
            
    print(ans)

if __name__ == "__main__":
    solve()
```

## 5. 总结
*   **拆位思考**：不要试图计算总状态，而是计算单步操作的**增量**。
*   **数学降维**：通过 $E[(L+1)^2 - L^2] = 2E[L] + 1$，将维护平方期望的问题降维成维护线性期望的问题。
*   **概率加权**：对于 `'?'`，利用线性性质直接对 `'0'` 和 `'1'` 的结果进行加权平均。
