# 题目分析

这道题本质上是计算**到达每一个分数的概率**。

- **游戏规则**：分数 $< K$ 时继续抽牌，抽牌范围 $[1, W]$，分数 $\ge K$ 停止。
    
- **状态定义**：$f[i]$ 为得分恰好为 $i$ 的概率。
    
- **转移核心**：想达到 $i$ 分，上一步必须在 $[i-W, i-1]$ 范围内，且上一步分值必须满足 $< K$。
    
- **优化手段**：由于 $f[i]$ 是前一段连续区间的概率和，我们用一个滑动窗口变量 $s$ 来实时维护这个和，避免 $O(W)$ 的遍历

```Python
class Solution:
    def new21Game(self, n: int, k: int, w: int) -> float:
    if k == 0 or n >= k + w: return 1.0
    
    # f[i] 表示达到 i 分的概率
    f = [1.0] + [0.0] * n
    s = 1.0  # 窗口内能继续抽牌的概率总和
    
    for i in range(1, n + 1):
        f[i] = s / w  # 当前分数的概率
        
        # 准备计算下一个 i，维护窗口 s
        if i < k: 
            s += f[i]  # 只有小于 k 的分值能继续贡献
        
        j = i - w
        if j >= 0 and j < k:
            s -= f[j]  # 移出窗口且之前被加进过 s 的才减去
            
    return sum(f[k:])


```
