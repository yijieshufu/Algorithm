# 题目分析 
### 1. 题目类型

- **核心分类**：动态规划 (DP)
    
- **子类型**：概率 DP / 坐标型 DP
    
- **数学背景**：马尔可夫链（状态转移概率固定）
### 2. 条件分析

- **状态空间小**：棋盘大小 $n \le 25$，步数 $k \le 100$。总状态数为 $100 \times 25 \times 25 = 62,500$，远小于算法竞赛常见的 $10^7$ 限制，适合 DP。
    
- **无后效性**：第 $i$ 步的状态只取决于第 $i-1$ 步的位置。
    
- **概率对称性**：每走一步有 8 个方向，每个方向概率均为 $1/8$。留在棋盘上的总概率 = $\sum(\text{到达棋盘内各格子的概率})$。

```Python
class Solution:
    def knightProbability(self, n: int, k: int, r: int, c: int) -> float:
        # f 表示当前步每个格子上的概率
        f = [[0] * n for _ in range(n)]
        f[r][c] = 1
        
        # 8 个移动方向
        dirs = [(-2, -1), (-2, 1), (-1, -2), (-1, 2), (1, -2), (1, 2), (2, -1), (2, 1)]
        
        for _ in range(k):
            nf = [[0] * n for _ in range(n)] # 滚动数组优化空间
            for i in range(n):
                for j in range(n):
                    if f[i][j] > 0:
                        for dr, dc in dirs:
                            ni, nj = i + dr, j + dc
                            if 0 <= ni < n and 0 <= nj < n:
                                nf[ni][nj] += f[i][j] / 8.0
            f = nf
            
        # 留在棋盘上的总概率即为所有格子概率之和
        return sum(map(sum, f))
```
