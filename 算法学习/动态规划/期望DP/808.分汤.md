# 题目分析
### 1. 题目描述与条件

- **初始状态**：汤 A 和 汤 B 各有 $n$ ml。
    
- **四种操作**：每轮等概率 ($0.25$) 执行以下之一：
    
    1. A-100, B-0
        
    2. A-75, B-25
        
    3. A-50, B-50
        
    4. A-25, B-75
        
- **终止规则**：任一汤被取完即停止。若剩余不足，取完剩余全部。
    
- **目标结果**：返回 $P(\text{A 先完}) + 0.5 \times P(\text{A, B 同时完})$。
    

---

### 2. 核心思想

- **坐标压缩**：所有操作均为 25 的倍数，令 $n = \lceil n/25 \rceil$。例如 100ml 记为 4 个单位。
    
- **数学观察**：A 的期望消耗（62.5）大于 B（37.5）。
    
- **大数特判**：当 $n$ 极大时，A 先完的概率趋近于 1。在 $n > 4450$ 时，直接返回 $1.0$ 以防内存溢出或超时。
    

---

### 3. 状态转移方程

定义 $f(a, b)$ 为 A 剩余 $a$ 单位、B 剩余 $b$ 单位时的概率：

$$f(a, b) = 0.25 \times [f(a-4, b) + f(a-3, b-1) + f(a-2, b-2) + f(a-1, b-3)]$$

---

### 4. 边界条件

- **A 先完**：$a \le 0, b > 0 \implies$ 返回 $1.0$
    
- **同时完**：$a \le 0, b \le 0 \implies$ 返回 $0.5$
    
- **B 先完**：$a > 0, b \le 0 \implies$ 返回 $0.0$
    

---
# 代码

```Python
class Solution:
    def soupServings(self, n: int) -> float:
        # 大数退化：n 很大时 A 必先完
        if n > 4800: return 1.0
        
        # 向上取整缩放
        m = (n + 24) // 25
        
        # f[i][j] 表示 A 剩 i, B 剩 j
        f = [[0.0] * (m + 1) for _ in range(m + 1)]
        
        # 初始化边界
        f[0][0] = 0.5
        for j in range(1, m + 1): f[0][j] = 1.0
        
        # 迭代状态转移
        for i in range(1, m + 1):
            for j in range(1, m + 1):
                # 对应四种操作：(4,0), (3,1), (2,2), (1,3)
                a1 = f[max(0, i-4)][j]
                a2 = f[max(0, i-3)][max(0, j-1)]
                a3 = f[max(0, i-2)][max(0, j-2)]
                a4 = f[max(0, i-1)][max(0, j-3)]
                f[i][j] = 0.25 * (a1 + a2 + a3 + a4)
        
        return f[m][m]
```
