单调队列:
- 解决**滑动窗口最值**问题
- 作为**优化动态规划（DP）** 的工具。

核心思想是：**“如果一个选手比你小，还比你强，你就可以退役了。”**

# 代码

```Python
from collections import deque

def monotonic_queue(a, n, k):
    """
    a: 数组; n: 长度; k: 窗口大小
    """
    q = deque()  # 存储下标，单调递减
    res = []
    
    for i in range(n):
        # q[-1] 最近进入的下标
        # 如果a[i]比你强，q[-1]换成i
        while q and a[q[-1]] <= a[i]:
            q.pop() #你滚吧
        q.append(i) #你进来
        
        if q[0] <= i - k: #判断老大是不是还在区间里面
            q.popleft()   #不在就左边弹出

        if i >= k - 1:
            res.append(a[q[0]]) #存入当前区间的最大值
    return res
```
