**完全背包 (Complete Knapsack)** 是动态规划中最基础的模型之一。其核心特征是：给定 $n$ 种物品和一个容量为 $m$ 的背包，每种物品的重量为 $w_i$，价值为 $v_i$，且**每种物品都有无限件可用**。目标是选出的物品总重量不超过 $m$，且总价值最大。
# 算法模型：一维滚动数组 DP
- **状态定义**：$f[j]$ 表示当前背包容量为 $j$ 时所能获得的最大价值。
- **转移方程**：$f[j] = \max(f[j], f[j - w] + v)$
- **遍历顺序**：外层遍历物品，内层**正序**遍历容量 $j \in [w, m]$。
    - _注：正序遍历保证了在计算 $f[j]$ 时，$f[j-w]$ 可能已经包含了当前物品，从而实现“无限次”选取。_

## 代码

```python
# 状态数组初始化
f = [0] * (m + 1)
for _ in range(n): # n物品数量
    w = next(it) # 当前物品重量
    v = next(it) # 当前物品价值
    
    # 完全背包核心：容量正序遍历
    # 从 w 开始，保证 j-w 始终合法
    for j in range(w, m + 1): # m背包总容量
        # 竞赛技巧：Python 中 if 判断更新通常比调用 max() 函数更快
        nv = f[j - w] + v
        if nv > f[j]:
            f[j] = nv
print(f[m])
```
