
这道题是经典的**等和子集划分问题**，属于**0/1 背包问题**的一种变形。

---

# 题目条件分析

- **公平分配**：意味着必须将所有宝藏分成两份，且每一份的重量总和**完全相等**。
    
- **不可分割**：每个宝藏是一个整体（$a_i$），不能切开。这决定了我们必须用**整数规划**或**动态规划**来解决，而不是简单的算术平均。
    
- **目标转化**：
    
    1. 计算所有宝藏的总重量 $Sum = \sum a_i$。
        
    2. 如果 $Sum$ 是**奇数**，一定无法平分，直接输出 `no`。
        
    3. 如果 $Sum$ 是**偶数**，问题转化为：能否从 $n$ 件宝藏中选出若干件，使其重量之和恰好等于 $Target = Sum / 2$
```Python
import sys

# 竞赛式大批量读取
it=iter(sys.stdin.read().split()) 
n = int(next(it))
a = [int(next(it)) for _ in range(n)]
s = sum(a)

# 总重为奇数必不可能
if s % 2 != 0:
    print("no")
else:
    m = s // 2
    # f 的第 j 位为 1 表示和 j 可达
    f = 1 
    for x in a:
        f |= (f << x)
    
    # 检查第 m 位是否为 1
    if (f >> m) & 1:
        print("yes")
    else:
        print("no")
```
