# 题目分析

- **本质**：判断是否能选出若干个数，其和恰好等于总和的一半 $s // 2$。
    
- **前置条件**：如果总和 $s$ 是奇数，绝对不可能平分，直接返回 `False`。
    
- **目标**：检查 $f$ 的第 $s // 2$ 位是否为 1。

# 代码

```Python
class Solution:
    def canPartition(self, a: List[int]) -> bool:
        s = sum(a)
        # 如果总和是奇数，无法平分
        if s % 2 != 0: 
            return False
            
        m = s // 2
        f = 1
        for x in a:
            f |= (f << x)
            
        # 检查第 m 位是否可达
        return (f >> m) & 1 == 1
```
