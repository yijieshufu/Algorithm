# 题目分析

- **本质：** 这是一个 **二维 0-1 背包问题**。
    
- **维度：** 1. 背包容量限制 1：`0` 的个数不超过 $m$。
    
    2. 背包容量限制 2：`1` 的个数不超过 $n$。
    
- **目标：** 选出最多的字符串数量（每个字符串的“价值”都是 1）。
    
- **为什么不用 Bitset？** Bitset 主要用于处理“是否可达”或“求和”，而本题需要记录“最大数量”，即 $dp[i][j]$ 存储的是数值而非布尔状态。

```Python
# strs: 输入字符串数组, m: 0的限制, n: 1的限制
# dp[i][j] 表示使用 i 个 0 和 j 个 1 能得到的最大字符串数量
dp = [[0] * (n + 1) for _ in range(m + 1)]

for s in strs:
    # 统计当前字符串的 0 和 1 个数
    a = s.count('0')
    b = s.count('1')
    
    # 二维背包倒序遍历，避免重复计算同一物品
    for i in range(m, a - 1, -1):
        for j in range(n, b - 1, -1):
            # 状态转移：选或不选当前字符串
            if dp[i - a][j - b] + 1 > dp[i][j]:
                dp[i][j] = dp[i - a][j - b] + 1

return dp[m][n]
```
