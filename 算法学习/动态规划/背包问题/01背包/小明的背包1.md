- **背包容量 ($V$):** $1 \le V \le 10^3$。这是你能承载的最大总体积。
    
- **物品数量 ($N$):** $1 \le N \le 10^2$。物品数量不多。
    
- **物品属性:** 每个物品 $i$ 有对应的体积 $w_i$ 和价值 $v_i$。
    
- **决策特点:** 每个物品只有“选”或“不选”两种状态（即 0/1），不能拆分。

```Python
import sys
it = iter(sys.stdin.read().split())
n=int(next(it))
V=int(next(it))
dp = [0] * (V + 1)
for i in range(1, n + 1):
    # 读取当前物品的体积 w 和价值 v
    w=int(next(it))
    v=int(next(it))
    for j in range(V, w - 1, -1):
        dp[j] = max(dp[j], dp[j - w] + v)
print(dp[V])
```
