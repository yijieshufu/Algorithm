这个问题是经典的 **01 背包问题** 的变体。

# 题目分析
### 1. 核心条件分析
- **箱子容量 $V$：** 这是背包的总承重（或总体积）。
    
- **物品数量 $n$：** 每个物品只有一件，且只能选择“装入”或“不装入”，这正是 **0-1 背包** 的特征。
    
- **物品体积 $x$：** 在本题中，物品的“体积”既是它占据的空间（重量），也是它的“价值”。
    
- **目标：** 使箱子剩余空间最小。
    
    - 剩余空间最小 $\iff$ 箱子中装入的物品总体积最大。
        
    - 因此，问题转化为：在总体积不超过 $V$ 的前提下，从 $n$ 个物品中挑选，使得挑选出的物品**体积之和最大**。


### 2. 状态定义与转移

我们可以使用一维数组进行动态规划优化（滚动数组）：

- **状态定义：** 设 $dp[j]$ 表示在容量为 $j$ 的箱子中，最多能装入的物品体积。
    
- **初始状态：** $dp[0 \dots V] = 0$。
    
- **状态转移方程：**
    
    对于每一个物品 $i$（其体积为 $v_i$），我们尝试更新 $dp$ 数组：

    $$

dp[j] = \max(dp[j], dp[j - v_i] + v_i)

$$
    
    其中 $j$ 的范围是从 $V$ 递减到 $v_i$

# 代码
```Python
import sys

# 处理输入
it=iter(sys.stdin.read().split())

V = int(next(it))
n = int(next(it))
a = [int(next(it)) for _ in range(n)]

# f[j] 表示容量为 j 时能装的最大体积
f = [0] * (V + 1)

for v in a:
    # 逆序遍历，确保每个物品只取一次 (01背包)
    for j in range(V, v - 1, -1):
        if f[j - v] + v > f[j]:
            f[j] = f[j - v] + v

# 输出最小剩余空间
print(V - f[V])
```
