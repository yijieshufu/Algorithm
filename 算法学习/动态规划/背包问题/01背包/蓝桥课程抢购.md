这道题是一道结合了**贪心策略**的**变种 0/1 背包问题**。
# 题目 条件分析

### 1. 约束条件拆解

- **消耗时间**：买一门课需要等待 $A$ 分钟，这相当于背包问题中的“体积”或“重量”。
    
- **截止时间**：必须在 $B$ 分钟内（包含 $B$）完成购买。这意味着你的**当前总耗时 $j$ 必须满足 $j \le B$**。
    
- **价值**：含金量 $C$ 就是背包问题中的“价值”。
    
- **互斥性**：一次只能买一门课，且不能中途退出。
    

### 2. 关键点：为什么要排序？

这是这类问题最容易出错的地方。在普通背包中，物品顺序不影响结果；但在有截止时间的问题中，**必须先处理先到期的课程**。

> **贪心策略证明**：
> 
> 假设有两门课 $L_1(A_1, B_1, C_1)$ 和 $L_2(A_2, B_2, C_2)$。如果 $B_1 < B_2$，我们应该优先考虑先买 $L_1$。因为如果你先买了截止时间晚的 $L_2$，可能会导致 $L_1$ 超时；但反过来，先买 $L_1$ 给 $L_2$ 留下的缓冲空间更大。

# 代码
```Python
import sys
def solve():
    # 读入所有数据
    it=iter(sys.stdin.read().split())
    n = int(next(it))
    courses = []
    max_b = 0
    for _ in range(n):
        a = int(next(it))
        b = int(next(it))
        c = int(next(it))
        courses.append((a, b, c))
        if b > max_b:
            max_b = b
    # 关键步骤：按截止时间 B 从小到大排序
    courses.sort(key=lambda x: x[1])
    # dp[j] 表示在第 j 分钟能拿到的最大价值
    dp = [0] * (max_b + 1)
    for a, b, c in courses:
        # 0/1 背包逆序遍历，且时间不能超过该课的截止时间 B
        for j in range(b, a - 1, -1):
            if dp[j - a] + c > dp[j]:
                dp[j] = dp[j - a] + c
    print(max(dp))
solve()
```
