# 题目分析

当你付钱买商品 $i$ 时，需要花费 $c_i$ 元和 $t_i$ 秒。

在结账的这 $t_i$ 秒内，你可以每秒“免费”获得 1 件其他商品。
- **结论：** 付费购买商品 $i$，实际上总共可以覆盖 $1 + t_i$ 件商品。

你的目标是：选择一部分商品付费，使得它们覆盖的总商品数 $\ge n$，且总费用 $c_i$ 之和最小。

---


```Python
import sys

# 提升 I/O 效率
it=iter(sys.stdin.read().split())

# 读取商品数量
n = int(next(it))

# f[j] 表示达到至少 j 件商品容量的最小花费
# 初始化为无穷大，f[0] 为 0
f = [10**18] * (n + 1)
f[0] = 0

for _ in range(n):
	t=int(next(it))
	c=int(next(it))
	# 支付一件商品，总共能覆盖 v 件商品
	v = t + 1
	# 逆序遍历滚动数组，实现 0/1 背包
	for j in range(n, 0, -1):
		# 状态转移：取“当前不买”与“买下这件并从剩余需求状态转移”的最小值
		p = j - v if j - v > 0 else 0
		if f[p] + c < f[j]:
			f[j] = f[p] + c			
print(f[n])
```
