# 题目分析

### 1. 核心逻辑转化：从“碰撞”到“差值”

假设有四块石头 $a, b, c, d$：

- 第一轮 $a$ 和 $b$ 碰撞，剩下 $|a - b|$。
    
- 第二轮 $|a - b|$ 和 $c$ 碰撞，剩下 $||a - b| - c|$。
    
- 以此类推。
    

本质上，无论碰撞顺序如何，最后剩下的重量都可以表示为：

$$\pm a \pm b \pm c \pm d \dots$$

即：把所有石头分成两堆（正号一堆 $S_1$，负号一堆 $S_2$），最后的结果就是两堆重量之差的绝对值 $|S_1 - S_2|$。

### 2. 数学抽象

为了让剩余重量最小，我们要让 $S_1$ 和 $S_2$ 尽量接近。

- 设总重量为 $s$。
    
- 则有 $S_1 + S_2 = s$。
    
- 要最小化 $|S_1 - S_2|$，等价于找到一个子集和 $j$，使其在不超过 $s // 2$ 的前提下尽可能大。
    
想象你有所有的石头，总重量为 $s$。你现在要把这些石头分成**两堆**（对应碰撞时的正号和负号）：

- **第一堆（子集）**：重量之和我们设为 $j$。
    
- **第二堆（剩余）**：因为总重量是 $s$，所以剩下的重量一定是 $s - j$。
    

我们要算的是这两堆之间的**差值**（也就是碰撞后剩下的重量）：

$$\text{差值} = \text{较大的那一堆} - \text{较小的那一堆}$$

假设我们找的 $j$ 是较小的那一堆（即 $j \le s/2$），那么：

$$\text{差值} = (s - j) - j$$

# 代码

```Python
class Solution:
    def lastStoneWeightII(self, stones: List[int]) -> int:
        a = stones
        s = sum(a)
        m = s // 2
        # --- 你的 Bitset 模板 ---
        f = 1
        for x in a:
            f |= (f << x)
        # 从 m 向下找第一个可达的位
        for j in range(m, -1, -1):
            if (f >> j) & 1:
                return s - 2 * j
        # -----------------------
        return 0
```
