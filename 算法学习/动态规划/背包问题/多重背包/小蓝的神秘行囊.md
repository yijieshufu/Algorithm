# 题目条件

在普通的 0/1 背包中，我们只考虑一个维度（如体积）。本题要求同时满足：

1. 总体积 $\le V$
    
2. 总重量 $\le M$
    

由于每件宝物只能取一次，这是一个 0/1 背包的变体。

### 状态定义

设 $f[j][k]$ 表示在当前体积限制为 $j$、重量限制为 $k$ 的情况下，能够获得的最大价值。

### 状态转移方程

对于每一件宝物 $(v_i, m_i, w_i)$：

$$f[j][k] = \max(f[j][k], f[j - v_i][k - m_i] + w_i)$$

其中 $j \ge v_i$ 且 $k \ge m_i$。

# 代码

```Python
import sys
it =iter(sys.stdin.read().split())

# N: 数量, V: 最大容量, M: 最大重量
n, v, m = int(next(it)),int(next(it)),int(next(it))

# 初始化 DP 数组，f[j][k] 表示体积为 j，重量为 k 时的最大价值
f = [[0] * (m + 1) for _ in range(v + 1)]

for _ in range(n):
    vi, mi, wi = int(next(it)),int(next(it)),int(next(it))
    # 逆序遍历，确保 0/1 背包特性（每件物品仅用一次）
    for j in range(v, vi - 1, -1):
        for k in range(m, mi - 1, -1):
            # 状态转移：选或不选当前宝物
            if f[j - vi][k - mi] + wi > f[j][k]:
                f[j][k] = f[j - vi][k - mi] + wi

print(f[v][m])
```
