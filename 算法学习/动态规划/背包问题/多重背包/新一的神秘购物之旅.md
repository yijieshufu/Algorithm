# 题目分析

- **属性**：每种物品有固定的**数量限制** $s$。
    
- **目标**：在总预算（容量） $m$ 限制下，求价值 $w$ 最大化。
    
- **状态定义**：`dp[j]` 表示花费预算为 $j$ 时能获得的最大价值。


```Python
import sys

# 1. 读入数据
it = iter(sys.stdin.read().split())
n, m = int(next(it)), int(next(it))
# a 存储拆分后的所有新物品 (价格, 价值)
a = []

# 2. 二进制拆分阶段
for _ in range(n):
    v, w, s = int(next(it)), int(next(it)), int(next(it))
    
    k = 1
    while k <= s:
        a.append((k * v, k * w))
        s -= k
        k <<= 1
    if s > 0:
        a.append((s * v, s * w))

# 3. 标准 01 背包阶段
f = [0] * (m + 1)
for cv, cw in a:
    # 逆序遍历确保每个拆分出的“组合物品”只用一次
    for j in range(m, cv - 1, -1):
        if f[j - cv] + cw > f[j]:
            f[j] = f[j - cv] + cw

print(f[m])
```
