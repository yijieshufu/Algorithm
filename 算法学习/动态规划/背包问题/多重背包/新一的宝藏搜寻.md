
# 题目分析

### **题目：多重背包问题 (Multiple Knapsack Problem)**

---

#### **1. 核心要素**

- **状态变量**：
    
    - $N$：宝物种类数。
        
    - $V$：背包最大容量。
        
    - $v_i$：第 $i$ 种宝物的体积。
        
    - $w_i$：第 $i$ 种宝物的价值。
        
    - $s_i$：第 $i$ 种宝物的最大数量（限定条件）。
        
- **目标**：在总体积 $\le V$ 的前提下，使总价值最大。

#### **2. 逻辑关系**

- **01 背包**：每种物品只能选 $0$ 或 $1$ 个。
    
- **完全背包**：每种物品可以选无限个。
    
- **多重背包**：每种物品可以选 $0, 1, 2, \dots, s_i$ 个。

#### **3. 数据范围与限制**

- $N \le 1000, V \le 2000$。
    
- $v_i, w_i, s_i \le 2000$。
    
- **注意**：若使用朴素的三重循环逻辑 $O(N \times V \times S_{avg})$，计算量约为 $4 \times 10^9$，会超出 Python 的运行时间限制（TLE）。

---

> [!TIP] **复习要点**
> 
> 多重背包的经典优化方式是 **二进制拆分 (Binary Splitting)**，即将 $s_i$ 个物品拆分为 $\log s_i$ 组新的物品，从而将其转化为 **01 背包问题** 求解。

```Python
import sys

# 快速读入所有数据
it=iter(sys.stdin.read().split())
n, m = int(next(it)), int(next(it))
f = [0] * (m + 1)
# 二进制拆分：将多重背包转换为 01 背包
qs = []
for _ in range(n):
    v, w, s = int(next(it)),int(next(it)), int(next(it))
    k = 1
    while k <= s:
        qs.append((k * v, k * w))
        s -= k
        k <<= 1
    if s > 0:
        qs.append((s * v, s * w))

# 01 背包状态转移（滚动数组优化）
for v, w in qs:
    for j in range(m, v - 1, -1):
        # 核心转移方程
        f[j]=max(f[j],f[j - v] + w)

print(f[m])
```
