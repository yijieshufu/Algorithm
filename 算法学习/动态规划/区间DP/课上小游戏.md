 # 算法笔记：小蓝的数字合并 (环形区间 DP)

**Tags**: #Algorithm #DP #IntervalDP #RingDP #Python #Lanqiao  
**Difficulty**: Medium-Hard

---

## 核心思路

**物理断环成链**：环形数组复制一份拼在后面，转化为线性区间 DP。

```python
a = [0] + raw_a + raw_a   # 下标从1开始，长度 m = 2n
```

---

## 关键区分 (易错点)

| 概念 | 含义 | 计算 | 用途 |
|:---|:---|:---|:---|
| **数值 `val[i][j]`** | 合并后**剩下**的数字 | `(a × b) % 10` | 区间最终状态 |
| **得分 `dp[i][j]`** | 合并过程**累加**的分数 | `⌊(a × b) / 10⌋` | **目标：求最大** |

> **性质**：`val[i][j]` = 区间所有数乘积的个位数（定值，可 $O(N^2)$ 预处理）

---

## 循环范围推导

### 终点公式
```python
j = i + length - 1          # 数 length 个数，移动 length-1 步
```

### 起点最大值推导
```python
约束：j ≤ m
即：i + length - 1 ≤ m
得：i ≤ m - length + 1      ← i 的最大合法值
```

### 为什么代码写 `m - length + 2`？
Python `range` **左闭右开**，要包含最大值需再 `+1`：
```python
range(1, (m - length + 1) + 1)  =  range(1, m - length + 2)
#      ↑ 公式最大值      ↑ +1因为右开
```

## 考场 Checklist

- [ ] **拼接**：`raw_a + raw_a` 简洁直观，长度 `m = 2n`
- [ ] **预处理 `val`**：避免 $O(N^4)$ 超时
- [ ] **区分运算符**：数值 `% 10`，得分 `// 10`
- [ ] **循环范围**：`range(1, m - length + 2)` 是因为右开要 `+1`
- [ ] **最终答案**：只遍历 `i ∈ [1, n]`，对应 `n` 种断环方式

## 代码

```python
import sys

it = iter(sys.stdin.read().split())
n = int(next(it))
raw_a = [int(next(it)) for _ in range(n)]

# 1. 断环成链 (下标从1开始)
a = [0] + raw_a + raw_a
m = 2 * n

# 2. 预处理 val[i][j] (区间乘积的个位数)
val = [[0] * (m + 1) for _ in range(m + 1)]
for i in range(1, m + 1):
    val[i][i] = a[i]

for length in range(2, m + 1):
    for i in range(1, m - length + 2):      # i_max = m-length+1，右开+1
        j = i + length - 1
        val[i][j] = (val[i][j-1] * a[j]) % 10

# 3. 区间 DP
dp = [[0] * (m + 1) for _ in range(m + 1)]

for length in range(2, n + 1):              # 只需算到长度 n
    for i in range(1, m - length + 2):
        j = i + length - 1
        for k in range(i, j):               # 枚举分割点
            score = dp[i][k] + dp[k+1][j] + (val[i][k] * val[k+1][j]) // 10
            dp[i][j] = max(dp[i][j], score)

# 4. 取答案 (n 个断开位置)
ans = max(dp[i][i + n - 1] for i in range(1, n + 1))
print(ans)
```
