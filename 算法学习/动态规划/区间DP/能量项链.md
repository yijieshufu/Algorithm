# 能量项链 - 解题笔记

## 题目特征
| 项目       | 内容                                          |
| :------- | :------------------------------------------ |
| **模型**   | 环形区间合并（石子合并变种）                              |
| **核心操作** | 相邻珠子 `(m,r)` ⊕ `(r,n)` → `(m,n)`，能量 `m×r×n` |
| **关键技巧** | 破环成链：数组复制一倍                                 |

## 状态设计

```python
dp[i][j] = 合并区间[i,j]能获得的最大能量
# 珠子i: (a[i], a[i+1])，珠子j: (a[j], a[j+1])
```

---

## 状态转移方程

```python
dp[i][j] = max{ dp[i][k] + dp[k+1][j] + a[i]×a[k+1]×a[j+1] }
            其中 k ∈ [i, j)
```

**含义**：最后一步将 `[i,k]` 和 `[k+1,j]` 合并，释放能量 = 左头 × 中间 × 右尾

---

## 代码模板

```python
def solve():
    n = int(input())
    a = list(map(int, input().split()))
    
    # 破环成链
    a = a + a
    m = 2 * n
    
    # 初始化
    dp = [[0] * m for _ in range(m)]
    
    # 区间DP：按长度从小到大枚举
    for length in range(2, n + 1):          # 区间长度（珠子个数）
        for i in range(m - length + 1):     # 起点（防越界：i + length - 1 < m）
            j = i + length - 1               # 终点
            for k in range(i, j):            # 枚举分割点
                dp[i][j] = max(dp[i][j], 
                    dp[i][k] + dp[k+1][j] + a[i] * a[k+1] * a[j+1])
    
    # 环形：找长度为n的区间最大值
    ans = max(dp[i][i+n-1] for i in range(n))
    print(ans)

solve()
```

---

## 关键细节

| 要点                            | 说明                             |
| :---------------------------- | :----------------------------- |
| **为什么 `length` 从2开始**         | 至少需要2颗珠子才能合并                   |
| **为什么 `length ≤ n`**          | 原环只有n颗，破环后找长度为n的连续区间           |
| **边界 `range(m-length+1)`**    | 保证 `j = i+length-1 < m`，防止数组越界 |
| **能量计算 `a[i]*a[k+1]*a[j+1]`** | 左区间头 × 连接点（左尾=右头）× 右区间尾        |


---

## 复杂度

- **时间**：O(N³) — 长度 × 起点 × 分割点
- **空间**：O(N²) — DP二维数组
