# 📝 算法笔记：二维前缀和处理区间包含问题

## 1. 问题建模 🔍

要判断一个列车区间 $[L, R]$ 是否被查询区间 $[l, r]$ **完全包含**，必须同时满足：

1. **起点不能太靠左**：$L \ge l$
    
2. **终点不能太靠右**：$R \le r$
    

### 坐标转化

将每一辆列车视为平面上的一个 **点 $(L, R)$**：

- **横轴 ($x$)**：代表起点 $L$。
    
- **纵轴 ($y$)**：代表终点 $R$。
    
- **约束**：由于 $L \le R$，所有点都分布在直线 $y = x$ 的左上方。
    

---

## 2. 区域查询与公式推导 📐

在坐标系中，满足 $L \in [l, N]$ 且 $R \in [1, r]$ 的点落在特定的矩形区域内。

### 通用二维前缀和公式

求矩形 $(x_1, y_1)$ 到 $(x_2, y_2)$ 的和：

$$
\text{Ans} = f[x_2][y_2] - f[x_1-1][y_2] - f[x_2][y_1-1] + f[x_1-1][y_1-1]
$$

### 本题的简化版

在本题中，由于查询区域总是“贴着”底边（$y_1=1$）和右边（$x_2=N$）：

- $f[N][0]$ 和 $f[l-1][0]$ 均为 $0$。
    
- 公式简化为：

    $$


\boxed{\text{Ans} = f[N][r] - f[l-1][r]}

$$
    

> **直观理解**：拿出所有终点合格的列车（$f[N][r]$），减去其中起点不合格（太靠左）的列车（$f[l-1][r]$）。

---

## 3. Python 实现 (v3.7.3) 🐍

根据简洁命名和 `sys.stdin` 习惯编写：

```Python
import sys

# 快速读入
data = sys.stdin.read().split()
if not data: exit()
it = iter(data)

n = int(next(it))  # 数轴长度
m = int(next(it))  # 列车数量
q = int(next(it))  # 查询数量

# a[L][R] 记录点，f[L][R] 记录前缀和
a = [[0] * (n + 2) for _ in range(n + 2)]
f = [[0] * (n + 2) for _ in range(n + 2)]

# 1. 标记坐标点
for _ in range(m):
    L, R = int(next(it)), int(next(it))
    a[L][R] += 1

# 2. 预处理二维前缀和
for i in range(1, n + 1):
    for j in range(1, n + 1):
        f[i][j] = f[i-1][j] + f[i][j-1] - f[i-1][j-1] + a[i][j]

# 3. 处理查询
res = []
for _ in range(q):
    l, r = int(next(it)), int(next(it))
    # 核心公式：高度对齐，只减宽度
    ans = f[n][r] - f[l-1][r]
    res.append(str(ans))

sys.stdout.write('\n'.join(res) + '\n')
```

---

## 4. 关键点复盘 🧠

- **高度对齐**：公式两项的第二个下标必须都是 $r$，确保我们只在 $R \le r$ 的范围内讨论。
    
- **数组边界**：定义 `n+2` 大小可以优雅地处理 `i-1` 为 0 的情况，无需额外判断。
    
- **复杂度**：预处理 $O(N^2)$，单次查询 $O(1)$。
