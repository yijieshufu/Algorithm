## 🧠 考场 1 分钟判型 (选哪个模板？)

拿到题目，只问自己一个问题：**题目最终求的是什么？**

1.  **求“某种分数的期望”、“序列连续段贡献（如平方和）”**
    $\implies$ **选【模板一：正序增量法】**
    *   *特征*：给你一个序列/字符串，包含随机的问号，从左往右算总分。
2.  **求“天数”、“步数”、“抽卡次数”**
    $\implies$ **选【模板二：倒序步数法】**
    *   *特征*：给你一个起点和终点（或集齐目标），每次操作有不同概率，求到达终点的代价。

## 🟢 【模板一】正序增量法 (求序列总贡献)[[期望DP/苏苏的01字符串]]

### 🛠️ 考场思考路径
1.  **明确变量**：我需要维护几阶期望？（平方和 $\to$ 维护 $E[L]$；立方和 $\to$ 维护 $E[L]$ 和 $E[L^2]$）。
2.  **推导增量**：$(L+1)^k - L^k$ 是多少？（如平方和增量是 $2L+1$）。
3.  **填空**：确定字符时的增量，随机字符时的加权增量。

### 🧩 默写骨架与填空 (Python 3)

```python
import sys

# 【骨架：正序 DP，滚动变量优化】
it = iter(sys.stdin.read().split())
try:
    n = int(next(it))
    s = next(it)
except StopIteration:
    pass

MOD = 10**9 + 7
inv2 = (MOD + 1) // 2  # 50% 概率必备：除以 2 的逆元

ans = 0 # 记录总期望分数
l1 = 0  # 记录以当前结尾的 E[L] (若是立方和，再加个 l2 = 0)

for c in s:
    if c == '1': 
        # ==========================================
        # 【填空区 A：确定发生时的增量】
        # 草稿纸推导: Delta = (L+1)^2 - L^2 = 2L + 1
        delta = (2 * l1 + 1) % MOD
        # 核心顺序：先用旧 l1 更新 ans，再更新 l1
        ans = (ans + delta) % MOD
        l1 = (l1 + 1) % MOD
        # ==========================================
        
    elif c == '0':
        # 确定中断，长度归零，总分不变
        l1 = 0
        
    else: # c == '?' 
        # ==========================================
        # 【填空区 B：随机发生时的加权转移】
        # 1. 计算若是 1 产生的增量，乘以概率 (1/2)
        delta = (2 * l1 + 1) % MOD
        ans = (ans + delta * inv2) % MOD 
        
        # 2. 状态值本身也要乘以概率
        l1 = (l1 + 1) * inv2 % MOD
        # ==========================================

print(ans)
```

> [!WARNING] 考场必查 (模板一)
> **顺序千万别写反！** 必须先算 `delta` 更新 `ans`，**最后**再 `l1 = l1 + 1`。如果先加了长度，算出来的增量就变成未来的了！

## 🔴 【模板二】倒序步数法 (求达成目标代价)

### 🛠️ 考场思考路径
遇到求步数/天数，在草稿纸上画个节点 $i$，写出裸方程：
$$dp[i] = 1 + P_{stay} \cdot dp[i] + \sum P_{go} \cdot dp[next]$$
**必须在草稿纸上移项！化简成：**
$$dp[i] = \frac{1 + \sum P_{go} \cdot dp[next]}{1 - P_{stay}}$$
提取出分子和分母，然后填入代码。如果涉及区间求和，立刻加上后缀和数组优化。

### 🧩 默写骨架与填空 (Python 3)

```python
import sys

# 【骨架：倒序 DP，后缀和优化，求逆元消自环】
it = iter(sys.stdin.read().split())
try:
    n = int(next(it))
    # 数组下标从 1 开始，方便与题目对齐
    a = [0] + [int(next(it)) for _ in range(n - 1)]
except StopIteration:
    pass

MOD = 998244353
# dp[i] 表示从 i 到目标 n 的期望步数 (终点 dp[n] 默认为 0)
dp = [0] * (n + 2) 
# suf[i] 表示 dp 的后缀和，用于 O(1) 求区间期望和
suf = [0] * (n + 2) 

# 核心：必须从 n-1 倒着推回起点
for i in range(n - 1, 0, -1):
    
    # ==========================================
    # 【填空区 A：去往其他状态的期望和 (sum_go)】
    # 如果是离散状态(如抽卡)，直接手写: P1*dp[x] + P2*dp[y]
    # 如果是连续区间(如飞行棋)，用后缀和: suf[L] - suf[R+1]
    # 注意减法必须加 MOD 防止负数！
    sum_go = (suf[i + 1] - suf[i + a[i] + 1] + MOD) % MOD
    
    # 【填空区 B：代数化简后的分子与分母】
    # 以飞行棋为例：dp[i] = (a[i] + 1 + sum_go) / a[i]
    # 这个公式是你考试前 3 分钟在草稿纸上移项得出的！
    numerator = (a[i] + 1 + sum_go) % MOD  # 分子
    denominator = a[i]                     # 分母
    
    # 【固定动作：分子 * 分母的逆元】
    inv_den = pow(denominator, MOD - 2, MOD)
    dp[i] = (numerator * inv_den) % MOD
    # ==========================================
    
    # 维护后缀和，供 i-1 的时候使用
    suf[i] = (suf[i + 1] + dp[i]) % MOD

# 起点即答案
print(dp[1])
```

> [!WARNING] 考场必查 (模板二)
> 1. **除法必须用逆元**：只要有 `MOD`，绝对不能出现 `/` 符号！全部改成 `* pow(分母, MOD-2, MOD) % MOD`。
> 2. **步数消耗不能忘**：方程的分子里一定包含代表“走了一步/过了一天”的常数（化简前是 $1$，化简后可能是 $a_i+1$ 等）。
> 3. **负数取模安全**：在 Python 中 `(A - B) % MOD` 虽然安全，但为了统一习惯或转 C++ 选手，写成 `(A - B + MOD) % MOD` 绝对不会错。
