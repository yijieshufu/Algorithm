# 01背包的位运算

### 位运算

在 Python 中，嵌套循环效率较低。对于 0/1 背包的**可行性问题**（即只需判断某个和是否可达），可以使用大整数的位运算来极大地加速。

**代码实现：**

```Python
def solve():
	it=iter(sys.stdin.read().split())
	n = int(next(it))
	a = [int(next(it)) for _ in range(n)]
	
    s = sum(a)
    m = s // 2
    
    # f 的第 k 位为 1 表示和 k 可达
    f = 1
    for x in a:
        f |= (f << x)
    
    # 从 s//2 向下找第一个为 1 的位
    for j in range(m, -1, -1):
        if (f >> j) & 1:
            print(s - j)
            break
solve()
```

# 二维 01背包

## 模板代码

```Python
import sys

def solve():
    # 1. 快速读入数据
    it = iter(sys.stdin.read().split())
    
    # 2. 初始化维度限制
    # m, n 通常代表两种不同的容量限制（如：重量、体积；或者 0 的个数、1 的个数）
    m = int(next(it)) 
    n = int(next(it))
    num_items = int(next(it)) # 物品数量
    
    # 3. 初始化 DP 表
    # dp[i][j] 表示在限制 1 为 i，限制 2 为 j 时的最优值
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    # 4. 遍历每个物品
    for _ in range(num_items):
        # a, b 为当前物品的两种“成本”，v 为价值
        a = int(next(it)) 
        b = int(next(it))
        v = int(next(it)) 
        
        # 5. 核心逻辑：二维倒序遍历
        # 必须从容量最大值倒序到物品成本值，以确保每个物品只被选一次
        for i in range(m, a - 1, -1):
            for j in range(n, b - 1, -1):
                # 状态转移方程
                if dp[i - a][j - b] + v > dp[i][j]:
                    dp[i][j] = dp[i - a][j - b] + v
                    
    # 6. 输出最终结果
    print(dp[m][n])

solve()
```
