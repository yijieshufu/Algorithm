
## 1. 题目背景与模板

**题目描述**：$n$ 个玩家进行 $k$ 轮传花游戏。已知每个玩家 $i$ 喜欢的人名单，每轮持花者等概率随机送给其中一人。求 $k$ 轮后各人持花的概率。

**输入条件**：$2 \le n \le 10, 0 \le k \le 10^{18}$，结果对 $10^9 + 7$ 取模。

**所选题型**：矩阵快速幂

## 2. 题目与核心思路

- **逻辑总结**：将概率转移的过程视为线性变换。若玩家 $i$ 传给 $j$ 的概率为 $p_{ij}$，则构建转移矩阵 $A$，使得 $A_{j,i} = p_{ij}$。由于 $k$ 极大，利用状态转移方程 $V_k = A^k V_0$，结合矩阵快速幂在对数时间内求解。
    
- **复杂度分析**：
    
    - **时间复杂度**：$O(n^3 \log k)$。$n=10, k=10^{18}$ 时，计算量约 $6 \times 10^4$ 次，远小于时限要求。
        
    - **空间复杂度**：$O(n^2)$，用于存储状态转移矩阵。

## 3. 我的盲点记录

- **分数取模（逆元处理）**：在模 $10^9+7$ 运算中，除以 $t_i$ 的操作必须转换为乘以 $t_i$ 的模逆元 $t_i^{mod-2} \pmod{mod}$。
    
- **转移矩阵的行列意义**：在 $V_{new} = A \times V_{old}$ 的乘法规则下，矩阵 $A_{j,i}$ 代表从**第 $i$ 列**（转移起点）到**第 $j$ 行**（转移终点）的概率分布，切勿写反。
    
- **初始状态与结果提取**：初始状态只有 1 号玩家有花，对应向量 $V_0 = [1, 0, \dots, 0]^T$。因此最终状态 $A^k V_0$ 实际上就是直接提取结果矩阵 $A^k$ 的**第一列**。

mod = 1000000007

# 核心算法：矩阵乘法 (1-indexed)
def mat_mul(a, b, n):

    for i in range(1, n+1):
        for k in range(1, n+1):
            if not a[i][k]: continue
            for j in range(1, n+1):
                c[i][j] = (c[i][j] + a[i][k] * b[k][j]) % mod
    return c

# 核心算法：矩阵快速幂
def mat_qpow(a, k, n):
    res = [[0]*(n+1) for _ in range(n+1)]
    for i in range(1, n+1): res[i][i] = 1
    while k:
        if k & 1: res = mat_mul(res, a, n)
        a = mat_mul(a, a, n)
        k >>= 1
    return res

def core(n, k, raw_adj): # [MOD]
    a = [[0]*(n+1) for _ in range(n+1)] # [MOD]
    for i in range(1, n+1): # [MOD]
        t = len(raw_adj[i]) # [MOD]
        if t > 0: # [MOD]
            inv = pow(t, mod - 2, mod) # [MOD]
            for v in raw_adj[i]: # [MOD]
                a[v][i] = inv # [MOD]
    return mat_qpow(a, k, n) # [MOD]

n = int(next(it)) # [MOD]
k = int(next(it)) # [MOD]
raw_adj = [[] for _ in range(n+1)] # [MOD]
for i in range(1, n+1): # [MOD]
    t = int(next(it)) # [MOD]
    for _ in range(t): # [MOD]
        raw_adj[i].append(int(next(it))) # [MOD]

res = core(n, k, raw_adj) # [MOD]
for i in range(1, n+1): # [MOD]
    print(res[i][1]) # [MOD]

## 5. 变体题型简述

1. **带权有向图转移**：若图中的边并非等概率选择，而是带有“权重”，将计算逆元的步骤替换为 $\frac{w_{ij}}{\sum w_{i}}$ 即可转化为概率矩阵。
    
2. **固定步数路径计数**：去掉概率计算，将转移矩阵退化为邻接矩阵（连通为 1，不连通为 0）。矩阵 $A^k$ 的元素 $A_{j,i}$ 即表示经过恰好 $k$ 步从点 $i$ 到点 $j$ 的路径方案数。
    
3. **多源多终点统计**：若初始状态有多个玩家持花，直接构建相应的初始一维状态矩阵 $V_0$，最终取 $A^k \times V_0$ 的完整结果向量即可。
