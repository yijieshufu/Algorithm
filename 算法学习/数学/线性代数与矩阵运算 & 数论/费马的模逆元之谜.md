
### 1. 题目背景与模板

**题目描述**：给定两个正整数 $a$ 和 $p$（$p$ 为素数），求 $a$ 在模 $p$ 下的逆元。若 $a$ 是 $p$ 的倍数，则逆元不存在。

**输入条件**：$1 \le a, p \le 10^5$，且 $p$ 是一个素数。

**所选题型**：数论 - 模逆元（快速幂实现）。

### 2. 题目与核心思路

**逻辑总结**：

利用**费马小定理**：若 $p$ 为质数，且 $gcd(a, p) = 1$，则 $a^{p-1} \equiv 1 \pmod p$。

由此推导得 $a \cdot a^{p-2} \equiv 1 \pmod p$，即 $a$ 的逆元为 $a^{p-2} \pmod p$。

若 $a$ 是 $p$ 的倍数（$a \% p == 0$），则 $a$ 与 $p$ 不互质，逆元不存在。

**时空复杂度**：

- **时间复杂度**：$O(\log p)$，快速幂的执行效率。
    
- **空间复杂度**：$O(1)$，仅需常数级变量。
    

---

### 3. 我的盲点记录

- **边界情况**：需注意当 $a$ 为 $p$ 的倍数时，费马小定理的前提条件不成立，必须输出 `Inverse doesn't exist`。
    
- **指数取值**：求逆元时，快速幂的指数固定为 $p-2$。

### 4. 最终范本代码

```Python
import sys; input=sys.stdin.read().split(); it=iter(input);

# 核心算法：快速幂
def qpow(a, k, p):
    res = 1
    a %= p
    while k:
        if k & 1: 
            res = res * a % p
        a = a * a % p
        k >>= 1
    return res

# 核心算法：封装逻辑
def solve():
    # 极简变量命名
    a = int(next(it))
    p = int(next(it))
    
    # -> 根据题目“a能被p整除”条件判定逆元是否存在
    if a % p == 0:
        print("Inverse doesn't exist")
    else:
        # -> 根据费马小定理，逆元即 a 的 p-2 次幂
        print(qpow(a, p - 2, p))

# 主流程
if it:
    solve()
```
