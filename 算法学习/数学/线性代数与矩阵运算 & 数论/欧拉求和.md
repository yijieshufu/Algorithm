

### 题目背景与模板

**题目描述**：计算给定区间 $[L, R]$ 内所有整数的欧拉函数 $\varphi(n)$ 之和。

**输入条件**：两个整数 $L, R$，$1 \le L \le R \le 10^7$。

**所选题型**：数论 - **线性筛求欧拉函数**。

**通用模板代码（带注释）**：

```Python
import sys; input=sys.stdin.read().split(); it=iter(input);

# 核心算法：线性筛求欧拉函数 (Linear Sieve for Euler's Totient Function)
def get_phi(n):
    phi = [0] * (n + 1)
    phi[1] = 1 # 1 的欧拉函数定义为 1
    primes = []
    st = [False] * (n + 1)
    
    for i in range(2, n + 1):
        if not st[i]:
            primes.append(i)
            phi[i] = i - 1  # 质数 p 的 phi(p) = p - 1
        for p in primes:
            if i * p > n: break
            st[i * p] = True
            if i % p == 0:
                # p 是 i 的最小质因子，i*p 的质因子种类与 i 相同
                phi[i * p] = phi[i] * p
                break
            else:
                # i 与 p 互质，利用积性函数性质：phi(i*p) = phi(i) * phi(p)
                phi[i * p] = phi[i] * (p - 1)
    return phi
```

### 题目与核心思路

**逻辑总结**：

1. **线性筛优化**：对于 $10^7$ 的数据量，单次计算 $\varphi(n)$ 的 $O(\sqrt{n})$ 算法会超时。必须利用线性筛在 $O(R)$ 复杂度内一次性预处理出所有值。
    
2. **状态转移推导**：
    
    - 若 $p | i$：$\varphi(i \cdot p) = \varphi(i) \cdot p$。
        
    - 若 $p \nmid i$：$\varphi(i \cdot p) = \varphi(i) \cdot (p - 1)$（积性性质）。
        
3. **空间管理**：Python 列表处理 $10^7$ 数据需注意内存限制，本题 1024MB 充足。
    

**时空复杂度**：

- **时间复杂度**：$O(R)$，线性筛保证每个数只被访问常数次。
    
- **空间复杂度**：$O(R)$，用于存储 `phi` 数组和标记数组。
    

---

### 我的盲点记录

1. **推导疑问**：用户最初不理解 `phi[i * p]` 在 `i % p == 0` 时为什么是 `phi[i] * p`。
    
    - **解答**：解释了欧拉函数的公式 $\varphi(n) = n \prod (1 - \frac{1}{p_j})$。当 $p$ 已经是 $i$ 的因子时，乘入 $p$ 只增加了 $n$ 的部分，而后面的乘积项（质因子种类）不变，故直接乘 $p$。
        
2. **算法选型**：明确了 $10^7$ 必须使用 $O(N)$ 算法而非 $O(N \log N)$ 或 $O(N \sqrt{N})$。

### 最终范本代码

```Python
import sys; input=sys.stdin.read().split(); it=iter(input);

# 核心算法：线性筛求欧拉函数
def get_phi(n):
    phi = [0] * (n + 1)
    phi[1] = 1
    primes = []
    st = [False] * (n + 1)
    
    for i in range(2, n + 1):
        if not st[i]:
            primes.append(i)
            phi[i] = i - 1
        for p in primes:
            if i * p > n: break
            st[i * p] = True
            if i % p == 0:
                # -> 根据欧拉函数性质：当 p 为 i 因子时，phi(i*p) = phi(i) * p
                phi[i * p] = phi[i] * p
                break
            else:
                # -> 根据积性函数性质：当 gcd(i, p)=1 时，phi(i*p) = phi(i) * (p-1)
                phi[i * p] = phi[i] * (p - 1)
    return phi

# 核心算法：纯函数封装（处理求和逻辑）
def solve():
    if not it: return
    l = int(next(it))
    r = int(next(it))
    
    # -> 预处理 1 到 R 的欧拉函数表
    p_lst = get_phi(r)
    
    s = 0
    # -> 累加 [L, R] 闭区间内的所有 phi 值
    for i in range(l, r + 1):
        s += p_lst[i]
        
    print(s)

# 主流程：处理 I/O 并执行
if it:
    solve()
```
