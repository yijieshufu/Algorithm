
### 1. 题目背景与模板

- **题目描述**：给定一个 $n \times n$ 的矩阵，求其行列式 $|A|$ 在模 $998244353$ 意义下的值。
    
- **输入条件**：$n \le 500$，矩阵元素 $\le 10^9$。
    
- **所选题型**：行列式求值 / 欧几里得消元法（辗转相除法）。

### 2. 题目与核心思路

- **逻辑总结**：
    
    1. **转化目标**：利用行列式的性质（行交换变号、行倍加值不变），将矩阵化为上三角矩阵。
        
    2. **消元策略**：针对 Python 3.7 在大模数下求逆元（`pow`）慢、易超时的痛点，改用**辗转相除法**。通过两行不断取模和交换，在 $O(\log A)$ 次内将 $A_{j,i}$ 变为 0。
        
    3. **计算结果**：行列式等于变换后对角线元素的乘积 $A_{1,1} \times A_{2,2} \times \dots \times A_{n,n}$，最后根据行交换次数补齐符号。
        
- **时空复杂度**：时间复杂度 $O(n^3 \log (\max A))$，空间复杂度 $O(n^2)$。
    

### 3. 我的盲点记录

- **逆元瓶颈**：原本的高斯消元需要计算 $O(n^2)$ 次逆元，`pow(a, mod-2, mod)` 在 $n=500$ 时开销巨大导致超时。
    
- **辗转相除法的优势**：它只涉及基础算术运算（减法、整除、乘法），这些在 Python 底层被高度优化。此外，它不依赖模数是否为质数，通用性更强。
    
- **常数优化**：在内层循环中，通过 `if t:` 过滤掉无效运算，并使用局部变量缓存行引用（`row_i = a[i]`）能显著提升速度。
    

### 4. 最终范本代码（针对 Python 极致优化版）



```Python
import sys; input=sys.stdin.read().split(); it=iter(input);
mod = 998244353

# 核心算法：行列式求值（极致常数优化版）
def get_det(a, n):
    res, sign = 1, 1
    for i in range(1, n + 1):
        # -> 预处理：先找一个非0行，减少 while 内部交换频率
        p = i
        while p <= n and a[p][i] == 0: p += 1
        if p > n: return 0
        if p != i:
            a[i], a[p] = a[p], a[i]
            sign = -sign
            
        cur = a[i] # -> 优化：固定当前行引用，减少寻址开销
        for j in range(i + 1, n + 1):
            nxt = a[j]
            while nxt[i]:
                t = cur[i] // nxt[i]
                if t:
                    # -> 优化：仅对 i 之后的列进行计算
                    for k in range(i, n + 1):
                        cur[k] = (cur[k] - t * nxt[k]) % mod
                # -> 辗转相除核心：交换行并翻转符号
                a[i], a[j] = nxt, cur
                cur, nxt = a[i], a[j]
                sign = -sign
        
        res = (res * cur[i]) % mod
    
    return (res * sign + mod) % mod

# 核心算法：纯函数封装
def solve():
    if not it: return
    try:
        n = int(next(it))
    except StopIteration: return
    
    # -> 强制 1-indexed，开 n+1 大小
    a = [[0]*(n+1) for _ in range(n+1)]
    for i in range(1, n + 1):
        row = a[i]
        for j in range(1, n + 1):
            # -> 输入直接取模，防止大数运算拖慢速度
            row[j] = int(next(it)) % mod
            
    ans = get_det(a, n)
    # -> 最终结果确保为非负数
    sys.stdout.write(str((ans + mod) % mod) + '\n')

# 主流程
if it:
    solve()
```
