
### 一、 题目背景与模板

**题目描述**：给定斐波那契数列定义 $F(1)=F(2)=1, F(n)=F(n-1)+F(n-2)$，求第 $N$ 项在模 $10^9+7$ 下的值。

**输入条件**：$T \le 10^4$ 组测试数据，$N \le 10^{18}$。

**所选题型**：**矩阵快速幂 (Matrix Exponentiation)**

**通用模板代码（Python 3.7.3）**：

### 二、 题目与核心思路

**逻辑总结**：

1. **构造转移矩阵**：将递推式 $F(n) = F(n-1) + F(n-2)$ 转化为矩阵形式 $\begin{pmatrix} F(n) \\ F(n-1) \end{pmatrix} = \begin{pmatrix} 1 & 1 \\ 1 & 0 \end{pmatrix} \begin{pmatrix} F(n-1) \\ F(n-2) \end{pmatrix}$。
    
2. **跳步计算**：根据递推，$\begin{pmatrix} F(n) \\ F(n-1) \end{pmatrix} = A^{n-1} \begin{pmatrix} F(1) \\ F(0) \end{pmatrix}$。
    
3. **结果提取**：计算出 $A^{n-1}$ 后，其左上角元素 `res[1][1]` 即为 $F(n)$（因为 $F(1)=1, F(0)=0$）。

**时空复杂度**：

- **时间复杂度**：$O(T \cdot m^3 \log N)$，其中 $m=2$ 是矩阵阶数。在 $N=10^{18}$ 时，$\log N \approx 60$，单次查询极快。
    
- **空间复杂度**：$O(m^2)$，仅需存储常数个 $2 \times 2$ 矩阵。

---

### 三、 我的盲点记录

1. **矩阵方程的含义**：最初不理解矩阵乘法如何代表递推。通过拆解发现，矩阵第一行 $(1, 1)$ 对应加法规律 $F(n) = F(n-1) + F(n-2)$，第二行 $(1, 0)$ 对应状态平移 $F(n-1) = F(n-1)$。
    
2. **幂次的选择**：明确了从 $F(1)$ 推进到 $F(n)$ 需要经过 $n-1$ 次矩阵乘法，因此 `qpow` 的幂次参数为 $n-1$。

### 四、 最终范本代码
```Python

import sys; input=sys.stdin.read().split(); it=iter(input);
mod = 1000000007

def mul(a, b, n):
    c = [[0]*(n+1) for _ in range(n+1)]
    for i in range(1, n+1):
        for k in range(1, n+1):
            if not a[i][k]: continue
            for j in range(1, n+1):
                c[i][j] = (c[i][j] + a[i][k] * b[k][j]) % mod
    return c

def qpow(a, k, n):
    res = [[0]*(n+1) for _ in range(n+1)]
    for i in range(1, n+1): res[i][i] = 1
    while k:
        if k & 1: res = mul(res, a, n)
        a = mul(a, a, n)
        k >>= 1
    return res

def solve(n):
    if n == 0: return 0
    if n <= 2: return 1
    
    # 构造转移矩阵 A = [[1, 1], [1, 0]]
    a = [[0]*3 for _ in range(3)]
    a[1][1] = 1; a[1][2] = 1 # -> 对应 F(n) = F(n-1) + F(n-2)
    a[2][1] = 1; a[2][2] = 0 # -> 对应 F(n-1) = F(n-1)
    
    # -> 目标是 F(n)，需要从 F(1) 状态推进 n-1 步
    res = qpow(a, n-1, 2)
    
    # -> 结果向量的第一项为 res[1][1]*F(1) + res[1][2]*F(0)，代入 F(1)=1, F(0)=0
    return res[1][1]

if input:
    t = int(next(it))
    for _ in range(t):
        n = int(next(it))
        print(solve(n))
```
