### 1. 题目背景与模板 📋

**题目背景：**

- **题目描述**：在一个 $N$ 维空间中，给出球面上 $N+1$ 个点的坐标，求该 $N$ 维球体的球心坐标。
    
- **输入条件**：$1 \le N \le 10$，坐标绝对值不超过 20000。
    
- **题型**：高斯-约旦消元法解线性方程组。
    

**通用模板代码：**

```Python
import sys; input=sys.stdin.read().split(); it=iter(input);

# 核心算法：高斯-约旦消元法（Gauss-Jordan Elimination）
def gauss(a, n):
    # a 为 (n+1) * (n+2) 的增广矩阵，索引从 1 开始
    # a[i][n+1] 存储方程右侧的常数项 b
    for i in range(1, n + 1):
        # 1. 寻找当前列绝对值最大的行作为主元，确保数值稳定性
        r = i
        for j in range(i + 1, n + 1):
            if abs(a[j][i]) > abs(a[r][i]): r = j
        a[i], a[r] = a[r], a[i]
        
        # 2. 归一化：将当前行主元系数变为 1
        if abs(a[i][i]) < 1e-8: continue 
        div = a[i][i]
        for j in range(i, n + 2): a[i][j] /= div
        
        # 3. 消元：将其他行的第 i 列全部化为 0，直接得出对角阵
        for j in range(1, n + 1):
            if i != j:
                mul = a[j][i]
                for k in range(i, n + 2):
                    a[j][k] -= mul * a[i][k]
    
    # 返回解向量 [c1, c2, ..., cn]
    return [a[i][n+1] for i in range(1, n + 1)]
```

### 2. 题目与核心思路 🧠

- **逻辑总结**：
    
    1. **几何转代数**：利用球面上点到球心距离相等的性质，列出 $N+1$ 个二次方程：$\sum (a_{i,j} - x_j)^2 = R^2$。
        
    2. **降次消元**：通过相邻两个方程作差，即 $(i+1)$ 号方程减去 $i$ 号方程，消去二次项 $x_j^2$ 和半径 $R^2$。
        
    3. **构造方程**：整理得线性方程 $\sum 2(a_{i+1,j} - a_{i,j})x_j = \sum (a_{i+1,j}^2 - a_{i,j}^2)$。
        
    4. **矩阵求解**：构造 $N \times (N+1)$ 的增广矩阵，调用高斯消元算法。
        
- **复杂度分析**：
    
    - 时间复杂度：$O(N^3)$，主要消耗在矩阵消元过程。
        
    - 空间复杂度：$O(N^2)$，用于存储增广矩阵。

### 3. 我的盲点记录 🔍

- **疑问 1**：不理解矩阵是如何“变魔术”解出答案的。
    
    - **解答**：模拟手算过程。通过“找主元、归一化、消元”三步，将复杂的系数矩阵逐步化简为单位矩阵，使每个未知数直接对应一个常数解。
        
- **疑问 2**：平方项是如何消失的？
    
    - **解答**：通过“方程相减”。利用 $(a-x)^2 - (b-x)^2 = a^2 - 2ax + x^2 - (b^2 - 2bx + x^2)$，其中的 $x^2$ 会被完全抵消，从而实现从二次到一次的跨越。

### 4. 最终范本代码 💻

```Python
import sys; input=sys.stdin.read().split(); it=iter(input);

# 核心算法：高斯-约旦消元法
def gauss(a, n):
    for i in range(1, n + 1):
        r = i
        for j in range(i + 1, n + 1):
            if abs(a[j][i]) > abs(a[r][i]): r = j
        a[i], a[r] = a[r], a[i]
        
        if abs(a[i][i]) < 1e-8: continue
        div = a[i][i]
        for j in range(i, n + 2): a[i][j] /= div
        
        for j in range(1, n + 1):
            if i != j:
                mul = a[j][i]
                for k in range(i, n + 2):
                    a[j][k] -= mul * a[i][k]
    return [a[i][n+1] for i in range(1, n + 1)]

# 核心算法：纯函数封装
def solve():
    n = int(next(it))
    # p[i][j] 存储第 i 个点的第 j 维坐标
    p = [[0.0]*(n+1) for _ in range(n+2)] # -> 遵循 1-indexed，存储 n+1 个点
    for i in range(1, n+2):
        for j in range(1, n+1):
            p[i][j] = float(next(it))
    
    # 构建增广矩阵 a
    a = [[0.0]*(n+2) for _ in range(n+1)] # -> 索引从 1 开始，大小为 n*(n+1)
    for i in range(1, n+1):
        for j in range(1, n+1):
            # -> 这里的逻辑是将第 i+1 个点和第 i 个点作差消元
            # 系数为 2 * (a_{i+1, j} - a_{i, j})
            a[i][j] = 2 * (p[i+1][j] - p[i][j])
            # 常数项为平方和之差：sum(a_{i+1, j}^2 - a_{i, j}^2)
            a[i][n+1] += p[i+1][j]**2 - p[i][j]**2
            
    ans = gauss(a, n)
    # 按要求输出，保留 3 位小数
    print(" ".join("{:.3f}".format(x) for x in ans))

# 主流程
if it:
    solve()
```
