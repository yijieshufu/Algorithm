### 题目与核心思路

**逻辑总结**：

本题利用指数的**二进制拆分**思想。将 $O(b)$ 的线性累乘优化为 $O(\log b)$。通过不断平方底数 $a$ 并结合位运算 `k & 1` 判断当前二进制位是否需要累乘进结果。

**费马小定理关联**：虽然本题只需计算 $a^b \bmod p$，但你模板中提到的 `n^(p-2) % p` 是该算法在**逆元计算**（处理除法取模）时的核心应用，前提是 $p$ 为质数且 $n, p$ 互质。

**时空复杂度**：

- **时间复杂度**：$O(\log b)$。
    
- **空间复杂度**：$O(1)$。

### 我的盲点记录

1. **输入处理**：在 Python 3.7.3 环境下，使用 `sys.stdin.read().split()` 配合迭代器 `next(it)` 是处理大规模输入的最优解。
    
2. **取模时机**：代码中的 `a %= p` 必不可少，防止底数 $a$ 初始值远大于 $p$ 导致第一次乘法运算量过大。
    
3. **参数命名**：你的模板中使用 `k` 代表指数，这在数学表达和位运算逻辑中非常直观。

import sys; input=sys.stdin.read().split(); it=iter(input);

# 核心算法：快速幂
def qpow(a, k, p):
    res = 1
    a %= p # -> 规范：先取模确保底数范围
    while k:
        if k & 1: 
            res = res * a % p # -> 逻辑：若当前二进制位为1则累乘
        a = a * a % p # -> 逻辑：底数平方倍增
        k >>= 1 # -> 逻辑：指数二进制右移
    return res
