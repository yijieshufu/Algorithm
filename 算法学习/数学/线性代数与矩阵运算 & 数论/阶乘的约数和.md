**输入条件**：输入包含一个正整数 $n$（$n \le 2 \times 10^5$）。

**所选题型**：数论（埃氏筛 + 勒让德定理 + 乘法逆元）。

## 题目与核心思路

- **逻辑总结**：
    
    1. **约数和定理**：若 $N = \prod p_i^{a_i}$，则约数和 $S = \prod \frac{p_i^{a_i+1}-1}{p_i-1}$。
        
    2. **阶乘分解**：利用勒让德定理计算 $n!$ 中质数 $p$ 的指数 $a = \sum \lfloor n/p^k \rfloor$。
        
    3. **模逆元**：由于 $mod$ 是质数，利用费马小定理 $\text{inv}(p-1) = (p-1)^{mod-2} \pmod{mod}$ 处理分母。
        
- **时空复杂度**：
    
    - **时间**：$O(n \log \log n)$ 筛选，统计过程趋近 $O(n)$，总时间复杂度极优。
        
    - **空间**：$O(n)$，主要为质数标记数组。

---

## 我的盲点记录

1. **模板调用一致性**：核心算法如 `qpow` 和 `get_primes` 必须独立封装，不应混在 `solve` 内部，以保持代码整洁。
    
2. **特殊值处理**：当 $n=1$ 时，约数和为 $1$，循环不会执行，`ans` 初始值应设为 $1$。
    
3. **变量命名**：严格遵守极简风格，`up` 为分子，`dn` 为分母逆元。

## 最终范本代码

```Python
import sys; input=sys.stdin.read().split(); it=iter(input);
mod = 998244353

# 核心算法：数值快速幂
def qpow(a, k):
    res = 1
    a %= mod
    while k:
        if k & 1: res = res * a % mod
        a = a * a % mod
        k >>= 1
    return res

# 核心算法：埃氏筛
def get_primes(n):
    isp = [True] * (n + 1)
    if n >= 0: isp[0] = False
    if n >= 1: isp[1] = False
    for i in range(2, int(n**0.5) + 1):
        if isp[i]:
            isp[i*i : n+1 : i] = [False] * len(range(i*i, n+1, i))
    return [i for i in range(2, n + 1) if isp[i]]

# 核心算法：纯函数封装
def solve():
    # -> 按照 1-indexed 规范，直接读取 n
    n = int(next(it))
    
    # -> 调用模板函数获取质数列表
    ps = get_primes(n)
    
    ans = 1
    for p in ps:
        # -> 勒让德定理：统计 p 在 n! 中的次数 a
        a = 0
        v = n
        while v >= p:
            a += v // p
            v //= p
        
        # -> 核心公式：(p^(a+1)-1) * inv(p-1)
        # -> qpow(p, a+1) 计算幂次，qpow(p-1, mod-2) 计算分母逆元
        up = (qpow(p, a + 1) - 1 + mod) % mod
        dn = qpow(p - 1, mod - 2)
        ans = ans * (up * dn % mod) % mod
        
    print(ans)

# 主流程：处理 I/O 并执行
if it:
    solve()
```
