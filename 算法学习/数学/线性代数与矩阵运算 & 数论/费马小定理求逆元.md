## 题目与核心思路

**逻辑总结**：

1. **费马小定理**指出：若 $p$ 为质数且 $n$ 不是 $p$ 的倍数，则 $n^{p-1} \equiv 1 \pmod p$。
    
2. 通过等式变形：$n \cdot n^{p-2} \equiv 1 \pmod p$，可知 $n$ 在模 $p$ 意义下的逆元即为 $n^{p-2}$。
    
3. 利用**快速幂**算法，可以在对数时间内完成大指数的幂运算。

**复杂度分析**：

- **时间复杂度**：$O(\log p)$，快速幂的标准复杂度。
    
- **空间复杂度**：$O(1)$，仅使用了常数个辅助变量。

---

## 我的盲点记录

- **逆元存在的前提**：只有当 $\gcd(n, p) = 1$ 时逆元才存在。本题通过“$p$ 是质数”且“$n \bmod p \neq 0$”直接保证了这一点。
    
- **算法选择**：如果 $p$ 不是质数，费马小定理将失效。此时需改用 **扩展欧几里得算法 (exgcd)** 或 **欧拉定理**（需已知 $\phi(p)$）。

import sys; input=sys.stdin.read().split(); it=iter(input);

# 核心算法：快速幂
def qpow(a, k, p):
    res = 1
    a %= p
    while k:

            res = (res * a) % p
        a = (a * a) % p
        k >>= 1
    return res

# 核心算法：纯函数封装
def solve():
    n = int(next(it))
