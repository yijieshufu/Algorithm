

### 📌 题目背景与模板

**题目描述**：给定 $a, b, m$，求解最小非负整数 $x$ 满足线性同余方程 $ax \equiv b \pmod m$。若无解输出 $-1$。

**输入条件**：$t$ 组数据，每组包含 $a, b, m$。

**所选题型**：数论 - 线性同余方程 / 扩展欧几里得算法 (exGCD)。

**通用模板代码（带注释）：**

```Python
# 核心算法：扩展欧几里得算法 (Extended Euclidean Algorithm)
def exgcd(a, b):
    # 返回 d=gcd(a, b) 和满足 ax + by = d 的一组特解 x, y
    if b == 0:
        return a, 1, 0
    d, x_prev, y_prev = exgcd(b, a % b)
    
    x = y_prev
    y = x_prev - (a // b) * y_prev
    return d, x, y
```

---

### 💡 题目与核心思路

1. **数学转化**：方程 $ax \equiv b \pmod m$ 转化为不定方程 $ax + my = b$。
    
2. **存在性判定**：利用裴蜀定理，方程有解当且仅当 $\gcd(a, m) \mid b$（即 $b$ 是 $a$ 和 $m$ 最大公约数的倍数）。
    
3. **求解流程**：
    
    - 用 `exgcd(a, m)` 得到 $ax' + my' = \gcd(a, m)$ 的解 $x'$。
        
    - 将 $x'$ 放大 $\frac{b}{\gcd(a, m)}$ 倍得到原方程的一个特解 $x$。
        
    - 原方程解的最小正周期为 $L = \frac{m}{\gcd(a, m)}$。
        
    - 最终解为 $x \pmod L$。
        
4. **时空复杂度**：
    
    - 时间复杂度：$O(T \log(\min(a, m)))$，每组数据仅需对数级时间。
        
    - 空间复杂度：$O(\log(\min(a, m)))$，为递归栈深度。

### 🔍 我的盲点记录

- **Q：$\gcd(4, 10)=2$ 是什么意思？**
    
    - **A**：表示 4 和 10 的**最大公约数**是 2。只有当 $b$ 是 2 的倍数时，方程才有解。
        
- **Q：为什么特解要乘以 $\frac{b}{\gcd(a, m)}$？**
    
    - **A**：因为 `exgcd` 算出的是右边等于 $\gcd$ 时的解，为了让等式右边变成 $b$，需要整体缩放 $b/\gcd$ 倍。

### ✅ 最终范本代码

```Python
import sys; input=sys.stdin.read().split(); it=iter(input);

# 核心算法：扩展欧几里得算法
def exgcd(a, b):
    if b == 0:
        return a, 1, 0
    d, x_prev, y_prev = exgcd(b, a % b)
    x = y_prev
    y = x_prev - (a // b) * y_prev
    return d, x, y

# 核心算法：纯函数封装
def solve():
    if not input: return
    t = int(next(it))
    for _ in range(t):
        a = int(next(it))
        b = int(next(it))
        m = int(next(it))
        
        d, x0, y0 = exgcd(a, m)
        
        # -> 根据裴蜀定理，b 必须是 gcd(a, m) 的倍数才有解
        if b % d != 0:
            print("-1")
            continue
        
        # -> 根据逻辑：原方程解 x = x0 * (b/d)，周期 L = m/d
        step = m // d
        # -> 使用取模运算直接获得 [0, step-1] 范围内的最小非负整数解
        ans = (x0 * (b // d)) % step
        print(ans)

# 主流程：处理 I/O 并调用
if it:
    solve()
```
