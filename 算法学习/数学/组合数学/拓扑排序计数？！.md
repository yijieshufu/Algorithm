### 题目背景与模板

- **题目描述**：求所有不同的有标号 $n$ 个点的 DAG 的拓扑序总数之和。
    
- **输入条件**：$T \le 10^5, n \le 5 \cdot 10^5$，结果对 $10^9+7$ 取模。
    
- **所选题型**：组合数学（贡献法）、快速幂、阶乘预处理。
    

**核心算法：预处理阶乘与逆元模板：**

```Python
def prepare(limit):
    fac = [1] * (limit + 1)
    inv = [1] * (limit + 1)
    for i in range(1, limit + 1):
        fac[i] = fac[i-1] * i % mod
    # 快速幂求逆元：a^(p-2) % p
    inv[limit] = pow(fac[limit], mod - 2, mod)
    for i in range(limit - 1, -1, -1):
        inv[i] = inv[i+1] * (i + 1) % mod
    return fac, inv
```

---

### 题目与核心思路

- **逻辑总结**：
    
    1. **转换视角**：原题求所有 DAG 的拓扑序之和，等价于统计“每个排列在多少个不同的 DAG 中是合法的”。
        
    2. **排列贡献**：对于任意一个排列，为了让它成为拓扑序，图中 $\frac{n(n-1)}{2}$ 对点对中的每一对，都只有“不连边”或“按排列顺序连边”这 $2$ 种合法情况。
        
    3. **计算推导**：总排列数为 $n!$，每个排列对应的合法图数量为 $2^{\frac{n(n-1)}{2}}$。
        
- **时空复杂度**：时间复杂度 $O(N + T \log N)$，空间复杂度 $O(N)$。
    

---

### 我的盲点记录

- **疑问**：为什么代码中没有直接调用 `C(n, k)` 或 `P(n, k)`？
    
- **解答**：虽然本题使用了预处理模板，但最终推导出的公式 $n! \times 2^{C_n^2}$ 仅直接使用了阶乘 `fac[n]`，而不需要进行复杂的组合数选取。不过，预处理模板依然是高效解决此类大批量阶乘运算的最佳方案。

### 最终范本代码


```Python

import sys; input=sys.stdin.read().split(); it=iter(input);
mod = 1000000007

# 核心算法：快速幂 用这个模板
def qpow(a, k, p):
    res = 1
    a %= p
    while k:
        if k & 1:
            res = res * a % p
        a = a * a % p
        k >>= 1
    return res

# 核心算法：预处理阶乘与逆元，用于快速计算组合数和排列数
def prepare(limit):
    fac = [1] * (limit + 1)
    inv = [1] * (limit + 1)
    for i in range(1, limit + 1):
        fac[i] = fac[i-1] * i % mod
    # 快速幂求逆元：a^(p-2) % p
    inv[limit] = pow(fac[limit], mod - 2, mod)
    for i in range(limit - 1, -1, -1):
        inv[i] = inv[i+1] * (i + 1) % mod
    return fac, inv

# 核心算法：纯函数封装
def solve():
    # -> 强制 1-indexed，根据题目数据规模 n <= 5*10^5 开数组
    limit = 500000
    fac, inv = prepare(limit)
    
    if not it: return
    t_str = next(it, None)
    if t_str is None: return
    t = int(t_str)
    
    res_list = []
    for _ in range(t):
        n = int(next(it))
        # -> 根据贡献法推导，本题逻辑为：n! * 2^(n*(n-1)/2)
        p = n * (n - 1) // 2
        # -> 使用 fac[n] 获取排列部分，使用 qpow 计算 2 的幂次
        ans = fac[n] * qpow(2, p, mod) % mod
        res_list.append(str(ans))
    
    sys.stdout.write('\n'.join(res_list) + '\n')

# 主流程：处理 I/O 并执行
if it:
    solve()
```
