
### 题目背景与模板

**题目描述**：给定正整数 $N, M$，构造两个长度为 $N$ 的序列 $A$ 和 $B$。要求：$1 \le A_i, B_i \le M$；每个序列内部元素互不相同；且对应位置 $A_i \neq B_i$。求方案数对 $10^9+7$ 取模。

**输入条件**：$1 \le N \le M \le 5 \times 10^5$。

**所选题型**：组合计数 + 容斥原理。

**通用模板代码（带注释）**：

```Python
import sys; input=sys.stdin.read().split(); it=iter(input);
mod = 1000000007

# 核心算法：预处理阶乘与逆元，用于快速计算组合数和排列数
def prepare(limit):
    fac = [1] * (limit + 1)
    inv = [1] * (limit + 1)
    for i in range(1, limit + 1):
        fac[i] = fac[i-1] * i % mod
    
    # 快速幂求逆元：a^(p-2) % p
    inv[limit] = pow(fac[limit], mod - 2, mod)
    for i in range(limit - 1, -1, -1):
        inv[i] = inv[i+1] * (i + 1) % mod
    return fac, inv

# 核心算法：组合数 C(n, k)
def C(n, k, fac, inv):
    if k < 0 or k > n: return 0
    return fac[n] * inv[k] % mod * inv[n-k] % mod

# 核心算法：排列数 P(n, k)
def P(n, k, fac, inv):
    if k < 0 or k > n: return 0
    return fac[n] * inv[n-k] % mod

# 核心算法：纯函数封装
def solve():
    pass

if it:
    solve()
```

---

### 题目与核心思路

- **逻辑总结**：
    
    1. **全集**：不考虑 $A_i \neq B_i$，方案数为 $(P_M^N)^2$。
        
    2. **容斥**：设属性 $P_i$ 为 $A_i = B_i$。我们要计算不满足任何 $P_i$ 的方案数。
        
    3. **计算“至少 $k$ 个位置相等”**：
        
        - 选 $k$ 个位置：$C_N^k$。
            
        - 这 $k$ 个位置上的数（$A, B$ 共享）：$P_M^k$。
            
        - 剩余 $N-k$ 个位置：$A$ 剩 $P_{M-k}^{N-k}$ 种，$B$ 剩 $P_{M-k}^{N-k}$ 种。
            
    4. **公式推导**：$\sum_{k=0}^{N} (-1)^k \cdot C_N^k \cdot P_M^k \cdot (P_{M-k}^{N-k})^2$。
        
- **时空复杂度**：
    
    - 时间：$O(M)$，预处理与单层循环。
        
    - 空间：$O(M)$，存储阶乘与逆元数组。

---

### 我的盲点记录

1. **逆元的作用**：在模运算下，除法不能直接进行，必须通过 $inv[x]$ 将除法转为乘法。
    
2. **排列数的应用**：在选取 $k$ 个位置的数时，因为位置本身是有序的（序列第 $i$ 位），所以使用 $P_M^k$ 而非 $C_M^k$。
    
3. **公式化简**：发现 $P_M^k \cdot P_{M-k}^{N-k} = P_M^N$。利用这个性质可以简化代码中的排列数调用，虽然不影响复杂度，但逻辑更清晰。

### 最终范本代码

```Python
import sys; input=sys.stdin.read().split(); it=iter(input);
mod = 1000000007

def prepare(limit):
    fac = [1] * (limit + 1)
    inv = [1] * (limit + 1)
    for i in range(1, limit + 1):
        fac[i] = fac[i-1] * i % mod
    inv[limit] = pow(fac[limit], mod - 2, mod)
    for i in range(limit - 1, -1, -1):
        inv[i] = inv[i+1] * (i + 1) % mod
    return fac, inv

def C(n, k, fac, inv):
    if k < 0 or k > n: return 0
    return fac[n] * inv[k] % mod * inv[n-k] % mod

def P(n, k, fac, inv):
    if k < 0 or k > n: return 0
    return fac[n] * inv[n-k] % mod

def solve():
    n = int(next(it))
    m = int(next(it))
    
    fac, inv = prepare(m)
    
    ans = 0
    # -> 预先提取 P(m, n) 是因为每一项容斥里都包含至少一组 P(m, k) * P(m-k, n-k)
    p_m_n = P(m, n, fac, inv) 
    
    for k in range(n + 1):
        # -> 核心容斥公式：(-1)^k * C(n, k) * 至少k位相同的方案数
        # -> 其中“至少k位相同”简化后为 p_m_n * P(m-k, n-k)
        term = C(n, k, fac, inv) * p_m_n % mod * P(m - k, n - k, fac, inv) % mod
        
        if k & 1:
            ans = (ans - term + mod) % mod
        else:
            ans = (ans + term) % mod
            
    print(ans)

if it:
    solve()
```
