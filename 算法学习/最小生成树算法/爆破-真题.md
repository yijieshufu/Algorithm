# 题目描述
- **核心目标**：在二维平面给定 $n$ 个圆，圆间相交则连通（代价 0），不相交则需用魔法回路**连接边缘**（代价为圆心距减去两圆半径之和），求使所有圆连通的最小代价总和。
- **数据边界**：$n \le 5000$，坐标 $\le 2000$。$n=5000$ 意味着 $O(n^2)$ 算法是 Python 的极限，需极简实现。。
# 分析
- **算法/数据结构**：**Prim 算法（普里姆算法）**。
- **为何适用**：
    - 本题是一个**稠密图**（每两个圆之间都存在潜在的边）。
    - Kruskal 需要存储并排序 $O(n^2)$ 条边（约 1250 万条），空间和排序时间开销巨大。
    - Prim 的 $O(n^2)$ 时间复杂度与 $O(n)$ 空间复杂度在稠密图中更具优势。
- **通用骨架**：
```python
    dist = [inf] * n; dist[0] = 0; vis = [False] * n
    for _ in range(n):
        u = 选取未访问点中 dist 最小的
        vis[u] = True
        更新所有与 u 相连的点的 dist
```
 **[块 1：几何距离建模]**  
两圆 $i$ 和 $j$ 的圆心距为 $d = \sqrt{(x_i-x_j)^2 + (y_i-y_j)^2}$。
- **边缘距离**：$w_{i,j} = \max(0, d - r_i - r_j)$。
- **逻辑简化**：若 $d \le r_i + r_j$，则两圆相交或相切，边权为 0。  
 **[块 2：Prim 状态转移]**  
维护一个 `lowcost` 数组，代表非树节点到当前生成树的最短距离。
- 初始化：`lowcost[0] = 0`，其余为无穷大。
- 迭代：每次选出 `lowcost` 最小的点 $u$，将其加入树中，并用 $u$ 去更新其邻居 $v$ 的 `lowcost[v] = min(lowcost[v], weight(u, v))`。
#### **[块 3：Python 性能“压榨”技巧]**
1. **预解构坐标**：在循环外将 $x, y, r$ 分别存入列表，循环内通过索引访问，避免 `circle.x` 这种属性查找。
2. **减少平方根计算**：虽然 $d$ 需要开根号，但可以先判断平方和 $(x_i-x_j)^2 + (y_i-y_j)^2$ 是否小于 $(r_i+r_j)^2$，若小于则距离直接记为 0，跳过 `math.sqrt`。
3. **局部变量化**：将 `math.sqrt` 赋值给局部变量（如 `sqrt = math.sqrt`）可以略微提升调用速度。
# 笔记

## 代码

```python
import sys
import math
# [起手式：标准 I/O 与数据打包]
it = iter(sys.stdin.read().split())
n = int(next(it))
# a: 存储圆信息 (x, y, r)
a = [(int(next(it)), int(next(it)), int(next(it))) for _ in range(n)]
# [块 1：状态定义]
f = [1e18] * n  # lowcost 数组，记录到生成树的最短距离
f[0] = 0.0
q = list(range(n)) # 未访问集合 (unvisited)
ans = 0.0
sq = math.sqrt # 局部化函数提速
while q:
    # 1. 寻找当前 f 值最小的点 u (变量 p 为索引，u 为节点编号)
    p = 0
    for i in range(1, len(q)):
        if f[q[i]] < f[q[p]]: 
            p = i
    u = q[p]
    ans += f[u]
    # [块 2：O(1) 极速斩杀 - 覆盖并弹出]
    q[p] = q[-1]
    q.pop()
    # 2. 更新剩余点到生成树的距离
    ux, uy, ur = a[u]
    for v in q:
        vx, vy, vr = a[v]
        # [块 3：算术降级 - 避免 **2]
        dx, dy = ux - vx, uy - vy
        d2 = dx * dx + dy * dy
        sr = ur + vr
        # 几何判定：若平方和 <= 半径和平方，则相交代价为 0
        c = 0.0 if d2 <= sr * sr else sq(d2) - sr
        if c < f[v]: 
            f[v] = c
print(f"{ans:.2f}")
```
