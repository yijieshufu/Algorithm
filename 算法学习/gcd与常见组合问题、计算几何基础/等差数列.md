# 题目描述
数学老师给出了等差数列中的 $N$ 个整数（顺序是乱的），我们需要找到一个包含这 $N$ 个数的最短等差数列，并输出这个数列一共有多少项。
# 分析
- **难度等级**：入门/简单
- **高频程度**：**必考高频题**

- **排序**：将给定的 $N$ 个数从小到大排序，确定最小值为首项，最大值为末项。
- **求间距**：计算相邻两个数之间的差值。
- **求 GCD**：求出所有差值的最大公约数，这便是我们要找的最大公差 $d$。
- **套公式**：项数 = $\frac{最大值 - 最小值}{d} + 1$。
    - _特例_：如果所有差值都为 0（即所有数都相同），公差为 0，项数直接输出 $N$






# 笔记

## 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
n = int(next(it))
a = [int(x) for x in it]
a.sort()
def gcd(a,b):
  while b:
    a,b=b,a%b
  return a
d = 0
for i in range(1,n):
  diff = a[i]-a[i-1]
  if diff == 0:
    d = diff
  else:
    d = gcd(d,diff)
min_a = a[0]
max_a = a[-1]
if d == 0:print(n)
else:
  ans = (max_a-min_a)//d +1
  print(ans)
```
