# 题目描述

有 $n$ 封信和 $n$ 个信封，每一封信都有一个正确对应的信封。现在要求**每一封信都装错**（即信封 $A$ 里面绝对不能装信 $A$），求出这种“全装错”的情况总共有多少种。
# 分析过程
错排公式

$$
D(n) = (n-1) \times [D(n-1) + D(n-2)]
$$

- **初始条件**（基础场景）：
    - $D(1) = 0$ （一个人没法戴错）
    - $D(2) = 1$ （两个人只能互换帽子，只有 1 种换法）

# 代码
```python
import os
import sys
# 请在此输入您的代码
it = iter(sys.stdin.read().split())
for val in it:
  n = int(val)
  d = [0] * 21
  d[1]=0;d[2]=1
  for i in range(3,n+1):
    d[i]=(i-1)*(d[i-1]+d[i-2])
  print(d[n])
```
