## 🧭 计算几何核心笔记：从零到一

### 1. 精度控制：给浮点数装个“放大镜”

- **痛点**：计算机存小数（浮点数）不准。直接用 `a == b` 会因为微小的误差导致判断错误（比如 `0.1 + 0.2` 可能不等于 `0.3`）。
- **核心思路**：引入一个极小值 $EPS$（通常设为 $10^{-8}$）。
    - **判断相等**：只要 $|a - b| < EPS$，就说明它们“够近了”，算相等。
    - **比较大小**：判断 $a > b$ 时，写成 $a > b + EPS$ 以排除误差干扰。

### 2. 向量运算：几何里的“瑞士军刀”
- **痛点**：直接算角度、坐标转换太复杂，容易出错。
- **核心思路**：把坐标看作从原点出发的箭头（向量）。
- **必杀技：叉积 (Cross Product)**
    - **解决什么**：算面积、判断左右、判断顺逆时针。
    - **通俗理解**：两个向量 $A$ 和 $B$ 围成一个平行四边形。
        - **数值**：叉积的大小 = 平行四边形的面积。
        - **正负**：$A \times B$ 为正，说明 $B$ 在 $A$ 的左侧（逆时针）；为负，说明在右侧。

```python
import sys
import math
class Point:
    def __init__(self, x, y):
        self.x = float(x)
        self.y = float(y)
    
    # 定义减法，方便求向量：A - B
    def __sub__(self, other):
        return Point(self.x - other.x, self.y - other.y)
# 核心：求向量模长（长度）
def get_len(a):
    return math.sqrt(a.x**2 + a.y**2)
# 核心：叉积运算
def cross(a, b):
    return a.x * b.y - a.y * b.x
```



| **运算**         | **Python 公式**   | **结果意义** | **解决的问题**         |
| -------------- | --------------- | -------- | ----------------- |
| **点积 (Dot)**   | `x1*x2 + y1*y2` | 标量（投影长度） | 算夹角、算投影、判直角       |
| **叉积 (Cross)** | `x1*y2 - x2*y1` | 标量（有向面积） | **算面积、判左右、判线段相交** |

### 3. 三大高频判定模板

#### A. 三角形面积（最快算法）
- **核心思路**：已知三个点 $A, B, C$，先求向量 $\vec{AB}$ 和 $\vec{AC}$，面积就是：

    $$


S = \frac{1}{2} | \vec{AB} \times \vec{AC} |

$$
- **优势**：不用算角度，不用开方，精度最高。
```python
# 1. 求三角形面积 (A, B, C 为顶点)
def get_area(a, b, c):
    return abs(cross(b - a, c - a)) / 2.0
```
#### B. 点到直线的距离
- **核心思路**：**面积反推法**。
    - 想象点 $P$ 与直线 $AB$ 组成一个三角形。
    - 距离 $h$ 就是这个三角形以 $AB$ 为底边时的高度。
    - 公式：$h = \frac{2 \times \text{面积}}{|\vec{AB}|}$。
```python
def dist_to_line(p, a, b):
    # 第一步：把“点”转化成“绳子”（向量）
    v_line = b - a   # 向量 AB，作为三角形的“底边”
    v_point = p - a  # 向量 AP，作为三角形的一条“腰”
    
    # 第二步：算出平行四边形的面积（即 2 倍的三角形面积）
    # cross(v_line, v_point) 算出的是以 AB 和 AP 为邻边的平行四边形面积
    area_x2 = abs(cross(v_line, v_point))
    
    # 第三步：算出底边的长度
    base_len = get_len(v_line)
    
    # 第四步：高 = 面积 / 底
    # 这里的 area_x2 已经是面积的 2 倍了，所以直接除以底就行
    return area_x2 / base_len

```
#### C. 线段相交（跨立实验）
- **核心思路**：判断线段 $AB$ 和 $CD$ 是否相交。
- **通俗理解**：如果相交，$C$ 和 $D$ 必须分别在直线 $AB$ 的左右两边，同时 $A$ 和 $B$ 也要在直线 $CD$ 的左右两边。
- **实现**：用叉积的正负号来判断是否“跨”在两边。

##### 如何用“叉积”来判断“左右”？

还记得我们学过的叉积 `cross(a, b)` 吗？

- 如果 `cross(向量1, 向量2) > 0`，说明向量 2 在向量 1 的**左边**。
- 如果 `cross(向量1, 向量2) < 0`，说明向量 2 在向量 1 的**右边**。

**判定公式：**
对于线段 $AB$ 和点 $C, D$：

1. 算出 `c1 = cross(B-A, C-A)` （点 $C$ 在 $AB$ 的哪边）
2. 算出 `c2 = cross(B-A, D-A)` （点 $D$ 在 $AB$ 的哪边）
3. 如果 `c1 * c2 < 0`，说明一正一负，**跨立成功**！
```python
def is_intersect(A, B, C, D):
    # 1. 判断 C, D 是否在直线 AB 的两侧
    # 向量 AB 叉乘 向量 AC
    cp1 = cross(B - A, C - A)
    # 向量 AB 叉乘 向量 AD
    cp2 = cross(B - A, D - A)
    
    # 2. 判断 A, B 是否在直线 CD 的两侧
    # 向量 CD 叉乘 向量 CA
    cp3 = cross(D - C, A - C)
    # 向量 CD 叉乘 向量 CB
    cp4 = cross(D - C, B - C)
    # 3. 双向跨立：两组叉积结果都必须是“一正一负”
    # 如果乘积为 0，说明某个点正好落在另一条直线上（也算相交）
    if (cp1 * cp2 <= 0) and (cp3 * cp4 <= 0):
        return True
    return False

```

### 4. 圆与园关系判定

#### 1. 核心思路：三个关键数值
判断两个圆（圆心分别为 $O_1, O_2$，半径分别为 $r_1, r_2$）的关系，只需要盯着这三个数：
1. **圆心距 $d$**：两点之间的直线距离。
2. **半径和 $S$**：$r_1 + r_2$。
3. **半径差 $D$**：$|r_1 - r_2|$。

#### 2. 判定逻辑（带 EPS 处理版）
因为 $d$ 是通过 `sqrt` 开方算出来的浮点数，不要使用 ==

| **关系** | **数学条件**    | **通俗理解**  | **精度代码写法**                    |
| ------ | ----------- | --------- | ----------------------------- |
| **相离** | $d > S$     | 离得远，没碰着   | `d > S + EPS`                 |
| **外切** | $d = S$     | 刚好背靠背挨着   | `abs(d - S) < EPS`            |
| **相交** | $D < d < S$ | 叠在一起了     | `d > D + EPS and d < S - EPS` |
| **内切** | $d = D$     | 小圆在内，刚好碰边 | `abs(d - D) < EPS`            |
| **内含** | $d < D$     | 小圆被大圆完全吞没 | `d < D - EPS`                 |
```python
import sys
import math
# 精度控制值
EPS = 1e-9

class Circle:
    def __init__(self, x, y, r):
        self.x = float(x)
        self.y = float(y)
        self.r = float(r)

# 1. 计算两点距离（圆心距）
def get_dist(c1, c2):
    return math.sqrt((c1.x - c2.x)**2 + (c1.y - c2.y)**2)
# 2. 核心判定函数
def check_relationship(c1, c2):
    d = get_dist(c1, c2)
    s = c1.r + c2.r          # 半径和
    diff = abs(c1.r - c2.r)  # 半径差
    # 注意判断顺序：先判相等（切），再判不等（离/含/交）
    if d > s + EPS:
        return "相离 (Apart)"
    if abs(d - s) < EPS:
        return "外切 (Tangential-Outer)"
    if abs(d - diff) < EPS:
        return "内切 (Tangential-Inner)"
    if d < diff - EPS:
        return "内含 (Contained)"
    # 如果以上都不是，且 d 在 diff 和 s 之间
    return "相交 (Intersecting)"
```
