# 题目描述
在 $1$ 到 $2021$ 之间，找出有多少个数与 $2021$ 的最大公约数 $\gcd(x, 2021) > 1$
# 分析
- **难度等级**：**入门**（蓝桥杯省赛高频必备题）。
- **评价**：这属于“送分题”，但陷阱在于如何快速、准确地处理较大的数字（如 2021）。

这道题有两种思考方式：
1. **正向枚举**：从 $1$ 到 $2021$ 挨个查 $\gcd(i, 2021)$，大于 $1$ 就计数。
2. **反向剔除（更高级）**：总数 $2021$ 减去“与 $2021$ 互质（$\gcd=1$）”的个数。
# 代码
```python
import math
ans = 0
for i in range(1, 2022):
    if math.gcd(i, 2021) > 1:
        ans += 1
print(ans) 
```
