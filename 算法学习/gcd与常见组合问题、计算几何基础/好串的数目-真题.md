# 题目
**题目大意：**

给你一个数字字符串（0-9），我们要数一数里面有多少个“好串”。

- **什么是“好串”？**
    
    1. 长度为 1 的字符串都是好串。
        
    2. 长度大于 1 的，如果能切成**两段**连续的“阶梯串”（连续非递减子串），它就是好串。
        
- **什么是“阶梯串”？**
    
    如果一个串里的每个数字，要么等于前一个数字，要么等于前一个数字加 1（比如 `12233` 或 `456`），它就是一个阶梯串。
    

**约束条件：**

- 字符串长度 $n$ 最大可达 $10^5$（对应评测用例最后几档）。
    
- 时间限制通常为 1 秒，这意味着我们需要 $O(n)$ 或 $O(n \log n)$ 的解法，暴力枚举子串（$O(n^2)$）在 $n=10^5$ 时会超时。
# 分析过程

### Step 1: 痛点与定位

- **痛点**：如果暴力枚举每一个子串 $[l, r]$ 再去判断是否能拆分，复杂度是 $O(n^3)$ 或 $O(n^2)$，这在蓝桥杯的高分档必死无疑。
    
- **题型定位**：**线性扫描 + 动态规划/预处理**。我们需要快速知道：从某个位置 $l$ 开始，最远能延伸到哪里，使得这个区间是一个“好串”。
    

---

### Step 2: 通用解法模板

对于这类“寻找满足条件的子串数量”且条件具有**单调性**或**可拼接性**的问题，通用模板是：

1. **预处理**：计算出每个位置 $i$ 为起点，最长的“阶梯串”能到哪里。
    
2. **贪心拼接**：对于每个起点 $l$，它的最长“好串”终点由“第一段阶梯”加上“紧接着的第二段阶梯”组成。
    

---

### Step 3: 大白话拆解

想象你在爬楼梯。这道题定义的“阶梯串”就是一种**平稳且坡度极小**的楼梯（要么平着走，要么只上一级）。

1. **第一段楼梯**：从你站的位置 $l$ 开始往后走，只要符合“相等或加 1”，你就一直走，直到走不动为止。假设终点在 $i$。
    
2. **第二段楼梯**：既然“好串”允许由两段阶梯拼接，那么从第一段断开的地方（$i+1$）开始，你再按同样的规则走一段新的阶梯。假设这段新的阶梯终点在 $j$。
    
3. **结论**：
    
    - 长度为 1 的串（你自己站的那格）是好串。
        
    - 任何从 $l$ 开始，到 $l+1$ 之间，一直到 $j$ 结束的子串，都可以拆成两段阶梯（一段是 $s[l...i]$ 的某部分，一段是 $s[i+1...j]$ 的某部分）。
        
    - 所以，以 $l$ 为起点的“好串”总数，就是从 $l$ 到 $j$ 的距离：$j - l + 1$。
        

**为什么这样是对的？**

因为如果 $s[l...r]$ 是好串，它必须能拆成两段。为了让 $r$ 尽可能远，我们要让第一段阶梯尽可能长，这样第二段才能从更远的地方接力。

# 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
s = next(it)
n = len(s)
f = [0]*(n)
f[n-1]=n-1
for i in range(n-2,-1,-1):
  curr = int(s[i])
  nxt = int(s[i+1])
  if nxt == curr or nxt == curr + 1:
    f[i]=f[i+1]
  else:
    f[i]=i
ans = 0 
for l in range(n):
  i = f[l]
  if i< n-1:
    j = f[i+1]
    ans+=(j-l+1)
  else:
    ans+=(n-l)
print(ans)
```
