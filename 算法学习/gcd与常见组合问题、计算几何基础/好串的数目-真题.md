# 题目
给你一个数字字符串（0-9），我们要数一数里面有多少个“好串”。
- **什么是“好串”？**
    1. 长度为 1 的字符串都是好串。
    2. 长度大于 1 的，如果能切成**两段**连续的“阶梯串”（连续非递减子串），它就是好串。
- **什么是“阶梯串”？**  
    如果一个串里的每个数字，要么等于前一个数字，要么等于前一个数字加 1（比如 `12233` 或 `456`），它就是一个阶梯串。

**约束条件：**
- 字符串长度 $n$ 最大可达 $10^5$（对应评测用例最后几档）。
- 时间限制通常为 1 秒，这意味着我们需要 $O(n)$ 或 $O(n \log n)$ 的解法，暴力枚举子串（$O(n^2)$）在 $n=10^5$ 时会超时。

# 分析

## 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
s = next(it)
n = len(s)
f = [0]*(n)
f[n-1]=n-1
# `f[i]` 代表从第 `i` 个字符开始，**最长的一段**符合规则的楼梯在哪里结束
for i in range(n-2,-1,-1):
  curr = int(s[i])
  nxt = int(s[i+1])
  if nxt == curr or nxt == curr + 1:
    f[i]=f[i+1]
  else:
    f[i]=i

ans = 0 
for l in range(n):
  i = f[l] #第一段
  if i< n-1:
    j = f[i+1] # 第二段
    ans+=(j-l+1) #计算长度
  else:
    ans+=(n-l) # 只有一段 直接到n
print(ans)
```
