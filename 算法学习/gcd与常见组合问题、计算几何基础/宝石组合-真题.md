# 题目

小蓝在一个树洞里发现了 $N$ 枚宝石，每枚宝石都有一个“闪亮度”属性 $H_i$。小蓝需要从这 $N$ 枚中挑选出 **3 枚** 宝石（假设它们的闪亮度分别是 $H_a, H_b, H_c$），组成一个组合。

这个组合的“精美程度”由一个看起来非常复杂的公式 $S$ 决定：

$$S = H_a H_b H_c \cdot \frac{LCM(H_a, H_b, H_c)}{LCM(H_a, H_b) \cdot LCM(H_a, H_c) \cdot LCM(H_b, H_c)}$$

**你的任务是：**

1. 找到一种挑选方案，使得 **$S$ 的值最大**。
    
2. 如果有多种方案都能达到这个最大值，挑选出 $H$ 值从小到大排序后，**字典序最小**的那一组。
    
3. **输入**：宝石数量 $N$ 和每个宝石的亮度 $H_i$。
    
4. **输出**：选出的那三个宝石的亮度值。
# 代码

```python
import os
import sys

# 请在此输入您的代码
it = iter(sys.stdin.read().split())
max_h=0
limit=10**5+1
cnt = [0]*(limit)
n =int(next(it))

for _ in range(n):
  h = int(next(it))
  cnt[h]+=1 # 记录亮度有多少颗
  if h>max_h:
    max_h=h # max_h 亮度最高的宝石

for g in range(max_h,0,-1): #最大公约数不能超过本身，那么从最大的数开始速度更快
  res=[]
  for m_g in range(g,max_h+1,g): # 只看g的倍数
    if(cnt[m_g])>0:
      take=min(cnt[m_g],3-len(res)) # 完美处理了 现在还差几颗
      for _ in range(take):
        res.append(m_g)
      if len(res)==3:
        print(*res)
        sys.exit()
```
