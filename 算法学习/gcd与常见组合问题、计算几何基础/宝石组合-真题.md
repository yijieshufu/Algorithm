
# 题目
小蓝在一个树洞里发现了 $N$ 枚宝石，每枚宝石都有一个“闪亮度”属性 $H_i$。小蓝需要从这 $N$ 枚中挑选出 **3 枚** 宝石（假设它们的闪亮度分别是 $H_a, H_b, H_c$），组成一个组合。
这个组合的“精美程度”由一个看起来非常复杂的公式 $S$ 决定：

$$
S = H_a H_b H_c \cdot \frac{LCM(H_a, H_b, H_c)}{LCM(H_a, H_b) \cdot LCM(H_a, H_c) \cdot LCM(H_b, H_c)}
$$

**你的任务是：**
1. 找到一种挑选方案，使得 **$S$ 的值最大**。
2. 如果有多种方案都能达到这个最大值，挑选出 $H$ 值从小到大排序后，**字典序最小**的那一组。
3. **输入**：宝石数量 $N$ 和每个宝石的亮度 $H_i$。
4. **输出**：选出的那三个宝石的亮度值。
# 分析过程
- **题型**：**数论（最小公倍数与最大公约数）+ 贪心构造**。
- **难度**：**中等偏难**。
- **频率**：**极高**（蓝桥杯非常喜欢在 $LCM$ 和 $GCD$ 上做文章）。
### 推导公式过程
**1. 预设：**
任取一质数 $p$，设 $H_a, H_b, H_c$ 中 $p$ 的指数分别为 $\alpha, \beta, \gamma$。
不妨设排好序为：$\alpha \ge \beta \ge \gamma$。
**2. 指数化简（关键步骤）：**
根据公式 $S = H_a H_b H_c \cdot \frac{LCM(H_a, H_b, H_c)}{LCM(H_a, H_b) \cdot LCM(H_a, H_c) \cdot LCM(H_b, H_c)}$，对应的指数运算为：
- **分子指数总和**：
    $(\alpha + \beta + \gamma) + \max(\alpha, \beta, \gamma) = \alpha + \beta + \gamma + \alpha$
- **分母指数总和**：
    $\max(\alpha, \beta) + \max(\alpha, \gamma) + \max(\beta, \gamma) = \alpha + \alpha + \beta$
- **最终指数**：
    $(\text{分子}) - (\text{分母}) = (2\alpha + \beta + \gamma) - (2\alpha + \beta) = \gamma$

**3. 结论：**
由于 $\gamma$ 是 $\alpha, \beta, \gamma$ 中的**最小值** ($\min$)，而所有质因子的最小值乘积即为 $GCD$。

$$
S = GCD(H_a, H_b, H_c)
$$

# 代码

```python
import os
import sys

# 请在此输入您的代码
it = iter(sys.stdin.read().split())
max_h=0
limit=10**5+1
cnt = [0]*(limit)
n =int(next(it))

for _ in range(n):
  h = int(next(it))
  cnt[h]+=1 # 记录亮度有多少颗
  if h>max_h:
    max_h=h # max_h 亮度最高的宝石

for g in range(max_h,0,-1): #最大公约数不能超过本身，那么从最大的数开始速度更快
  res=[]
  for m_g in range(g,max_h+1,g): # 只看g的倍数
    if(cnt[m_g])>0:
      take=min(cnt[m_g],3-len(res)) # 完美处理了 现在还差几颗
      for _ in range(take):
        res.append(m_g)
      if len(res)==3:
	        print(*res)
        sys.exit()
```
