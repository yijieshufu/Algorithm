# 判断质数
## 试除法
### 是什么？ 
拿一堆比 $n$ 小的数字，一个一个去试着除 $n$，看能不能整除

```python
import math
def is_prime(n):
    # 1. 基础检查
    if n < 2: return False
    if n == 2: return True
    if n % 2 == 0: return False # 剔除偶数
    # 2. 确定搜索范围（只需到根号n）
    limit = int(math.isqrt(n))
    # 3. 只测试奇数作为因数
    for d in range(3, limit + 1, 2):
        if n % d == 0:
            return False
    return True
```

## 埃氏筛
### 1. 它解决了什么痛点？
如果我们要找 $1$ 到 $n$ 之间所有的质数（素数），最朴素的办法是：对每个数字都进行一次“质数判定”（看它能不能被 $2$ 到 $\sqrt{n}$ 之间的数整除）。
### 场景理解
埃氏筛的思路非常巧妙，它不是“寻找质数”，而是 **“排除合数”**。
- **生活类比：** 想象你在一个班级里，所有人初始都贴着“好人”标签。
    1. 从 2 号同学开始，2 号是好人，但他所有的**倍数**（4, 6, 8...）都被判定为“坏人”。
    2. 找到下一个还没被贴“坏人”标签的（比如 3 号），3 号是好人，但他所有的**倍数**（6, 9, 12...）也要贴上“坏人”标签。
    3. 以此类推，最后剩下没被贴“坏人”标签的，全是好人（质数）。
```python
prime = []
is_prime = [True] * N
def Era(n):
    is_prime[0] = is_prime[1] = False
    for i in range(2, n + 1):
        if is_prime[i]:
            prime.append(i)
            if i * i > n: # 优化到 √n
                continue
            for j in range(i * i, n + 1, i):
                is_prime[j] = False
```

# 组合数
当你想从 $n$ 个不同的苹果里挑出 $k$ 个带走，但**不在乎拿的先后顺序**时，组合数 $C(n, k)$ 就能直接告诉你一共有多少种拿法。  
**核心思路：**
- **计算公式**：

    $$


C(n, k) = \frac{n!}{k!(n-k)!}

$$
    其中 $n!$ 是阶乘，表示把 $n$ 个数全部排成一排的方案数。
- **（递推法）**： ==只能处理n,m比较小的数==
	- 它的逻辑是：
		- 对于第 $n$ 个苹果，你要么**选它**（剩下的要在 $n-1$ 个里选 $k-1$ 个）
		- 要么**不选它**（剩下的要在 $n-1$ 个里选 $k$ 个）。
    - 公式：$C(n, k) = C(n-1, k-1) + C(n-1, k)$。
```python
def init_C(n): 
    C[0][0] = 1 # 0个里面选0个只有一种方法
    for i in range(1, n + 1):
        C[i][0] = 1 #i个 选0个也是一种
        for j in range(1, i + 1):
            C[i][j] = C[i - 1][j - 1] + C[i - 1][j]
    return C
```

# 快速幂
**快速幂（Fast Exponentiation）** 是一种高效计算底数的 $n$ 次幂的算法。

python自带快速幂： pow(a,n,MOD)
```python
def quick_pow(a, n):
    ans = 1
    while n > 0:
        if n & 1:  # 如果该二进制位存在
            ans = ans * a % MOD
        a = a * a % MOD
        n >>= 1  # n除以2,判断下一个二进制位
    return ans
```
# 整数唯一分解定理
## 1. 是什么？（定义）

任何一个大于 1 的整数 $n$，都可以**唯一地分解成若干个质数**的乘积。如果不考虑这些质数的**排列顺序**，这种分解形式是**唯一**的。
数学表达式为：
$$

n = p_1^{a_1} \cdot p_2^{a_2} \cdot \dots \cdot p_k^{a_k}

$$
其中 $p_1 < p_2 < \dots < p_k$ 是质数，$a_i$ 是正整数（即该质数出现的**次数/指数**）。
## 2. 有什么用？（核心应用）
### ① 求约数个数

如果 $n = p_1^{a_1} p_2^{a_2} \dots p_k^{a_k}$，那么 $n$ 的**正**约数个数为：
$$

(a_1 + 1)(a_2 + 1) \dots (a_k + 1)

$$
_原理：每个质因子 $p_i$ 可以选 $0$ 到 $a_i$ 个，共有 $a_i+1$ 种选法。_
### ② 求约数之和
$n$ 的所有**正**约数之和为：
$$

\prod_{i=1}^{k} (1 + p_i + p_i^2 + \dots + p_i^{a_i}) = \prod_{i=1}^{k} \frac{p_i^{a_i+1} - 1}{p_i - 1}

$$
### ③ 最大公约数 (GCD) 与 最小公倍数 (LCM)

假设 $a = p_1^{x_1} p_2^{x_2} \dots$，$b = p_1^{y_1} p_2^{y_2} \dots$

- $\text{gcd}(a, b) = p_1^{\min(x_1, y_1)} \cdot p_2^{\min(x_2, y_2)} \dots$
- $\text{lcm}(a, b) = p_1^{\max(x_1, y_1)} \cdot p_2^{\max(x_2, y_2)} \dots$
### ④ 判断完全平方数

如果所有指数 $a_i$ 都是**偶数**，那么这个数就是一个完全平方数。
