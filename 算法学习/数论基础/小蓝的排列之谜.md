## 1. 题目大意与痛点

- **数组内容：** $1$ 到 $n$ 的全排列。
    
- **恒等位置：** 如果第 $i$ 个位置上的数就是 $i$，那它就是“乖孩子”。
    
- **几乎恒等：** 至少有 $n-k$ 个乖孩子。
    
    - 这意味着，“调皮的孩子”（不在原位的）数量可以为 $0, 1, 2, \dots, k$。
        
- **限制条件：** $k \le 4$ 是解题的关键突破口。

## 题目分析

### 2. 核心思路：分两步走

我们要算“刚好有 $i$ 个人乱坐”的情况（$i$ 可以是 $0, 1, 2, \dots, k$）。

#### 第一步：选出“调皮鬼” —— $C(n, i)$

从 $n$ 个学生里，挑出 $i$ 个准备乱坐的学生。

- 比如 $n=5, i=2$，你可以挑（学生1，学生2），也可以挑（学生4，学生5）。
    
- 选法有多少种？就是组合数 $C(n, i)$。
    

#### 第二步：让“调皮鬼”彻底坐错 —— $D(i)$

这一步最关键。这 $i$ 个被选出来的学生，**必须全部坐错位子**。

- **$D(0)=1$：** 0个人乱坐。大家都坐对了，只有 **1** 种情况。
    
- **$D(1)=0$：** 1个人乱坐。这不可能！如果只有他一个人乱坐，他能坐哪？只要他坐了别人的位子，那个人也就乱坐了。所以至少得 2 个人才能乱坐。
    
- **$D(2)=1$：** 2个人乱坐（A和B）。唯一的办法是：A 坐 B 的位子，B 坐 A 的位子。只有 **1** 种情况。
    
- **$D(3)=2$：** 3个人乱坐（A、B、C）。
    
    - 情况1：A->B, B->C, C->A
        
    - 情况2：A->C, C->B, B->A
        
    - 只有这 **2** 种。


由于这道题 $k$ 最大只有 4，我们直接手算出 $D(0)$ 到 $D(4)$ 即可：

- $D(0) = 1$
    
- $D(1) = 0$
    
- $D(2) = 1$
    
- $D(3) = 2$
    
- $D(4) = 9$

## 代码

```python
import os
import sys
# 请在此输入您的代码
it = iter(sys.stdin.read().split())
n=int(next(it))
k=int(next(it))
MAX_N = 1000 + 1 # 题目 n 最大 15 
c = [[0] * MAX_N for _ in range(MAX_N)]
for i in range(MAX_N):
        c[i][0] = 1 # 任何数选 0 个，都只有 1 种取法（空取）
        for j in range(1, i + 1):
            c[i][j] = c[i-1][j] + c[i-1][j-1]
d=[1,0,1,2,9]
res=0
for i in range(k+1):
  res+=c[n][i]*d[i]
print(res)
```
