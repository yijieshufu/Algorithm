# 题目
如果一个正整数只能被 1 和它本身整除，就称为一个**质数**。

最小的几个质数依次是 $2, 3, 5, 7, 11, 13, \dots$

**请问，第 2025 个质数是多少？**

## 代码

```python
import os
import sys
limit=100000
is_prime=[True]*(limit+1)
is_prime[1]=False
prime=[]
target=2025
for i in range(2,limit+1):
  if is_prime[i]:
    prime.append(i)
    if len(prime) == target:
      print(prime[-1])
    for j in range(i*i,limit+1,i):
      is_prime[j]=False
```
