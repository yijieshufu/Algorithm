# 题目描述

- **给定资源**：你手头只有三个数字可用：`0`, `1`, `2`（可以重复使用）。
- **任务**：组装一个长度为 $n$ 的正整数。
- **必须满足的规矩**：
    1. **它得是奇数**：这意味着最后一个数字必须是 `1`（因为 `0` 和 `2` 都是偶数）。
    2. **它不能有前导 0**：这意味着第一个数字不能是 `0`（除非 $n=1$，但通常多位数首位不为0）。
- **结果处理**：因为结果可能非常大，最后要对 `1000000007` 取余数。
# 分析思路

### 核心思路是什么？
我们可以把这个 $n$ 位数想象成一列有 $n$ 个车厢的火车，我们要往每个车厢里放数字：
1. **最后一节车厢（个位数）**：  
    为了让整个数是“奇数”，这节车厢**只能**放数字 `1`。
    - **选择权：1 种**。
2. **第一节车厢（最高位）**：  
    为了让它合法（不以 `0` 开头），这节车厢**不能**放 `0`。所以只能放 `1` 或 `2`。
    - **选择权：2 种**。
3. **中间的车厢（如果有的话）**：  
    中间的每一节车厢都非常自由，`0`, `1`, `2` 随便放。
    - **选择权：每节车厢都有 3 种**。
# 代码
```python
import os
import sys
it = iter(sys.stdin.read().split())
n=int(next(it))
MOD=1000000007
if n==1:
  print(1)
else:
  print(2*pow(3,n-2,MOD)%MOD)
```
