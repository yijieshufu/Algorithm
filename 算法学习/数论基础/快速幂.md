# 题目
给定三个正整数 $b, p, k$，计算 $b$ 的 $p$ 次方除以 $k$ 的余数，即求 $b^p \pmod k$。

## 代码

```python
import os
import sys
# 请在此输入您的代码
it = iter(sys.stdin.read().split())
b = int(next(it))
p = int(next(it))
k = int(next(it))
def quick_pow(b,p,k):
  res = 1
  while p > 0: 
    if p & 1:
      res = res * b % k
    b = b * b % k
    p >>= 1
  return res
print(quick_pow(b,p,k))
```
