## 1. 题目大意与痛点

- **数组内容：** $1$ 到 $n$ 的全排列。
- **恒等位置：** 如果第 $i$ 个位置上的数就是 $i$，那它就是“乖孩子”。
- **几乎恒等：** 至少有 $n-k$ 个乖孩子。
    - 这意味着，“调皮的孩子”（不在原位的）数量可以为 $0, 1, 2, \dots, k$。
- **限制条件：** $k \le 4$ 是解题的关键突破口。
## 题目分析
由于这道题 $k$ 最大只有 4，我们直接手算出 $D(0)$ 到 $D(4)$ 即可：
- $D(0) = 1$
- $D(1) = 0$
- $D(2) = 1$
- $D(3) = 2$
- $D(4) = 9$
**错排公式 $D(n)$ 的递推式**：
- $D(1) = 0$（一个人不可能拿错自己的钥匙）
- $D(2) = 1$（两个人只能互换钥匙才能都拿错）
- $D(n) = (n-1) \times [D(n-1) + D(n-2)]$
## 代码

```python
import os
import sys
# 请在此输入您的代码
it = iter(sys.stdin.read().split())
n=int(next(it))
k=int(next(it))
MAX_N = 1000 + 1 # 题目 n 最大 15 
c = [[0] * MAX_N for _ in range(MAX_N)]
for i in range(MAX_N):
        c[i][0] = 1 # 任何数选 0 个，都只有 1 种取法（空取）
        for j in range(1, i + 1):
            c[i][j] = c[i-1][j] + c[i-1][j-1]
d=[1,0,1,2,9]
res=0
for i in range(k+1):
  res+=c[n][i]*d[i]
print(res)
```
