# 题目描述
- **目标**：寻找质数序列中的第 **100,002** 个质数（十万零二）。

- **已知条件**：第 1 个质数是 2，第 2 个质数是 3，以此类推。
    
- **题型**：填空题（算出结果后直接 `print` 即可）。


# 代码

```python
import os
import sys
limit = 2000000
N = 100002
is_prime = [True] * (limit + 1)
is_prime[1] = False # 1 不是质数
primes = []
for i in range(2,limit+1):
  if is_prime[i]:
    primes.append(i)
    # 如果已经找到了足够数量的质数，就可以停下了
    if len(primes) == N:
        print(primes[-1])
        break
    for j in range (i*i,limit+1,i):
      is_prime[j]=False
```
