# 题目
计算**组合数** $C_n^m$。

# 分析
### 1. 它解决了什么痛点？

在生活中，我们经常遇到这种问题：我有 10 个不同的苹果，想分给你 3 个，一共有多少种拿法？

如果手动一个一个数，非常容易数错。这个算法就是为了快速算出“**从 $n$ 个不同的东西里，不看顺序地挑出 $m$ 个，一共有多少种可能性**”。

### 2. 核心思路是什么？

我们可以用**杨辉三角**（Pascal's Triangle）的逻辑来理解。

想象你在排队领礼品：

- **情景 A**：你一定要领。那么剩下的 $n-1$ 个人里只需要再挑 $m-1$ 个领就行了。
    
- **情景 B**：你坚决不领。那么剩下的 $n-1$ 个人里得挑满 $m$ 个才行。
    
    所以：$C_n^m = C_{n-1}^{m-1} + C_{n-1}^m$。

## 代码

```python
import os
import sys
# 请在此输入您的代码
it = iter(sys.stdin.read().split())
t = int(next(it))
MAX_N = 15 + 1 # 题目 n 最大 15 
c = [[0] * MAX_N for _ in range(MAX_N)]
for i in range(MAX_N):
        c[i][0] = 1 # 任何数选 0 个，都只有 1 种取法（空取）
        for j in range(1, i + 1):
            c[i][j] = c[i-1][j] + c[i-1][j-1]
for _ in range(t):
  n = int(next(it))
  m = int(next(it))
  print(c[n][m])
```
