# 题目
### 1. 核心条件

- **输入**：一个长度为 $n$ 的数组 $a$，以及变换次数 $m$。
    
- **变换规则**：对于数组中的每个元素 $a_i$，将其更新为 $a_i \times \text{bitCount}(a_i)$。
    
- **关键函数**：$\text{bitCount}(x)$ 代表 $x$ 在二进制下“1”的个数。
    
- **数据规模**：$n \le 1000$，$m \le 5$，$a_i \le 1000$。这意味着即便经过 5 次变换，数值也不会膨胀到产生溢出的程度，Python 可以轻松处理。


```python
import os
import sys
# 请在此输入您的代码
it =iter(sys.stdin.read().split())
n=int(next(it))
b=[0]+[int(next(it)) for i in range(n)]
m=int(next(it))
for _ in range(m):
  for k in range(1,n+1):
    b[k]=b[k]*bin(b[k]).count('1')
print(*(b[i] for i in range(1, n + 1)))
```
