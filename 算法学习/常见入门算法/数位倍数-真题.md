# 题目

**题目名称**：数位和统计

**核心要求**：统计在闭区间 $[1, 202504]$ 之内，有多少个整数满足“**各位数字之和是 5 的倍数**”。

**例子说明**：

- 数字 **5**：数位和为 5，是 5 的倍数（$5 \times 1$）。
    
- 数字 **19**：数位和为 $1 + 9 = 10$，是 5 的倍数（$5 \times 2$）。
    
- 数字 **8025**：数位和为 $8 + 0 + 2 + 5 = 15$，是 5 的倍数（$5 \times 3$）。

转化为字符串
```python
# 请在此输入您的代码
res =0 
for i in range (1,202505):
  a=str(i)
  m=len(a)
  b=0
  for j in range(0,m):
    b +=int(a[j])
  if b % 5==0:
    res+=1
print(res)
```
