## 题目描述：门牌制作

**【题目背景】** 小蓝要为一条街的住户制作门牌号。这条街一共有 **2020** 位住户，门牌号从 **1 到 2020** 依次编号。

**【制作规则】** 制作门牌的方法是先准备 **0 到 9** 这十个数字字符，然后根据需要将字符粘贴到门牌上。

- **举例：** 门牌号 `1017` 需要依次粘贴字符 `1`、`0`、`1`、`7`。
    
- **消耗统计：** 制作 `1017` 号门牌共需要 **1个**字符`0`，**2个**字符`1`，**1个**字符`7`。

**【问题】** 请问要制作所有的 **1 到 2020** 号门牌，总共需要多少个**字符 `2`**？

## 题型
数类模拟题 

十进制整数拆解

## 代码
转化为字符串，直接数2
```python
ans =0
for i in range(1,2021):
    s=str(i)
    ans +=s.count("2")
print(ans)
```
十进制整数拆解
```python
ans = 0
for i in range(1,2021):
    while i>0:
     if i % 10==2:
       ans+=1
     i= i // 10
print(ans)
```
