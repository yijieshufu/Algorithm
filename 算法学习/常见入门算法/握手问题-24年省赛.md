## 📘 题目详细描述

**题目背景：** 小蓝组织了一场 50 人的算法交流会。

**握手规则：**

1. **理想情况：** 每个人都要和除了自己以外的所有人握手一次。
    
2. **特殊情况：** 在这 50 人中，有 **7 个人** 比较特殊。这 7 个人**彼此之间**没有握手，但他们照常和剩下的 43 个人握手了。
    
3. **判定标准：** A 和 B 握手只算作 **1 次**（即不分先后顺序的组合）。
    

**求解目标：** 计算这场会议中实际一共进行了多少次握手？

## 题型

条件枚举
## 解题
通常采用 **“整体扣除法”**，就像计算“分苹果”时，先算出一共应该有多少个，再减去没发出去的那部分。

在一个有 $n$ 个人的聚会上，如果每个人都和别人握手，总次数就是：

$$
\frac{n \times (n - 1)}{2}
$$

import os
import sys
ans = 0 
for i in range (1,51):
  for j in range(i+1,51):
    if i <= 7 and j <= 7:

    else:
      ans+=1
print(ans)
