# 题目
## 2. 详细条件拆解

我们可以将题目中的所有限制条件归纳为以下四个维度：

### **A. 时间范围 (Range)**

- **起点：** 2000 年 1 月 1 日
    
- **终点：** 2024 年 4 月 13 日
    
- **要求：** 你需要遍历这段时间内**每一天**。

### **B. 日期格式化 (Format)**

- 每一天的日期必须严格按照 **“YYYYMMDD”** 的格式转换成一个 8 位数。
    
- **示例：** 2024 年 1 月 1 日 $\rightarrow$ `20240101`。
    
- **注意：** 月份和日期不足两位时，前面必须补 `0`。

### **C. 笔画映射规则 (Mapping)**

你需要将 8 位日期中的每一个数字，对应到它汉字的“笔画数”。题目给出了一个明确的对照表：

|**数字**|**对应汉字**|**笔画数**|
|---|---|---|
|0|零|**13**|
|1|一|1|
|2|二|2|
|3|三|3|
|4|四|5|
|5|五|4|
|6|六|4|
|7|七|2|
|8|八|2|
|9|九|2|

> **关键坑点：** 数字 `0` 对应的笔画数是 **13**，这是计算中最容易拉开差距的地方。

### **D. 行为决策逻辑 (Decision)**

对于每一天，计算 8 个数字对应的总笔画数 $S$：

- 如果 $S > 50$：当天练习**篮球**（这是我们要统计的目标）。
    
- 如果 $S \le 50$：当天练习**书法**。

---

## 3. 核心统计目标

**最终任务：** 统计在指定的 2000 年 1 月 1 日到 2024 年 4 月 13 日这段时间内，小蓝总共有多少天在**练习篮球**。

# 代码
```python
import datetime
date01 = datetime.date(2000, 1, 1) # 转化为2000-01-01的形式
date02 = datetime.date(2024, 4, 13) 
day = datetime.timedelta(days=1) # 天数加一
x = [13, 1, 2, 3, 5, 4, 4, 2, 2, 2]
c = 0
while date01 <= date02:
    date = map(int, list(str(date01).replace('-', '')))
    ans = 0
    for i in date:
        ans += x[i]
    if ans > 50:
        c += 1
    date01 += day
print(c)
```
