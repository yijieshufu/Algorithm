# 题目条件梳理

- **敌方初始状态**：血量（HP）为 $2025$。
    
- **我方三位英雄的攻击规律**：
    
    - **英雄 A**：每一回合固定输出 $5$ 点伤害。
        
    - **英雄 B**：
        
        - 奇数回合（1, 3, 5...）：输出 $15$ 点伤害。
            
        - 偶数回合（2, 4, 6...）：输出 $2$ 点伤害。
            
    - **英雄 C**：
        
        - 回合数 $\pmod 3 = 1$：输出 $2$ 点伤害。
            
        - 回合数 $\pmod 3 = 2$：输出 $10$ 点伤害。
            
        - 回合数 $\pmod 3 = 0$：输出 $7$ 点伤害。
            
- **结束条件**：当敌方血量 $\le 0$ 时，游戏结束。
    
- **求解目标**：游戏在第几回合结束？
