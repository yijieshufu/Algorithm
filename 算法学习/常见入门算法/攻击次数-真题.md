# 题目条件梳理

- **敌方初始状态**：血量（HP）为 $2025$。
    
- **我方三位英雄的攻击规律**：
    
    - **英雄 A**：每一回合固定输出 $5$ 点伤害。
        
    - **英雄 B**：
        
        - 奇数回合（1, 3, 5...）：输出 $15$ 点伤害。
            
        - 偶数回合（2, 4, 6...）：输出 $2$ 点伤害。
            
    - **英雄 C**：
        
        - 回合数 $\pmod 3 = 1$：输出 $2$ 点伤害。
            
        - 回合数 $\pmod 3 = 2$：输出 $10$ 点伤害。
            
        - 回合数 $\pmod 3 = 0$：输出 $7$ 点伤害。
            
- **结束条件**：当敌方血量 $\le 0$ 时，游戏结束。
    
- **求解目标**：游戏在第几回合结束？


# 代码

```python
import os
import sys
# 请在此输入您的代码
hp =2025
res = 1
while hp>=0:
  damage =0
  damage +=5
  if res % 2==1:
    damage+=15
  else:
    damage+=2
  if res % 3==1:
    damage+=2
  elif res % 3==2:
    damage+=10
  elif res % 3==0:
    damage+=7
  hp-=damage
  if hp <= 0:
    break
  res+=1
print(res)
```
