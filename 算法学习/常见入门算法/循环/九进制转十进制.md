要求将一个**九进制**数 $(2022)_9$ 转换成**十进制**数

n进制转化为10进制
```Python
import os
import sys
# 请在此输入您的代码
s = "2022"
m=len(s)
a = [0] * (m + 1)
for i in range(1,m+1):
  a[i]=int(s[i-1])
def n_to_10(n,a,m):
  res = 0
  for i in range(1,m+1):
    res = res*n+a[i]
  return res
print(n_to_10(9,a,m))
```

**十进制**数转换为 n 进制
