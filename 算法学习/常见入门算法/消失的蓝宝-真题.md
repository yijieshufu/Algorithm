# 题目描述：蓝宝的失踪

下一次出现时间 $N$ 满足以下条件：
1. $N + 20250412$ 能被 $20240413$ 整除。
2. $N + 20240413$ 能被 $20250412$ 整除。
现在，请你帮小蓝找出满足条件的**最小正整数 $N$**，预测蓝宝的下一次出现时间。
# 代码
```python
import os
import sys
# 请在此输入您的代码
A = 20250412
B = 20240413
def gcd(x,y):
  while y:
    x,y=y,x % y
  return x
gcd = gcd(A,B)
lcm= A*B // gcd
print(lcm-A-B)
```


枚举
N + A =kB    N  =kB -A   要得到N最小 就去找k最小 然后枚举
N +B =kA
