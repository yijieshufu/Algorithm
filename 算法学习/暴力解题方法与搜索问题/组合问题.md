# 题目
- **核心动作**：在 $1 \sim n$ 中选取 $m$ 个数，按**字典序**输出所有组合方案。
- **数据边界**：$n+(n-m) \le 25$。这意味着 $n$ 最大约为 25（当 $m=n$ 时）或更小（当 $m$ 较小时），搜索空间极小（最大组合数 $C_{25}^{12} \approx 5.2 \times 10^6$），暴力 DFS 即可秒杀。
# 分析
- **算法/数据结构**：**DFS 回溯法（组合型枚举）**。
- **为何适用**：题目要求输出所有可能的“选择方案”（即组合），且要求**字典序**。DFS 通过固定顺序搜索，天然满足字典序要求。
- **通用骨架**：
```python
def dfs(step, start):
    if step == m: # 收集结果
        return
    for i in range(start, n + 1):
        path.append(i)
        dfs(step + 1, i + 1)
        path.pop()
```
- **定级**：基础题。
- **核心难点**：理解如何通过控制 `for` 循环的起点 `start` 来避免重复（如选了 $[1, 2]$ 就不再选 $[2, 1]$）。

- **[块 1：状态定义]** 使用 `res` 列表存储当前已选的数字。`dfs(u, start)` 函数中，`u` 代表当前已选了几个数，`start` 代表当前从哪个数字开始尝试选取，确保选取的数字序列单调递增，从而保证字典序且不重不漏。
- **[块 2：核心推导与剪枝]** 每一层循环从 `start` 到 `n`。一个小优化（剪枝）：如果剩余可选的数字个数（$n - i + 1$）小于还需要选的个数（$m - u$），则直接跳过。
- **[块 3：递归终点]** 当已选个数 `u == m` 时，将 `res` 中的元素转为字符串并按空格拼接输出。

组合枚举 DFS，传个 `start` 别回头，剪枝判剩余，字典序自成。

## 代码

```python
import sys

# 输入流初始化
it = iter(sys.stdin.read().split())
n = int(next(it))
m = int(next(it))

res = [] # [块 1: 路径记录]

def dfs(u, start):
    if u == m: # [块 3: 边界到达]
        print(*(res))
        return
    
    # 剪枝优化：如果后面剩下的数不够凑齐 m 个，就没必要选了
    for i in range(start, n + 1): # [块 2: 核心选择逻辑]
        if n - i + 1 < m - u: break 
        res.append(i)
        dfs(u + 1, i + 1) # 传 i+1 保证不重复选且有序
        res.pop()

dfs(0, 1)
```
