# 题目描述
- **题型**：**组合枚举 / 状态压缩 (Bitmask)**
- **难度**：⭐⭐⭐（中等）
- **是否必掌握**：**是**。蓝桥杯非常喜欢在填空题考这种“有限空间内的合法状态计数”。
- **核心逻辑**：在 25 个格子里选 13 个放白棋（剩下 12 个放黑棋），检查在没有任何一行、一列或对角线是“同色五连”的情况下，有多少种摆法。
# 代码

```python
import os
import sys
import itertools

# 1. 预处理 获胜线
lines =[] # 胜利集合
#行
for i in range(0,25,5):
  lines.append((set(range(i,i+5))))
#列
for j in range(5):
  lines.append((set(range(j,25,5))))  # 存入的集合
#对角线
lines.append({0,6,12,18,24})
lines.append({4,8,12,16,20})
all_pos = set(range(25))
ans = 0 
# 2. 白棋 combinations
for write in itertools.combinations(range(25),13):
  w_set=set(write) # 白色集合
  b_set=all_pos-w_set # 剩下的是黑色集合
  is_draw = True # 预定平局
  for line in lines:
    if line<= w_set or line<= b_set: # 黑或者白 有个是胜利 集合
      is_draw =False
      break
  if is_draw:ans+=1 
print(ans)
```

# 笔记

反向思考，求平局 --》我先计算出胜利的情况  
	列举所有的情况 减去 胜利的 = 平局

`itertools.combinations(range(25),13) ` 从0-24中选择13个数字 返回所有 情况集合

集合的包含判定：  
	`line<= w_set or line<= b_set`
