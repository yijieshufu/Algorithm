# 题目描述
- **核心动作**：在 5×5 棋盘（25 格）中，白棋先手填满全盘（白 13 子，黑 12 子），求没有任何一方达成“五连珠”（横/竖/斜）的合法状态总数。
- **数据边界**：状态空间为 $C_{25}^{13} = 5,200,300$。需在 12 条可能的胜线上进行判定。
# 笔记
- **算法/数据结构**：**组合枚举 + 位运算/集合判定**。
- **为何适用**：总状态数约 500 万，在 Python 中通过 `itertools.combinations` 配合高效判定，可在 10 秒内跑出结果（填空题只需本地运行）。
- **通用骨架**：
```python
# 生成所有组合 -> 判定合法性 -> 累加计数
for subset in combinations(all_pos, k):
    if not any(line.issubset(subset)): # 检查胜线
        ...
```

- **定级**：基础题（结果填空）。
- **核心难点**：正确识别 **12 条胜线**（5横、5竖、2斜）以及明确**白棋 13 子、黑棋 12 子**的数量分配。

- **[块 1：胜线预处理]** 建立 12 个集合，分别代表 5 行、5 列及 2 条主副对角线。这是判定的基础依据。
- **[块 2：组合状态生成]** 利用 `itertools.combinations` 直接生成白棋（先手）占据的 13 个位置，剩余位置自动归黑棋。
- **[块 3：双向过滤判定]** 对于每种状态：
    1. 检查白棋集合是否包含任何一条胜线。
    2. 检查黑棋集合（补集）是否包含任何一条胜线。
    3. 仅当两者都不包含时，判定为平局。

## 代码

```python
import os
import sys
import itertools

# 1. 预处理 获胜线
lines =[] # 胜利集合
#行
for i in range(0,25,5):
  lines.append((set(range(i,i+5))))
#列
for j in range(5):
  lines.append((set(range(j,25,5))))  # 存入的集合
#对角线
lines.append({0,6,12,18,24})
lines.append({4,8,12,16,20})
all_pos = set(range(25))
ans = 0 
# 2. 白棋 combinations
for write in itertools.combinations(range(25),13):
  w_set=set(write) # 白色集合
  b_set=all_pos-w_set # 剩下的是黑色集合
  is_draw = True # 预定平局
  for line in lines:
    if line<= w_set or line<= b_set: # 黑或者白 有个是胜利 集合
      is_draw =False
      break
  if is_draw:ans+=1 
print(ans)
```
