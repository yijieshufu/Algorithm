# 题目

- **核心动作**：在 $n \times n$ 的网格中，从所有格子均匀随机出发，求到达终点 $(n, n)$ 的最短步数期望值。
- **移动方式**：上下左右移动（代价 1）或使用 $m$ 个双向传送门（代价 1）。
- **数据边界**：$n, m \le 2000$。格点总数 $N^2 = 4 \times 10^6$。
# 分析
- **算法/数据结构**：**单源最短路 (BFS)**。
- **为何适用**：题目所有边权（网格移动与传送门）均为 $1$，且目标是求从一个终点到全图所有点的距离。
- **通用骨架**：
```python
# 逆向思维：从终点出发跑一次 BFS，距离即为所有点的最短步数
dist = [-1] * (N*N)
queue = [target]
while queue:
    curr = queue.pop(0) # 赛场建议用头指针模拟
    for neighbor in adj[curr]:
        if dist[neighbor] == -1:
            dist[neighbor] = dist[curr] + 1
            queue.append(neighbor)
```
- **定级**：中等题。
- **核心难点**：**Python 常数优化**。在 $4 \times 10^6$ 规模下，Python 必须规避函数调用、多维数组索引和频繁的边界逻辑判断，否则会超时（TLE）。

- **[块 1：一维平铺与边界哨兵]**  
    将二维坐标 $(r, c)$ 映射为 $r \times (n+2) + c$。在 $n \times n$ 外部多包围一圈“哨兵”节点并标记为已访问。这样 BFS 时只需判断 `dist[v] == -1`，无需每次检查坐标是否越界。
- **[块 2：双向传送门建图]**  
    由于传送门是双向的，且端点可能重复，使用**邻接表存储**。只有带有传送门的节点才触发传送逻辑，减少对普通节点的无效判断。
- **[块 3：极速 BFS 循环]**  
    使用大列表 `q` 和头指针 `head` 模拟队列，这是 Python 3.7.3 中最快的队列实现方式。
- **[块 4：期望值汇总]**  
    期望 $E = \frac{\sum \text{dist}}{n \times n}$。只需累加所有有效格子的距离即可。

## 代码

```python
import sys

# --- 块 1：最高效输入与环境初始化 ---
input_data = sys.stdin.read().split()
it = iter(input_data)
n = int(next(it))
m = int(next(it))

# 一维平铺：stride 为行跨度，多加 2 列作为哨兵边界
stride = n + 2
size = stride * stride
dist = [-1] * size

# 设置哨兵边界：外围一圈初始化为 -2（代表不可访问）
for i in range(stride):
    dist[i] = -2                    # 第一行
    dist[size - stride + i] = -2    # 最后一行
    dist[i * stride] = -2           # 第一列
    dist[i * stride + n + 1] = -2   # 最后一列

# --- 块 2：传送门预处理 ---
portals = [[] for _ in range(size)]
has_p = [0] * size  # 标记位，用于加速判断 标记此地有“暗道”
for _ in range(m):
    r1, c1, r2, c2 = int(next(it)), int(next(it)), int(next(it)), int(next(it))
    u = r1 * stride + c1
    v = r2 * stride + c2
    portals[u].append(v)
    portals[v].append(u)
    has_p[u] = 1
    has_p[v] = 1

# --- 块 3：单源 BFS 过程 ---
# 从终点 (n, n) 开始反向搜索
target = n * stride + n
dist[target] = 0
q = [target]
head = 0

# 预存移动偏移量：右, 左, 下, 上
offs = [1, -1, stride, -stride]

while head < len(q):
    u = q[head]
    head += 1
    d_next = dist[u] + 1
    
    # 1. 尝试四向移动
    for o in offs:
        v = u + o
        if dist[v] == -1:  # 利用哨兵，一步判断越界和访问情况
            dist[v] = d_next
            q.append(v)
    
    # 2. 尝试传送门移动 (仅当当前点有传送门时执行)
    if has_p[u]:
        for v in portals[u]:
            if dist[v] == -1:
                dist[v] = d_next
                q.append(v)

# --- 块 4：汇总结果 ---
total = 0
for r in range(1, n + 1):
    row_start = r * stride
    for c in range(1, n + 1):
        total += dist[row_start + c]

print("{:.2f}".format(total / (n * n)))
```
