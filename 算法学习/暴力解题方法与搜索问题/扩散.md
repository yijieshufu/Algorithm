# 题目
- **核心动作**：在二维网格中，给定 4 个初始黑点，每分钟向四周扩散 1 格，求 2020 分钟后黑点的总数。
- **本质**：求 4 个以初始点为中心、曼哈顿距离 $L_1 \le 2020$ 的**正方形（旋转 45° 的菱形）并集的格点数**。
- **关键数据**：初始点 $P_1(0,0), P_2(2020,11), P_3(11,14), P_4(2000,2000)$；时间 $T=2020$。
# 分析
- **算法/数据结构**：**暴力枚举 / BFS**。
- **为何适用**：扩散规则固定且对称，格点总数在 $10^7$ 级别，Python 在 1 分钟内能跑出结果，适合填空题。
- **通用骨架**（曼哈顿扫描）：
```python
for x in range(min_x, max_x):
    for y in range(min_y, max_y):
        if any(abs(x-px) + abs(y-py) <= T for px, py in points):
            ans += 1
```
- **定级**：基础题（省赛填空题前两题水平）。
- **核心难点**：合理确定扫描的边界范围，避免漏算或过度计算导致超时。

## 代码

```python
import sys

# 初始点坐标 [块 1]
pts = [(0, 0), (2020, 11), (11, 14), (2000, 2000)]
t = 2020
ans = 0

# 确定扫描边界：min_coord - t 到 max_coord + t [块 1]
x_range = range(-2020, 2020 + 2020 + 1)
y_range = range(-2020, 2000 + 2020 + 1)

# 暴力枚举格点 [块 2 & 3]
for x in x_range:
    for y in y_range:
        # 只要满足任意一个初始点的曼哈顿距离约束即可
        for px, py in pts:
            if abs(x - px) + abs(y - py) <= t:
                ans += 1
                break # 只要变黑就停止检查其他点，优化性能

print(ans) # 赛场直接填结果
```
