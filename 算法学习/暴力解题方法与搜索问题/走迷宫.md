# 题目描述
**在一个 $N \times M$ 的网格中，1 代表路，0 代表墙。给定起点 $(x_1, y_1)$ 和终点 $(x_2, y_2)$，求从起点到终点的 **最少步数**。如果走不通，输出 -1。

- **核心动作**：在 $N \times M$ 的 0/1 矩阵中，寻找从起点到终点的**最短路径长度**。
- **数据边界**：$N, M \le 10^2$。总格子数仅 $10^4$，这在 Python 眼里就是“秒杀”级规模，完全不用担心 TLE。
# 分析
- **算法/数据结构**：**BFS（广度优先搜索）**。
- **为何适用**：在边权均为 1（或相等）的网格图中，BFS 具有“按层扩散”的特性，**第一次**到达目标点时所经过的步数必然是最短路。
- **通用骨架**：
```python
q = deque([start])
dist[start] = 0
while q:
    u = q.popleft()
    for v in neighbors(u):
        if v valid and dist[v] == -1:
            dist[v] = dist[u] + 1
            q.append(v)
```
- **定级**：**基础题**。
- **核心难点**：坐标系转换（题目通常给 1-indexed，代码需转为 0-indexed）以及四连通方向的边界处理。

- **[块 1：地图与距离初始化]**：使用 $N \times M$ 矩阵存储迷宫，另开 `dist` 矩阵记录步数（初始化为 -1 表示未访问），将起点坐标加入双端队列 `deque`。
- **[块 2：方向向量控制]**：定义 `dx = [-1, 1, 0, 0]` 和 `dy = [0, 0, 1, -1]`，通过循环一次性处理上下左右四个邻居。
- **[块 3：合法性判定与扩散]**：判定新坐标 $(nx, ny)$ 是否越界、是否为障碍物 (0)、是否已访问过。若合法，更新 `dist[nx][ny] = dist[x][y] + 1` 并入队。

## 代码

```python
import sys
from collections import deque

# 1. 极简 I/O 起手式
it = iter(sys.stdin.read().split())
n, m = int(next(it)), int(next(it))
g = [[int(next(it)) for _ in range(m)] for _ in range(n)]
# 坐标转 0-indexed [块 1]
x1, y1 = int(next(it))-1, int(next(it))-1
x2, y2 = int(next(it))-1, int(next(it))-1

# 2. 核心状态定义 [块 1]
dist = [[-1] * m for _ in range(n)]
q = deque([(x1, y1)])
dist[x1][y1] = 0
dx, dy = [-1, 1, 0, 0], [0, 0, 1, -1]
ans = -1

# 3. BFS 主循环 [块 2, 3]
while q:
    r, c = q.popleft()
    if r == x2 and c == y2:
        ans = dist[r][c]
        break
    for i in range(4):
        nr, nc = r + dx[i], c + dy[i]
        # 边界与合法性检查 [块 3]
        if 0 <= nr < n and 0 <= nc < m and g[nr][nc] == 1 and dist[nr][nc] == -1:
            dist[nr][nc] = dist[r][c] + 1
            q.append((nr, nc))

print(ans)
```
