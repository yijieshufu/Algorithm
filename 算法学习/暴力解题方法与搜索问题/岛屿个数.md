# 题目描述
**核心动作**：在 $M \times N$ 的网格中，统计“非子岛屿”的数量。子岛屿定义为被其他岛屿“环”包围的陆地。 **数据边界**：$M, N \le 50$，$T \le 10$。数据范围极小，允许 $O(T \times M \times N)$ 的多次 BFS。
# 分析
**算法/数据结构**：**双重 BFS / Flood Fill（海水 8 连通，陆地 4 连通）**。  
**为何适用**：  
	判定“子岛屿”的本质是看该岛屿是否被陆地包围。  
	若海水能从“外海”通过 8 个方向（含斜向）渗透并接触到某个岛屿，则该岛屿必不是子岛屿。  
**通用骨架**：
```python
# 扩充边界后的 Flood Fill
q = [(0, 0)]
while q:
    curr = q.pop()
    for nx, ny in neighbors(curr):
        if is_water: q.append((nx, ny)) # 扩散海水
        if is_land: search_and_mark_island(nx, ny) # 发现并标记整个岛屿
```
**定级**：中等题（省赛常考连通性变形）。  
**核心难点**：理解 **“海水 8 向连通”**。只有海水能斜着穿过陆地的缝隙，才能准确判断一个岛屿是否真的被“环”锁死在内部。  
**[块 1：地图扩边与预处理]**  
为了让“外海”连通，在原地图 $M \times N$ 外包裹一圈 '0'（变为 $(M+2) \times (N+2)$）。从 $(0,0)$ 出发的海水均被视为外海。  
**[块 2：外海扩散（8 向）]**  
使用 BFS 从 $(0,0)$ 开始扩散。如果遇到 '0'，继续扩散；如果遇到未访问过的 '1'，说明发现了一个新的“独立岛屿”。  
**[块 3：岛屿消解（4 向）]**  
一旦在【块 2】中触碰到 '1'，立即启动一个独立的 BFS/DFS，将该 '1' 及其所有**上下左右**相邻的 '1' 全部标记为已访问。这个过程计为 `ans += 1`。
