# 题目描述
题型:约束满足问题 (CSP)  
**核心考点**：回溯算法 (DFS)、剪枝优化、矩阵遍历。 **难度**：⭐⭐⭐（中等） **掌握程度**：**必须掌握**。  
**通俗转述：**  
你要在一个 $6 \times 6$ 的棋盘里填 0（白）和 1（黑）。棋盘已经给你摆好了一些棋子。  
**规则**
1. **五五开**：每行、每列的 0 和 1 数量必须都是 3 个。
2. **事不过三**：任何一行或一列，不能有连续 3 个相同的颜色（即不能出现 `000` 或 `111`）。
3. **各美其美**：每一行的排布必须是唯一的（不能有两行长得完全一样）；每一列也必须是唯一的。
