# 题目描述

- **点阵规模**：在一个 $N \times N$ 的**点阵**中（注意：是点，不是格子。$N$ 个点意味着边上有 $N-1$ 条小线段）。
- **目标图形**：寻找正方形。
- **关键点**：正方形**不仅包括“横平竖直”的，还包括“歪着”的**（斜着的）。只要四个顶点都在点阵的点上，就合法。
- **数据范围**：$N$ 最大为 $10^6$。这意味着我们不能用暴力坐标枚举，必须找到数学规律，时间复杂度要在 $O(N)$ 左右。
- **计算要求**：求总方案数，结果对 $10^9+7$ 取模。
# 代码
```python
import os
import sys
# 请在此输入您的代码
it=iter(sys.stdin.read().split())
N = int(next(it))
MOD = 10**9+7
dp=[0]* (N+1)
ans = 0
for i in range(1,N): # i代表边长为i的正方形
  dp[i]=(i*(N-i)*(N-i)) % MOD # 边长为i的正方形里面可以画i个正方形 N-i代表每条边可以取的位置  x y一样则乘
  ans  = (ans+dp[i])% MOD
print(ans)
```
