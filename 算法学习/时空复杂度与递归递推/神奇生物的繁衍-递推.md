# 题目描述

1. **成熟期**：幼年生物需要 **5 年** 才能长成成年生物。
    
2. **繁殖规律**：只有**成年**生物才能生孩子，每年 1 月 1 日生 1 只。
    
3. **生存状态**：长生不老，**永远不死**。
    
4. **初始状态**：第 0 年（2000年）只有 1 只刚出生的**幼年**生物。
    
5. **目标**：求 $n$ 年后的总数，结果对 $10^9 + 7$ 取模

## 模拟
### 模拟走势（以 $n=6$ 为例）

- **第 0 年**：1 只幼年（2000年出生）。 总数：**1**
    
- **第 1-4 年**：那只生物还没长大。 总数：**1**
    
- **第 5 年**：那只生物终于成年了！它立刻生了 1 只宝宝。 总数：1（老的）+ 1（小的）= **2**
    
- **第 6 年**：老的那只继续生 1 只，5年前生的那个还没长大。 总数：2（旧的）+ 1（新的）= **3**

## 代码

```python
import os
import sys

# 请在此输入您的代码
it =iter(sys.stdin.read().split())
n=int(next(it))
mod = 10**9+7
dp = [0]*(n+2)
for i in range(1,n+2):
  if i <= 5:
    dp[i]=1
  else:
    dp[i]=(dp[i-1]+dp[i-5])%mod
print(dp[n+1])
```
