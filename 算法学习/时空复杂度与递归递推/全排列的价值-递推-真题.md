# 题目描述
 
  ## 问题描述 
  对于一个排列 $A = (a_1, a_2, \dots, a_n)$，定义价值 $c_i$ 为 $a_1$ 至 $a_{i-1}$ 中小于 $a_i$ 的数的个数，即：

$$
 c_i = \left|\{a_j \mid j < i,\ a_j < a_i\}\right| 
$$

 定义 $A$ 的价值为： $\sum_{i=1}^n c_i$  
  给定 $n$，求 1 至 $n$ 的全排列中所有排列的价值之和。
  ## 输入格式 
  输入一行包含一个整数 $n$。
# 思路分析
根据对称性，在全部 $n!$ 个排列里，**顺序对的总数**一定等于**逆序对的总数**。  
在 $n!$ 个排列中，这对位置贡献“价值”的次数正好是 **$\frac{n!}{2}$** 次。

题目要求的是 $\sum c_i$，其实就是问：在数组中，满足 $j < i$ 的下标对共有多少组？  
这其实就是一个组合问题：从 $n$ 个位置里选 2 个出来，左边的叫 $j$，右边的叫 $i$。
- **组合公式**：$C_n^2 = \frac{n(n-1)}{2}$
    > **结论 2**：整个排列中，一共有 **$\frac{n(n-1)}{2}$** 组这样的位置对。

$$

\text{总价值} = \frac{n(n-1)}{2} \times \frac{n!}{2}

$$

# 代码

```python
import os
import sys
it =iter(sys.stdin.read().split())
n = int(next(it))
MOD = 998244353
if n < 2:
    print(0)   
f = [0] * (n + 1)
f[1]=1
for i in range(2,n+1):
  if i == 2:  # 先除去2
    f[i]=f[i-1]
  else:
    f[i]=f[i-1]*i % MOD
ans = (n * (n-1) // 2 * f[n] )%MOD
print(ans)
```
