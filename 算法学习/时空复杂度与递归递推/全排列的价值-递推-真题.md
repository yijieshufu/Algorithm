# 题目描述
### 1. 什么是“价值”？

想象有 $n$ 个人（身高分别是 $1$ 到 $n$）随机排成一队。

对于队伍里的每一个人，我们看他**前面**有多少人比他**矮**。

这个排列的“价值”，就是把每个人看到的“前面比我矮的人数”全部加起来。

- **用数学语言说**：就是统计一个排列中所有满足 $j < i$ 且 $a_j < a_i$ 的数字对 $(a_j, a_i)$ 的数量。这在算法中通常被称为**顺序对**（与逆序对相对）。
    

### 2. 题目要求做什么？

我们要把 $1$ 到 $n$ 的**所有可能排法**（即 $n!$ 种全排列）都列出来，算出每一个排法的“价值”，最后把这些价值全部加在一起。

### 3. 输出要求

由于全排列的数量和价值总和增长得非常快，结果会很大，所以最后只需要输出总和除以 **998244353** 的余数。

# 分析过程

### 第一步：换个视角（核心逻辑）

不要去想“一个排列有多少价值”，我们要想：**“任意两个数凑成一对，它们能贡献多少价值？”**

想象有两张椅子（位置 $A$ 和 位置 $B$），我们从 $1$ 到 $n$ 里随便挑两个数字（比如数字 **3** 和数字 **5**）放进去。

1. **只有两种坐法**：要么是 `(3, 5)`，要么是 `(5, 3)`。
    
2. **谁有价值？** 题目定义价值是“前面的比后面的小”。所以 `(3, 5)` 有 1 点价值，而 `(5, 3)` 价值是 0。
    
3. **概率是多少？** 因为是全排列，每种数字出现的概率都是公平的。这意味着在成千上万种排队方式里，**3 坐在 5 前面** 的次数，和 **5 坐在 3 前面** 的次数，一定是**刚好一半对一半**。
    

**结论**：任意一对数字，在所有的排列中，有一半的时间是“前小后大”（贡献价值），另一半时间是“前大后小”（不贡献）。

---

### 第二步：算一算总账（通俗解释）

我们要算的“总价值”，其实就是：**有多少对数字** $\times$ **每对数字在多少个排列中贡献了价值**。

1. **总共有多少对数字？**
    
    从 $n$ 个数字里选 2 个数字凑成一对，组合数公式是 $\binom{n}{2} = \frac{n(n-1)}{2}$。
    
    _比如 $n=3$（数字是 1, 2, 3），对子有：(1,2), (1,3), (2,3)，共 3 对。_
    
2. **每一对数字在多少个排列中“前小后大”？**
    
    总共有 $n!$ 种排列。根据刚才的“一半对一半”原理，每一对数字在其中 $\frac{n!}{2}$ 个排列里是“前小后大”的。
    
3. **总公式：**

    $$


\text{总价值} = \frac{n(n-1)}{2} \times \frac{n!}{2}

$$

# 代码

```python
import os
import sys
it =iter(sys.stdin.read().split())
n = int(next(it))
MOD = 998244353
if n < 2:
    print(0)   
f = [0] * (n + 1)
f[1]=1
for i in range(2,n+1):
  if i ==2:  # 先除去2
    f[i]=f[i-1]
  else:
    f[i]=f[i-1]*i % MOD
ans = (n * (n-1) // 2 * f[n] )%MOD
print(ans)
```
