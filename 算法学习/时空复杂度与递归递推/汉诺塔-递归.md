
# 📋 题目描述与条件汇总

**核心目标**：将 N 个圆盘从 A 杆全部移动到 C 杆。

**限制条件**：

1. **一次一盘**：每次只能移动一个圆盘。
    
2. **大不压小**：任何时候，大圆盘都不能叠在小圆盘上面。
    
3. **辅助中转**：可以利用 B 杆作为临时存放点，也可以把盘子移回 A 杆，但必须遵守前两条规则。
    
4. **编号规则**：N 个圆盘由上到下依次编号为 1 到 N（最上面最小的是 1 号，最下面最大的是 n 号）。
    

**输入要求**：

- N：总圆盘数。
    
- M：我们需要找出的第 M 步操作。
    

**输出要求**：

- 第一行：第 M 步的具体操作，格式为 `#No: a->b`（例如 `#2: A->C`）。
    
- 第二行：完成搬运的总最少步数。



# 题目分析

### 2. 核心思路：找“分身”帮我干活

想象你是一个包工头，你要把 $n$ 层塔从 A 搬到 C。你只需要下达三个指令：

1. **分身 A 出场**：你去把上面 $n-1$ 层，先给我搬到 B 杆去呆着（别问怎么搬，那是分身的事）。
    
2. **自己亲自动手**：现在 A 杆只剩最底下那块最大的盘子（第 $n$ 号），你亲自把它从 A 搬到 C。
    
3. **分身 B 出场**：你去把暂时在 B 杆的那 $n-1$ 层，全部搬到 C 杆的最上面。
    

**为什么这能成功？**

因为当你叫“分身”去搬 $n-1$ 层时，他会用同样的方法叫“小分身”去搬 $n-2$ 层……直到遇到只有 **1 层**的情况（**递归出口**），小分身直接搬走就行了。

# 代码

```python
import os
import sys

# 请在此输入您的代码
it = iter(sys.stdin.read().split())


N = int(next(it))
M = int(next(it))

cnt =0
def hanoi(n,a,b,c):
  global cnt
  if n==1:
    cnt+=1
    if cnt == M:
      print(f"#{n}: {a}->{c}")
    return
  hanoi(n-1,a,c,b)
  cnt+=1
  if cnt==M:
    print(f"#{n}: {a}->{c}")
  hanoi(n-1,b,a,c)

hanoi(N,"A","B","C")

```
