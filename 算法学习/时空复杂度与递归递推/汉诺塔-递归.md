
# 📋 题目描述与条件汇总
**核心目标**：将 N 个圆盘从 A 杆全部移动到 C 杆。  
**限制条件**：
1. **一次一盘**：每次只能移动一个圆盘。
2. **大不压小**：任何时候，大圆盘都不能叠在小圆盘上面。
3. **辅助中转**：可以利用 B 杆作为临时存放点，也可以把盘子移回 A 杆，但必须遵守前两条规则。
4. **编号规则**：N 个圆盘由上到下依次编号为 1 到 N（最上面最小的是 1 号，最下面最大的是 n 号）。

**输入要求**：
- N：总圆盘数。
- M：我们需要找出的第 M 步操作。

**输出要求**：
- 第一行：第 M 步的具体操作，格式为 `#No: a->b`（例如 `#2: A->C`）。
- 第二行：完成搬运的总最少步数。
# 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
N = int(next(it))
M = int(next(it))
cnt =0
def hanoi(n,a,b,c):
  global cnt
  if n==1: # 只有一个圆盘的时候
    cnt+=1
    if cnt == M:
      print(f"#{n}: {a}->{c}")
    return
  hanoi(n-1,a,c,b) # 第一步：把上面n-1看成整体 从 a-c-b
  cnt+=1  # 移动底部的圆盘 此时A剩下最大的圆盘 第cnt次移动
  if cnt==M:
    print(f"#{n}: {a}->{c}")
  hanoi(n-1,b,a,c) # 再把移动到b的 n-1个圆盘移动到通过a移动到c
hanoi(N,"A","B","C")
```
