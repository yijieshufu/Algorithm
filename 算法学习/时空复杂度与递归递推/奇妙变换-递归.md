## 题目简述：递归函数求值

**定义**：$f(x) = \begin{cases} 2 \times x \times f(x-6) & (x > 10) \\ x \times (x - 1) & (x \leq 10) \end{cases}$

**任务**：输入 $n$ ($1 \le n \le 10^5$)，输出 $f(n) \pmod{998244353}$。

**样例**：输入 `15` $\to$ 输出 `2160`。

递推





递归
```python
import sys
sys.setrecursionlimit(100000)
def solve(x):
	if x > 10:
		result = (2*x*solve(x-6)) % 998244353
	else:
		result = (x * (x-1)) % 998244353
	return result
n=int(input())
print(solve(n))
```
