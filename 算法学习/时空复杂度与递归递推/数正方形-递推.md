# 题目描述

- **点阵规模**：在一个 $N \times N$ 的**点阵**中（注意：是点，不是格子。$N$ 个点意味着边上有 $N-1$ 条小线段）。
- **目标图形**：寻找正方形。
- **关键点**：正方形**不仅包括“横平竖直”的，还包括“歪着”的**（斜着的）。只要四个顶点都在点阵的点上，就合法。
- **数据范围**：$N$ 最大为 $10^6$。这意味着我们不能用暴力坐标枚举，必须找到数学规律，时间复杂度要在 $O(N)$ 左右。
- **计算要求**：求总方案数，结果对 $10^9+7$ 取模。

# 分析

我们不直接数正方形，而是给每一个“歪正方形”找一个“正着的家”：
- **第一步：找“外壳”**  
    任何一个斜着放的正方形，都可以被唯一地套在一个“正着放”的大正方形（外壳）里。
- **第二步：壳里住多少人？**  
    如果外壳的边长是 $i$（即占用 $i+1$ 个点），那么这个外壳内部通过旋转，刚好能构造出 **$i$ 种** 不同的正方形（包括 1 个正的和 $i-1$ 个歪的）。
- **第三步：数“壳”有多少个？**  
    在 $N \times N$ 的大点阵中，边长为 $i$ 的正向外壳，横向有 $(N-i)$ 种摆法，纵向有 $(N-i)$ 种摆法。所以，边长为 $i$ 的外壳总数是 **$(N-i)^2$** 个。
- **第四步：求总和**  
    把所有外壳边长（$i$ 从 1 到 $N-1$）的情况累加起来：
$$
\text{Total} = \sum_{i=1}^{N-1} i \times (N-i)^2 \pmod{10^9+7}
$$

```python
import os
import sys
# 请在此输入您的代码
it=iter(sys.stdin.read().split())
N = int(next(it))
MOD = 10**9+7
dp=[0]* (N+1)
ans = 0
for i in range(1,N):
  dp[i]=(i*(N-i)*(N-i)) % MOD
  ans  = (ans+dp[i])% MOD
print(ans)
```
