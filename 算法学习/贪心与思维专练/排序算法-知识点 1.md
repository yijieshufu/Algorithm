# 三种经典排序（选择，插入，冒泡）
### 选择排序
工作原理：  
	是将排序的元素分为两堆  
		1. 排好序的元素；  
		2. 待排序的元素。
```python
def selection_sort(a, n):
    for i in range(1, n): # i是当前要放的位置
        p = i # p当前排列的位置
        for j in range(i + 1, n + 1):# j从i后面找数
            if a[j] < a[p]:# 找到一个最小的
                p = j # 记录最小的坐标
        a[i], a[p] = a[p], a[i] # 交换位置
```
### 插入排序
基本工作原理：  
	将数组分为两部分：  
		1. 排好序的部分；  
		2. 未排序的部分

```python
# 下标从0开始
def insertion_sort(a, n):
    for i in range(1, n):
        key = a[i] #从1开始拿元素
        j = i - 1 #找i前面的所有数
        while j >= 0 and a[j] > key: # 直到找到一个比他i的数
            a[j + 1] = a[j] #往后移数
            j -= 1 # 持续往前
        a[j + 1] = key # 填入
```

### 冒泡排序
每次遍历时，冒泡排序会将  
	**相邻元素两两比较**，  
	如果**前一个元素比后一个元素大**，  
	就**交换**这两个元素的位置

```python
def bubble_sort(a, n):
    for i in range(n - 1):  # 多少轮的冒泡次数
        swapped = False #交换标志位
        for j in range(n - i - 1): # 两两比较
            if a[j] > a[j + 1]:# 如果前面大于后面
                a[j], a[j + 1] = a[j + 1], a[j]#交换
                swapped = True
        if not swapped:  # 若未发生交换，提前退出
            break
```
