# 题目描述
某日，小蓝静坐书桌前，目光所及，展现着 $n$ 个数字，它们依次为 $a_1, a_2, \dots, a_n$，熠熠生辉。

小蓝悟到，如果一个数能够以若干个（**至少两个**）连续的正整数相加表示，那么它就蕴含诗意。例如，数字 6 就蕴含诗意，因为它可以表示为 $1 + 2 + 3$。而 8 则缺乏诗意，因为它无法用连续的正整数相加表示。

小蓝希望他面前的所有数字都蕴含诗意，为此，他决定从这 $n$ 个数字中删除一部分。

请问，小蓝需要删除多少个数字，才能使剩下的数字全部蕴含诗意？

# 分析
- **题目名称**：数学诗人（连续正整数之和）
- **核心考点**：数论（奇因数）、位运算（判断 2 的幂）
- **难度系数**：⭐⭐（基础必刷题）
- **重要程度**：**高频**。

**数学结论：**  
一个正整数 $a_i$ 能表示为**至少**两个连续正整数之和，**当且仅当它不是 2 的幂**（即 $a_i \neq 2^k$）。  
**为什么？**
1. **奇数**：任何大于 1 的奇数 $2n+1$ 都可以写成 $n + (n+1)$。
2. **有奇因数的偶数**：如果一个数有奇数因子（比如 6 有因子 3），我们总能以这个因子为基础构造出连续序列。
3. **2 的幂**：$1, 2, 4, 8, 16, \dots$ 这些数完全没有奇数因子（除了 1），它们无法被拆解。  
所以，题目要求删除多少个数，其实就是问：这组数里有多少个**是 2 的幂**？

**判定 2 的幂**：最快的方法是位运算。如果 $x$ 是 2 的幂，那么 `x & (x - 1)` 一定等于 `0`。
```python
import os
import sys
it = iter(sys.stdin.read().split())
n = int(next(it))
res = 0
for _ in range(n):
  a = int(next(it))
  if a>0 and (a&(a-1))==0:
    res+=1
print(res)
```
