# 题目描述
- **场景设定**：在一条一维直线上（数轴），散布着 $n$ 个显示器和 $n$ 个电源插座。
- **连接规则**：每一台显示器必须且只能连接一个插座，每个插座也只能为一个显示器供电（一一对应）。
- **目标任务**：计算并输出所有连接电源线（直线距离）的**最小消耗总长度**。
# 分析
- **题型**：**贪心算法 (Greedy)**
- **难度等级**：简单
- **高频判定**：**省赛必须掌握的基础题**。

### 通用解法模板
这种“一维距离之和最小化”的问题，核心逻辑就是 **“排好序，对对碰”**。  
**逻辑转化**：  
想象有两排人要握手，为了让大家移动的总距离最短，最左边的显示器理应连接最左边的插座。  
如果产生“交叉连接”（比如左边的显示器连了右边的插座），那么路径必然会重叠，导致总长度增加。
- 将显示器坐标 $x$ 从小到大排序。
- 将插座坐标 $y$ 从小到大排序。
- 计算每一对对应的距离并累加：$\text{ans} = \sum |x_i - y_i|$。






# 笔记

## 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
n = int(next(it))
x = [int(next(it)) for _ in range(n)]
y = [int(next(it)) for _ in range(n)]
x.sort()
y.sort()
ans = 0
for i in range(n):
  # abs() 计算一维直线距离
  ans += abs(x[i] - y[i])
print(ans)
```
