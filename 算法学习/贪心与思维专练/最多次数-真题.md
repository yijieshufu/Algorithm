# 题目描述

给出一个字符串 $s$，从中寻找由 `l`, `q`, `b` 三个字母组成的任意排列的子串（共 6 种：`lqb`, `lbq`, `qlb`, `qbl`, `blq`, `bql`）。要求切割出的单词互不重叠，求最多能切出多少个。
# 分析
**难度评分**：⭐（入门级） **考察频率**：⭐⭐⭐⭐⭐（高频必备）
# 代码
```python
import os
import sys
it = iter(sys.stdin.read().split())
s = next(it)
l = len(s)
ans = 0
i = 0
target={'l','q','b'}
while i <=l-3:
  sub = s[i:i+3]
  if set(sub) == target:
    ans+=1
    i+=3
  else:i+=1
print(ans)
```
