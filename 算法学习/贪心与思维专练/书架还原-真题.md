# 题目描述

给定 $N$ 本书的乱序排列，每次可以**交换任意两本书**，求恢复成 $1, 2, \dots, N$ 顺序的最少交换次数。

# 分析
- **难度评级**：⭐⭐（中等偏易）
- **备考建议**：**省赛高频必备题**。必须掌握“置换环”思想，它是解决“最小交换次数”类问题的银弹。
## 通用解法——置换环 模板
任何一个排列都可以看作是由多个**互不相交的环**组成的。
1. 如果一个数字已经在正确的位置（比如 1 号书在 1 号位），它自成一个长度为 1 的环，交换次数为 0。
2. 如果 $k$ 个数字形成一个环，把它们全部归位需要 $k-1$ 次交换。
3. **最终答案 = $\sum (\text{每个环的长度} - 1)$
# 代码
```python
import os
import sys
it = iter(sys.stdin.read().split())
n = int(next(it))
a = [0]+[int(next(it)) for _ in range(n)]
vis = [False]*(n+1)
ans = 0 
for i in range(1,n+1):
  if not vis[i]:
    curr= i 
    cnt = 0
    while not vis[curr]:
      vis[curr] = True
      curr = a[curr]
      cnt+=1 # 当前错误的书籍数量 环的大小
    if cnt>1:
      ans+=(cnt-1) # 环-1也就是 交换多少次恢复到正确的顺序
print(ans)
```
