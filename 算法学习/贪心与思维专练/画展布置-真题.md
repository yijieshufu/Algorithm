# 题目描述
从 $N$ 个数中选 $M$ 个，按某种顺序排列，使得相邻项平方差的绝对值之和最小
# 分析
- **难度评级**：⭐⭐（基础必刷题）
- **蓝桥考点**：排序 + 贪心 + 滑动窗口
### 通用解法模板 —— “有序序列的贡献”

当题目要求 $\sum |x_{i+1} - x_i|$ 最小化时，**核心逻辑**如下：

1. **排序**：将选出的元素按从小到大排序，这样能消除“折返”，使得路径最短。
2. **公式化简**：对于有序序列 $C_1, C_2, \dots, C_M$，其相邻差绝对值之和为：

    $$


(C_2 - C_1) + (C_3 - C_2) + \dots + (C_M - C_{M-1}) = C_M - C_1

$$
    即：**最大值 - 最小值**。

题目给出的数值 $L$ 定义为平方差绝对值之和。
	令 $C_i = B_i^2$，那么 $L = \sum |C_{i+1} - C_i|$。
	根据上面的分析，我们要让 $C_{max} - C_{min}$ 最小。

- 先对原始的艺术价值 $A$ 进行**升序排序**。
- 要在有序序列中找 $M$ 个数，使得它们的“最大平方值 - 最小平方值”最小，这 $M$ 个数一定是**连续**的。
- 利用**滑动窗口**（大小为 $M$），遍历所有可能的区间 $[i, i+M-1]$，计算 $A[i+M-1]^2 - A[i]^2$，取最小值即可。

# 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
n = int(next(it))
m = int(next(it))
a = [int(x) for x in it]
a.sort()
ans = float('inf') # 极大值
for i in range(n-m+1):
  diff = a[i+m-1]**2 - a[i]**2
  if diff< ans:
    ans = diff
print(ans)

```
