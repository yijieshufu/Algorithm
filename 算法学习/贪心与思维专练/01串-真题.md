# 题目描述
将数字 $0, 1, 2, 3 \dots$ 的二进制形式按顺序拼接在一起，形成一个**无限长**的字符串（如：$0, 1, 10, 11, 100 \dots$ 拼成 `011011100...`）。给定一个位置 $x$，求该字符串前 $x$ 位中有多少个数字 `1`。

- **题型**：**数位统计 + 二分查找** (Binary Digit Counting & Binary Search)
- **难度**：⭐⭐⭐⭐ (中等偏难)
- **蓝桥杯地位**：**高频必备题**。对于 $x \le 10^{18}$ 的规模，任何线性模拟都会超时，必须使用对数级别的算法。
# 解题思路
长度为L：一个组  
组内 1 的总数 = “首位1” 加上”后面（L-1）位 0、1的平均分配“  
首位：一共有：$2^{L-1}$个数 首位都为1 则有$2^{L-1}$个1  
剩下的L-1位：01分布均匀 则为 $(L-1) \times 2^{L-2}$

每组总长度计算：  
	每一组的总长度：$2^{L-1} \times L$  
	$L$ 位的二进制数，总是从 $2^{L-1}$ 开始，到 $2^L-1$ 结束 （数的大小范围）  
	数字个数 $\times$ 每个数的位数 = （$2^L-1$ - $2^{L-1}$ +1) * L = $2^{L-1} \times L$

# 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
X = int(next(it))
def get_len(n): # 计算 0到n 拼接成二进制的长度
  if n<0:return 0 # 边界判断 < 0 和 == 0
  res = 1
  if n==0:return 1
  l,r,bit=1,1,1 # 起点 终点 和层数
  while r<=n:
    res+=(r-l+1)*bit # 个数*层数长度 = 这层的长度  累加
    bit+=1    # 下一层 
    l = r+1   # 下一层 起点 
    r = l*2-1 # 终点
  if l <= n: # 最后一层 剩下的 长度
    res+=(n-l+1)*bit
  return res


def get_ones(n):# 0到n所有二进制中1的个数
  if n < 0: return 0 # 边界的判定
  res = 0 
  for i in range(62): # 题目条件是X<10^18 大概二进制有60 位 开个 62位 足够
    circle = 1<<(i+1) # 二进制的周期
    res+= ((n+1)//circle)*(circle//2)  # 周期数*每周期的1（55开）
    res+=max(0,(n+1)%circle-circle//2) # 剩下周期里面的1
  return res
low,hight=0,X # 二分查找 范围 0-X
N=X # 假设 数字N 为X
while low<=hight:
  mid = (low+hight)//2
  if get_len(mid)>=X:
      N = mid
      hight= mid-1
  else:
    low= mid+1
ans = get_ones(N-1)  # 计算 0-N-1 完整的1
rem = X-get_len(N-1) # 计算0到N-1的长度 
X_bin = bin(N)[2:]
for i in range(rem): # 计算出N-1长度 到x之间还有多少个1 
  if X_bin[i]=='1':
    ans+=1
print(ans)
```
