# 题目描述
寻找满足 $(A(i) - B(i)) \pmod{100} = 0$ 的 $i$ 的个数。
- $A(i) = 1 + 2 + \dots + i = \frac{i(i+1)}{2}$
- $B(i) = 1 \times 2 \times \dots \times i = i!$

# 分析
- **难度**：⭐⭐⭐（中等）
- **是否必考**：**是**。大数找规律是省赛填空题或编程前三题的常客。
## 大概思路
- **寻找“消失点”**：观察 $B(i)$（阶乘）何时会变成 $100$ 的倍数。一旦 $B(i) \equiv 0 \pmod{100}$，复杂的阶乘就消失了。
- **确定周期**：当 $B(i)$ 变为 $0$ 后，题目简化为 $A(i) \equiv 0 \pmod{100}$。因为 $A(i)$ 是关于 $i$ 的二次函数，它在模运算下一定有固定的周期。
- **分段计算**：先处理“消失点”之前的特殊情况，再计算完整周期的个数，最后补上结尾的余数。

## 具体分析
#### 1. 阶乘的“100倍数”临界点
我们要找 $B(i) = i!$ 何时能被 $100$ 整除。
- $100 = 2^2 \times 5^2$。
- 当 $i=10$ 时，$10!$ 包含因子 $5$ 和 $10$（即 $5 \times 5$），同时也包含足够的因子 $2$。
- **结论**：当 $i \ge 10$ 时，$B(i) \equiv 0 \pmod{100}$。此时条件简化为：$A(i) \equiv 0 \pmod{100}$。
#### 2. 累加和的“周期律”

$A(i) = \frac{i(i+1)}{2}$。
- $A(i) \equiv 0 \pmod{100} \implies \frac{i(i+1)}{2} = 100k \implies i(i+1) = 200k$。
- 这意味着我们需要寻找 $i(i+1)$ 能被 **$200$** 整除的 $i$。
- 周期显然是 **$200$**。在 $[1, 200]$ 范围内，满足条件的 $i$ 为：
    - **$24$**: $24 \times 25 = 600$
    - **$175$**: $175 \times 176 = 30800$
    - **$199$**: $199 \times 200 = 39800$
    - **$200$**: $200 \times 201 = 40200$
- 每个周期有 **$4$** 个满足条件的数。

#### 3. 特殊项检查 ($i < 10$)
在 $i$ 还没到 $10$ 之前，$B(i)$ 不能忽略：
- $i=1$: $A(1)=1, B(1)=1 \implies 0$ (✅)
- $i=3$: $A(3)=6, B(3)=6 \implies 0$ (✅)
- 检查其余 $i \in [1, 9]$ 均不满足。

```python
import os
import sys
n = 2024041331404202
ans = 2 # 小于10 里面有 1 和 3满足
q = n //200
r = n % 200
ans += q * 4
for x in [24, 175, 199, 200]:
  if 10 <=x<=r:
    ans+=1
print(ans)
```
