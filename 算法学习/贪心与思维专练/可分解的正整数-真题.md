# 题目描述

定义一种特殊的连续整数序列：  
	这种序列由**连续递增**的整数组成，并满足以下条件：  
	1. 序列长度至少为 3。  
	2. 序列中的数字是**连续递增的整数**（即相邻元素之差为 1），可以包括正整数、负整数或 0。  
例如，`[1, 2, 3]`、`[4, 5, 6, 7]` 和 `[-1, 0, 1]` 是符合条件的序列，而 `[1, 2]`（长度不足）和 `[1, 2, 4]`（不连续）不符合要求。 

现给定一组包含 $N$ 个正整数的数据 $A_1, A_2, \dots, A_N$。  
如果某个 $A_i$ 能够表示为符合上述条件的连续整数序列中**所有元素的和**，则称 $A_i$ 是可分解的。

请你统计这组数据中可分解的**正整数的数量**。
### 可分解数样例说明
- $A_i​$=3 是可分解的，因为 [0,1,2] 的和为 0+1+2=3。
- $A_i$​=6 是可分解的，因为 [1,2,3] 的和为 1+2+3=6。
- $A_i​$=15 是可分解的，因为 [4,5,6] 的和为 4+5+6=15。

# 代码
```python
import sys
# 1. 极简 I/O 起手式
it = iter(sys.stdin.read().split())
n = int(next(it))
ans = 0

for _ in range(n):
    a = int(next(it))
    # 2. 核心逻辑：A >= 2 必可由对称构造法分解
    if a > 1:
        ans += 1

print(ans)
```
# 分析

**连续序列和** 本质：**对称抵消**

若使用暴力枚举起点和长度，复杂度 $O(A^2)$，面对大数必超时。需通过数学推导将 $O(A)$ 降为 $O(1)$。

 算法识别信号
- 连续递增整数（公差为 1）
- 允许包含 **0 和负数**
- 序列长度限制（$\ge 3$）
- 求和等于目标值 $A$

[块1：核心构造]  
利用负数和正数对称抵消。  
序列从 $-(A-1)$ 开始，到 $(A-1)$ 结束，其和恒为 $0$。  
[块2：目标求和]  
在上述序列后再加一个项 $A$，总序列为 $[-(A-1), \dots, 0, \dots, A-1, A]$。  
此时总和 $S = 0 + A = A$。  
[块3：长度约束]  
该构造序列的长度 $L = A - (-(A-1)) + 1 = 2A$。  
题目要求 $L \ge 3$，即 $2A \ge 3 \Rightarrow A \ge 1.5$。  
[块4：特值判定]  
因为 $A$ 是正整数，故当 $A \ge 2$（即 $A > 1$）时，长度 $2A$ 至少为 $4$，满足条件。  
若 $A=1$，长度为 $2$，不满足长度 $\ge 3$。
