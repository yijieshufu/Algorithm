# 题目描述

小蓝在玩一款叫“劲舞团”的游戏，游戏规则是：按提示依次按下对应键位，人物就能随节奏跳舞。  
如果连续 K 次敲击中，**任意相邻两次敲击的时间间隔都不超过 1 秒（≤1s）**，那么这就算一次“K连击”。
现在给你一份游戏记录文件 `log.txt`，里面包含 N 条记录，每条记录有三个字段：
- 正确的敲击字符
- 小蓝实际打出的字符
- 打出该字符的毫秒级时间戳
你的任务是：**找出最长的 K 连击长度，并输出这个 K 值。**

# 代码

```python
import sys
with open("log.txt", "r") as f:
    it = iter(f.read().split())
ans = 0
cur = 0
pre_t = -2000
while True:
    try:
        target = next(it)
        user = next(it)
        t = int(next(it))
        if target == user:
            if cur == 0 or t - pre_t <= 1000:
                cur +=1
            else:
                ans = max(ans, cur)
                cur = 1
            pre_t = t
        else:
            ans = max(ans, cur)
            cur = 0
    except StopIteration:
        # 迭代器耗尽，跳出循环
        break
ans = max(ans, cur)
print(ans)
```
