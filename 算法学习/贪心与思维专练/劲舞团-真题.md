# 题目描述

如果连续 K 次敲击中，**任意相邻两次敲击的时间间隔都不超过 1 秒（≤1s）**，那么这就算一次“K连击”。  
现在给你一份游戏记录文件 `log.txt`，里面包含 N 条记录，每条记录有三个字段：
- 正确的敲击字符
- 小蓝实际打出的字符
- 打出该字符的毫秒级时间戳  
你的任务是：**找出最长的 K 连击长度，并输出这个 K 值。**
# 代码

```python
import sys

# 块 1：极简 I/O 起手式 (针对赛场标准输入)
it = iter(sys.stdin.read().split())
# 若是读取文件则换为: it = iter(open("log.txt").read().split())

ans = 0
cur = 0
pre_t = -2000 # 初始化为极小值，确保第一笔匹配时 t - pre_t > 1000

while True:
    try:
        tar = next(it)
        usr = next(it)
        t = int(next(it))
        
        # 块 2：核心推导 - 分情况讨论状态转移
        if tar == usr:
            if t - pre_t <= 1000:
                cur += 1 # [认知块：连击延续]
            else:
                cur = 1  # [认知块：对字但超时，重开连击]
            pre_t = t
            if cur > ans: ans = cur # [认知块：步步更新最大值]
        else:
            cur = 0 # [认知块：错字，连击彻底中断]
            
    except StopIteration:
        break

print(ans)
```
