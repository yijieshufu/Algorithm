# 题目描述
这道题其实可以看作一个 **“滚雪球”** 的过程，但目标是让  **“雪球滚动的总功”** 最小。
- **场景还原**：你手头有 $n$ 个数字，每次选两个数 $a$ 和 $b$ 删掉，然后换成一个新数 $c = a + b$。
- **计费规则**：每合成一次，你就要支付 $a+b$ 元。
- **最终目标**：支付的总金额最少。
# 分析
这道题 就是  哈夫曼树

# 代码


```python
import os
import sys
import heapq 
it = iter(sys.stdin.read().split())
n = int(next(it))
a = [int(x) for x in it]
heapq.heapify(a) # 堆化
res = 0 
while len(a)>1:
  s = heapq.heappop(a) + heapq.heappop(a)
  res +=s
  heapq.heappush(a,s)
print(res)
```
