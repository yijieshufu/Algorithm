# 题目描述
将 $N$ 个能量电池（能量值为 $A_i$）分成**两组**，要求每组**至少一个**电池，且两组的**能量异或和相等**。
# 分析
- **难度**：⭐（基础必备）
- **重要性**：⭐⭐⭐⭐⭐（异或性质是蓝桥杯省赛的“常客”，必须刻在 DNA 里）
- 
设第一组电池的异或和为 $X$，第二组电池的异或和为 $Y$。  
题目要求：$X = Y$。  
根据异或的性质，如果 $X = Y$，那么 $X \oplus Y = 0$。  
因为这两组电池是由原本的 $N$ 个电池**全部**分出来的（分成两组意味着没有剩余），所以：

$$
A_1 \oplus A_2 \oplus \dots \oplus A_N = X \oplus Y
$$

因此，只要**所有电池的异或总和等于 0**，就一定可以**分成相等的两组**。
# 代码

```python
import os
import sys
it = iter(sys.stdin.read().split())
t = int(next(it))
for _ in range(t):
  n = int(next(it))
  res = 0
  for _ in range(n):
    res ^= int(next(it))
  if res == 0:
    print("YES")
  else:print("NO")
```
