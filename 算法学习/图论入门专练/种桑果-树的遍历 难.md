# 题目描述
在一棵以 0 为根、拥有 $N$ 个节点的树中，每个节点初始有一个权值为 $A_i$ 的果实。随后进行 $Q$ 次操作，每次在节点 $a$ 新长出一个权值为 $S$ 的果实，并询问：在当前节点 $a$ 的**子树**中，权值小于 $S$ 的果实总数（包括初始果实和之前长出的果实）。 **数据范围**：$N, Q \le 10^4$，$A_i, S \le 10^6$。
# 分析
**分块 (Square Root Decomposition) + DFS 序**
- **为什么适合**：
    - **子树查询**：通过 DFS 序（入时 $tin$ 和出时 $tout$）将子树转换为一段连续的区间 $[tin[a], tout[a]]$。
    - **动态更新 + 范围统计**：题目涉及点更新（增加果实）和区间统计（小于 $S$ 的个数）。
    - **复杂度平衡**：$N, Q$ 为 $10^4$ 量级，Python 处理 $O(Q \sqrt{N} \log \sqrt{N})$ 的复杂度最为稳妥。分块利用 `bisect` 内置模块能极大提升执行速度，远快于手写的平衡树或复杂的动态主席树。
### 1. 核心转化：树的“降维打击” (DFS 序)
在树上直接遍历子树寻找“小于 $S$ 的值”非常低效。我们利用 **DFS 序 (Euler Tour)** 将树拍扁成数组：
- **记录时间戳**：通过一次 DFS，记录每个节点进入的时间 `tin[u]` 和离开的时间 `tout[u]`。
- **关键性质**：节点 $u$ 的所有子孙节点，其 `tin` 值一定落在连续区间 $[tin[u], tout[u]]$ 内。
- **结论**：子树统计问题完美转化为**数组区间统计问题**。
### 2. 算法选型：分块 (Square Root Decomposition)
由于题目存在**动态增长**（不断有新果实长出）和**区间查询**，我们需要平衡更新和查询的复杂度：
- **为什么不选线段树**：在 Python 3.7.3 环境下，复杂的树状数据结构常数较大，而分块算法可以配合内置的 `bisect` 模块（C 语言实现）达到极高的运行效率。
- **分块策略**：将长度为 $N$ 的 DFS 序列分成 $\sqrt{N}$ 个块（本题取 $sz \approx 120$）。
- **块内维护**：每个块维护一个**有序列表**，用于执行二分查找。
---
### 3. 操作流程详解
#### A. 预处理阶段
1. **建图与 DFS**：获取所有节点的 `tin` 和 `tout` 区间。
2. **数据重组**：按 `tin` 顺序将初始果实大小放入 `pos` 数组（一个列表的列表）。
3. **块初始化**：将每个块内的所有果实合并到一个大列表 `b[i]` 中，并执行 `sort()` 使其有序。
#### B. 动态操作 (Query & Update)
每次在节点 $a$ 长出大小为 $S$ 的新果实时：
1. **区间查询**：
    - **散块**：对区间 $[tin[a], tout[a]]$ 两端的残缺块，直接遍历 `pos` 数组统计小于 $S$ 的个数。
    - **整块**：对中间完整的块，使用 `bisect_left(b[i], S)` 快速得到统计结果（复杂度 $O(\log \sqrt{N})$）。
2. **实时更新**：
    - 将 $S$ 加入 `pos[tin[a]]`。
    - 使用 `bisect.insort(b[idx], S)` 保持所属块的有序性，为下次查询做准备。
# 代码

```python
import sys, bisect

# 增加递归深度
sys.setrecursionlimit(20000)

# 快速输入
it = iter(sys.stdin.read().split())
n, q = int(next(it)), int(next(it))
a = [int(next(it)) for _ in range(n)]

# 建图
g = [[] for _ in range(n)]
for _ in range(n - 1):
    u, v = int(next(it)), int(next(it))
    g[u].append(v); g[v].append(u)

# DFS 序
tin, tout = [0]*n, [0]*n
timer = -1
def dfs(u, p):
    global timer
    timer += 1
    tin[u] = timer
    for v in g[u]:
        if v != p: dfs(v, u)
    tout[u] = timer
dfs(0, -1)

# 分块初始化 把“树上的果实”变成了“排好队的果实”
pos = [[] for _ in range(n)]
for i in range(n): pos[tin[i]].append(a[i])

# 分块算法
sz = 120 # 块大小  题目中 $N = 10,000$，理论值应该是 $\sqrt{10,000} = 100$。
bn = (n + sz - 1) // sz # 块
b = [[] for _ in range(bn)] # 块 的容器
# 把排好队的果实（pos 里的数据）按位置扔进对应的纸箱里 
#extend 是把列表里的果实一个个取出来放进大纸箱 b
for i in range(n): b[i//sz].extend(pos[i])  
for i in range(bn): b[i].sort() # 让每个纸箱里的果实按大小排好序

# 处理操作
ans = []
for _ in range(q):
    u, s = int(next(it)), int(next(it))
    l, r = tin[u], tout[u]
    bl, br = l // sz, r // sz
    cnt = 0
    
    if bl == br: # 同块 
        for i in range(l, r + 1):
            for x in pos[i]:
                if x < s: cnt += 1
    else:#  区间横跨了多个块
        for i in range(l, (bl + 1) * sz):# 左散块
            for x in pos[i]:
                if x < s: cnt += 1
        for i in range(br * sz, r + 1): # 右散块
            for x in pos[i]:
                if x < s: cnt += 1
        for i in range(bl + 1, br): #  中间整块
            cnt += bisect.bisect_left(b[i], s)
            
    ans.append(str(cnt))
    # 更新：新增果实并维护块有序
    pos[tin[u]].append(s)
    bisect.insort(b[tin[u]//sz], s)

sys.stdout.write('\n'.join(ans) + '\n')
```
