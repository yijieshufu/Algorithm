# 题目描述
给定一棵包含 $n$ 个节点和 $n-1$ 条带权边的树。从任意点出发，需访问所有节点至少一次，求完成该任务的最短路径长度总和。 **数据范围**：$n \le 10^5$，$w_i \le 10^9$。
# 分析
**树的直径 + 贪心策略**
- **贪心**：若要访问树上所有节点并返回起点，每条边必须往返各一次，总长度为 $2 \times \sum w_i$。
- **优化**：本题不要求返回起点，因此可以少走一条从起点到终点的路径。为了使总长度最小，应让这条“少走的路径”尽可能长。
- **结论**：最短路径 = $2 \times (\text{所有边权重之和}) - (\text{树的最长路径/直径})$。

**中等题** 这是一道经典的树形结构贪心题。虽然结论不难推导，但涉及**树的直径计算**（两次 DFS 或 BFS），在 $10^5$ 的数据量下，Python 需注意递归深度限制。

### 解题思路

1. **读取输入**：构建邻接表，同时累加所有边权的和 `total_sum`。
2. **计算直径**：
    - 从任意节点（如节点 1）出发，通过 BFS/DFS 找到距离它最远的节点 $P$。
    - 从节点 $P$ 出发，再次通过 BFS/DFS 找到距离它最远的节点 $Q$。
    - $P$ 到 $Q$ 的距离即为树的直径 `max_dist`。
3. **计算答案**：`ans = 2 * total_sum - max_dist`。
4. **注意**：Python 处理 $10^5$ 的树时，建议使用**迭代 BFS** 或手动增加递归深度。
