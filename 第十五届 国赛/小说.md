## 题目 3328: 小说

```python
import sys

def solve():
    # 从标准输入读取 n
    # 对于 O(1) 的公式题，sys.stdin.read() 性能非常稳健
    line = sys.stdin.read().strip()
    if not line:
        return
    n = int(line)
    
    # 处理特殊情况和通用公式
    if n == 1:
        print(1)
    else:
        # 正确的构造公式：2n^2 - 3n + 4
        # 能够处理 n = 10^9 的超大数据量
        ans = 2 * n * n - 3 * n + 4
        print(ans)

if __name__ == "__main__":
    solve()
```
