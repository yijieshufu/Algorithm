## **试题 B: IPv6**  明天看

```python
# MOD: 模数，10^9 + 7 是算法竞赛的标准配置，防止大数溢出并满足题目要求
MOD = 10**9 + 7 

# N: 一个 16 进制段的所有可能取值数量 (0000 到 ffff)，即 2^16 = 65536
N = 65536 

# C: 一个 16 进制段中“非零”取值的数量 (1 到 ffff)，共 65535 个
C = 65535 

# sl: 计算单段在 65536 种取值下，去掉前导零后的字符总长度
# 16*1: 0-15 (0-f) 共16个数，长为1
# 240*2: 16-255 (10-ff) 共240个数，长为2
# 3840*3: 256-4095 (100-fff) 共3840个数，长为3
# 61440*4: 4096-65535 (1000-ffff) 共61440个数，长为4
sl = 16*1 + 240*2 + 3840*3 + 61440*4 

# base: 计算所有 2^128 个地址的初始总长度
# 8 * sl * pow(N, 7, MOD): 
#   - 共有8个位置（段）。
#   - 每一个位置上，sl 这组长度之和会出现 N^7 次（因为其他7个位置在任意变化）。
# 7 * pow(N, 8, MOD):
#   - 每个地址如果不压缩，固定有 7 个冒号。
#   - 总共有 N^8 个地址。
# 计算 $N$ 的 7 次方，然后对 $10^9+7$ 取余数
base = (8 * sl * pow(N, 7, MOD) + 7 * pow(N, 8, MOD)) % MOD


savings = 0

# i 从 0 到 255 遍历：代表 IPv6 八个段“是否为 0”的所有 256 种状态
for i in range(256):
    # mask: 将数字转为 8 位二进制字符串。例如 i=6 变为 '00000110'
    # '1' 表示该段数值为 0，'0' 表示该段数值非 0
    mask = bin(i)[2:].zfill(8) # `.zfill(8)`补零，使字符串总长度达到 8
    
    max_s = 0 # 记录当前这种分布状态下，使用 :: 最多能省下几个字符
    
    if i == 255: # 特殊情况：全 0 地址 (::)
        # 原本：0:0:0:0:0:0:0:0 (8个0 + 7个冒号 = 15字符)
        # 压缩后：:: (2字符)
        # 省下：15 - 2 = 13 字符
        max_s = 13
        
    elif i > 0:
        import re
        # 使用正则表达式找出所有连续的 '1'（即连续的零段）
        for m in re.finditer('1+', mask):# 找出字符串中所有连续出现的 1
            k = len(m.group()) # k 是连续 0 的段数
            start, end = m.span() # 获取这段 0 的起止位置
            
            # 压缩规则计算：
            # 1. 如果连续的 0 在两端（如 ::1 或 1::）：
            #    省下的长度是 2k - 2。
            # 2. 如果连续的 0 在中间（如 1::1）：
            #    原本是 :0:0: (5字符)，变成 :: (2字符)，省下 3 字符。
            #    公式为 2k - 1。
            s = (2*k - 2) if (start == 0 or end == 8) else (2*k - 1)
            
            # 题目规定：只能压缩一段。为了“最短”，必然选省得最多的那一段。
            max_s = max(max_s, s)
    
    # weight: 计算这种“零分布状态”在现实中对应多少个具体的 IPv6 地址
    # - 出现 '1' 的地方，数值固定为 0（只有 1 种可能）。
    # - 出现 '0' 的地方，数值可以是 1 到 65535 中的任意一个（有 C = 65535 种可能）。
    # 状态次数 = C ^ (非零段的数量)
    weight = pow(C, 8 - mask.count('1'), MOD)
    
    # 节省总长 = ∑(该状态下的最大节省量 * 该状态出现的次数)
    savings = (savings + max_s * weight) % MOD
    
# 最终答案 = (原始总长度 - 总节省长度) 取模
# 在 Python 中，(a - b) % MOD 即使结果为负数也会自动处理成正余数，非常安全
print((base - savings) % MOD)
```
