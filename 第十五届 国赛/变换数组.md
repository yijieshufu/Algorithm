## 题目 3337 -变换数组

```python
import sys

def solve():
    # 1. 快速读入：应对可能的大规模数据
    input_data = sys.stdin.read().split()
    if not input_data:
        return
    
    n = int(input_data[0])
    a = [int(x) for x in input_data[1:n+1]]
    m = int(input_data[n+1])
    
    # 2. 模拟变换
    for _ in range(m):
        # 优化点：如果数组中所有元素的 bit_count 都变成了 1，则后续变换不再改变数值
        all_stable = True 
        
        for i in range(n):
            
            bc = a[i].bit_count()
            
            if bc > 1:
                a[i] *= bc
                all_stable = False
                
        # 如果本轮没有任何元素发生变化（即全是 2 的幂或 0），提前退出循环
        if all_stable:
            break
            
    # 3. 格式化输出
    print(*(a))

if __name__ == "__main__":
    solve()
```
