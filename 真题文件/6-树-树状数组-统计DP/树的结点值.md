# 树的结点值

### 题目核心逻辑分析

1. **树的结构**：根节点为 1。输入给出了每个节点 $i+1$ 的父节点 $F_{i+1}$。
    
2. **关键约束**：$F_i < i$。这是一个非常重要的提示，意味着父节点的编号永远小于子节点。这保证了我们在处理数组时，编号较大的节点一定是编号较小节点的子孙，或者在不同的分支上。
    
3. **子树统计**：我们需要统计子树信息。通常统计子树信息可以使用 DFS（深度优先搜索），但由于本题满足 $F_i < i$，我们可以使用更高效的**逆序遍历（自底向上）** 来累加贡献。
    
4. **奇偶性判断**：
    
    - 如果节点 $i$ 是奇数，它的值 = 子树内奇数编号节点的总数。
        
    - 如果节点 $i$ 是偶数，它的值 = 子树内偶数编号节点的总数。

### 解题思路：动态规划 / 树形累加

我们可以维护两个数组：

- `odd_count[i]`：以 $i$ 为根的子树中，奇数编号结点的个数。
    
- `even_count[i]`：以 $i$ 为根的子树中，偶数编号结点的个数。

**算法步骤**：

1. **初始化**：遍历所有节点 $i$（从 1 到 $n$）。如果 $i$ 是奇数，`odd_count[i] = 1`；如果是偶数，`even_count[i] = 1`。
    
2. **逆序累加**：从 $n$ 递减到 2 遍历节点。对于每个节点 $i$，找到它的父节点 $p$：
    
    - `odd_count[p] += odd_count[i]`
        
    - `even_count[p] += even_count[i]`
        
    - （因为 $p < i$，当我们处理到 $p$ 时，它已经收集了所有子节点传递上来的信息）。
        
3. **输出结果**：根据 $i$ 的奇偶性，输出对应数组中的值。

```python
import sys  
  
  
def solve():  
    # --- 1. 快速读取数据 ---    # sys.stdin.read().split() 会一次性读取所有输入，并按空格/换行切分成列表  
    # 这在处理 2*10^5 级别的数据量时比 input() 快得多  
    input_data = sys.stdin.read().split()  
    if not input_data:  
        return  
  
    n = int(input_data[0])  # 第一行是节点总数 n  
    # --- 2. 存储树的结构 ---    # parents[i] 存储编号为 i 的节点的父节点编号  
    # 我们开 n+1 大小的空间，这样可以直接用编号 1~n 作为下标，不用减1，方便理解  
    parents = [0] * (n + 1)  
  
    # 题目说接下来的 n-1 行，第 i 行是第 i+1 个节点的父节点  
    # 遍历范围从 2 到 n，依次填充父节点信息  
    for i in range(2, n + 1):  
        parents[i] = int(input_data[i - 1]) # 得到了除根节点 每个节点父亲节点  
  
    # --- 3. 初始化统计数组 (状态定义) ---  
    # odd_c[i]: 存储以 i 为根的子树中，奇数编号节点的总数  
    # even_c[i]: 存储以 i 为根的子树中，偶数编号节点的总数  
    odd_c = [0] * (n + 1)  
    even_c = [0] * (n + 1)  
  
    # 初始状态：每个节点本身也是自己子树的一部分  
    for i in range(1, n + 1):  
        if i % 2 == 1:  
            odd_c[i] = 1  # 节点 i 是奇数，初始化奇数计数为 1        else:  
            even_c[i] = 1  # 节点 i 是偶数，初始化偶数计数为 1  
    # --- 4. 自底向上累加 (核心算法) ---  
    # 关键：题目保证父节点编号 < 子节点编号。  
    # 这意味着如果我们从大到小 (n 递减到 2) 遍历，当我们处理节点 i 时，  
    # 它的所有子孙节点一定已经先被处理过了。  
    # 这是一种“自底向上”的动态规划思想。  
  
    for i in range(n, 1, -1):  
        p = parents[i]  # 找到当前节点 i 的父亲 p        # 把子树 i 的统计结果，累加到它父亲 p 的统计结果中  
        odd_c[p] += odd_c[i]  
        even_c[p] += even_c[i]  
  
    # --- 5. 格式化输出结果 ---    results = []  
    for i in range(1, n + 1):  
        # 题目要求：节点 i 对应的输出是其子树内与其【奇偶性相同】的节点数  
        if i % 2 == 1:  
            results.append(str(odd_c[i]))  # i 是奇数，取 odd_c        else:  
            results.append(str(even_c[i]))  # i 是偶数，取 even_c  
    # 用换行符连接所有结果，一次性输出，效率更高  
    sys.stdout.write('\n'.join(results) + '\n')  # join() 有一个硬性规定：列表里的所有元素必须都是字符串（string）。  
  
  
if __name__ == "__main__":  
    solve()
```
