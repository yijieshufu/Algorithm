这是一个非常棒的问题！向量旋转公式在几何编程中非常常用，但直接看公式 $(dx, dy) \rightarrow (-dy, dx)$ 确实很抽象。

我们可以不用复杂的数学公式，而是通过**“直角三角形旋转”**的直观图像来理解它。

### 1. 直观理解：把线段看作一个直角三角形

想象一下，你在钉板上连接了两个点 $A$ 和 $B$。

假设 $A$ 是 $(0, 0)$，$B$ 是 $(3, 1)$。

那么向量 $\vec{AB}$ 就是：**向右走 3 步，向上走 1 步**。记作 $(dx, dy) = (3, 1)$。

现在，我们要把这个向量**逆时针旋转 90度**（竖起来）。

请看下图的变换逻辑：

#### 步骤解析：

1. **交换角色**：
    
    - 原来是“横着长，竖着短”（宽3，高1）。
        
    - 旋转90度竖起来后，就变成了“竖着长，横着短”（高3，宽1）。
        
    - **结论**：$dx$ 和 $dy$ 的数值必须交换。原来的 $3$ 变成了 $y$ 轴方向，原来的 $1$ 变成了 $x$ 轴方向。
        
2. **确定方向（正负号）**：
    
    - 原来的向量是向**右上**指的。
        
    - **逆时针**旋转90度后，向量变成了指向**左上**。
        
    - “左”意味着 $x$ 是负的，“上”意味着 $y$ 是正的。
        
    - 所以，新的 $x$ 变成了 $-1$，新的 $y$ 变成了 $3$。
        

**总结公式**：

从 $(3, 1)$ 变成了 $(-1, 3)$。

对应到公式就是：$(dx, dy) \rightarrow (-dy, dx)$。

### 2. 结合题目：如何找到 C 和 D？

理解了旋转向量，找正方形的另外两个点其实就是“平移”。

假设正方形四个点是 $A \to B \to C \to D$（逆时针顺序）。

已知 $A(x_1, y_1)$ 和 $B(x_2, y_2)$。

1. **算出向量 $\vec{AB}$**：
    
    - $dx = x_2 - x_1$
        
    - $dy = y_2 - y_1$
        
2. **算出垂直向量（指向正方形内部）**：
    
    - 我们需要从 $B$ 点出发，沿着垂直于 $AB$ 的方向走一段距离，就能找到 $C$。
        
    - 这个“垂直方向”就是把 $\vec{AB}$ 逆时针旋转90度。
        
    - 旋转后的向量 $\vec{v} = (-dy, dx)$。
        
3. **坐标相加**：
    
    - **点 $C$** = 点 $B$ + 向量 $\vec{v}$
        
        - $x_C = x_2 + (-dy) = x_2 - dy$
            
        - $y_C = y_2 + (dx) = y_2 + dx$
            
    - **点 $D$** = 点 $A$ + 向量 $\vec{v}$
        
        - $x_D = x_1 + (-dy) = x_1 - dy$
            
        - $y_D = y_1 + (dx) = y_1 + dx$
            

### 3. 一个具体的例子

假设 $A$ 是 $(1, 1)$，$B$ 是 $(4, 2)$。

- **向量 $\vec{AB}$**：$dx = 4-1=3, \quad dy = 2-1=1$。即 $(3, 1)$。
    
- **旋转向量**：$(-dy, dx)$ 即 $(-1, 3)$。意思是“向左1步，向上3步”。
    

现在找 $C$ 和 $D$：

- **找 $C$**（从 $B$ 出发走）：$B(4, 2) + (-1, 3) \rightarrow (4-1, 2+3) = (3, 5)$。
    
- **找 $D$**（从 $A$ 出发走）：$A(1, 1) + (-1, 3) \rightarrow (1-1, 1+3) = (0, 4)$。
    

你可以拿纸笔画一下 $(1,1), (4,2), (3,5), (0,4)$ 这四个点，你会发现它们完美围成了一个倾斜的正方形！

### 总结记忆法

- **交换**：旋转90度，长宽互换，所以 $x, y$ 互换。
    
- **变号**：
    
    - **逆时针**（向左转）：$x$ 变负，公式是 $(-y, x)$。
        
    - **顺时针**（向右转）：$y$ 变负，公式是 $(y, -x)$。
