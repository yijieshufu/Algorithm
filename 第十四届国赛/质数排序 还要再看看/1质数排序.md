
### 一、 核心思考过程分析

#### 1. 逆序对的分类讨论

题目规定：质数下标位置上的数提取后升序排列，非质数下标（1和合数）不动。

在任意一个排列中，下标对 $(i, j)$（假设 $i < j$）是否产生[[逆序对]]，取决于它们的下标类型：

- **情况 A：两个下标都是质数**
    
    由于质数位置被强制升序排序，排序后永远满足 $a_i < a_j$。因此，这类位置对**永远不会产生逆序对**，贡献为 0。
    
- **情况 B：两个下标都是非质数**
    
    由于非质数位置的数没有被移动，且数值是随机分布的，根据对称性，在 $n!$ 个排列中，$a_i > a_j$ 的概率精确为 $1/2$。
    
- **情况 C：一个质数下标，一个非质数下标**
    
    这是代码处理最精妙的地方。虽然质数位置被排序了，但非质数位置的数是随机的。对于一个特定的非质数位置 $i$，它与所有质数位置集合产生的逆序对总数，在统计学意义上依然服从对称分布。
    

#### 2. 数学期望与常数构造

代码中定义了两个关键常数 $w$ 和 $g$：

- **$g = \frac{n!}{2}$**：代表了一对“自由”位置在所有排列中产生逆序对的次数。
    
- **$w = \frac{n!}{s+1}$**（其中 $s$ 是质数总数）：这与非质数项与质数序列进行比较时的组合概率有关。当一个数与 $s$ 个已排序的数混合时，它插入的位置有 $s+1$ 种等概率情况，产生逆序对的数量遵循等差数列。
    

#### 3. 为什么倒序遍历？

代码在计算非质数与质数之间的贡献时，使用了 `A`（左侧质数个数）和 `B`（右侧质数个数）。通过从 $n$ 到 1 倒序遍历，可以动态地维护每一个非质数位置左右两侧的质数分布情况，从而一次性算出该位置对整体逆序对和的贡献。

### 二、 代码详细注解

```Python
import sys

# 使用 buffer.readline 提高大数据量下的读取效率
input = sys.stdin.buffer.readline

def main():
    # 读取输入 n，表示排列的长度
    n = int(input())
    
    # --- 1. 预处理质数表 (埃氏筛) ---
    # pr[i] = 0 表示 i 是质数；pr[i] = 1 表示非质数
    pr = [0] * (1000002) # 先假设所有的数据都是质数
    pr[1] = 1 # 1 不是质数
    p = 998244353
    
    for i in range(2, 1000001):
        if pr[i] == 0:
            # 如果 i 是质数，则将其倍数标记为非质数
            for j in range(i + i, 1000001, i):
                pr[j] = 1

    # --- 2. 统计质数和非质数的总数 ---
    s = 0 # 记录 1 到 n 之间质数的个数
    t = 0 # 记录 1 到 n 之间非质数的个数
    for i in range(1, n + 1):
        if pr[i] == 0:
            s += 1
        else:
            t += 1

    # --- 3. 计算贡献系数 w 和 g ---
    w = 1
    g = 1
    # 计算 w = n! / (s + 1) % p
    # s+1 种插入位置，每种情况的权重是 n! / (s+1)
    for i in range(1, n + 1):
        if i != s + 1:
            w = w * i % p
            
    # 计算 g = n! / 2 % p
    # 代表任意两个自由位置产生逆序对的平均基础贡献
    for i in range(1, n + 1):
        if i != 2:
            g = g * i % p

    # A 用来记录当前位置左侧的质数数量，初始化为总数 s
    A = s
    # B 用来记录当前位置右侧的质数数量
    B = 0
    ans = 0

    # --- 4. 计算非质数 vs 非质数的贡献 ---
    # 公式：C(t, 2) * (n! / 2)
    # t 个非质数位置中选 2 个，每个对有 1/2 概率是逆序对
    ans = (ans + t * (t - 1) // 2 % p * g) % p

    # --- 5. 计算非质数 vs 质数的贡献 ---
    # 倒序遍历每个下标
    for i in range(n, 0, -1):
        if pr[i] == 0:
            # 如果当前下标是质数
            A -= 1 # 左侧待处理质数减 1
            B += 1 # 右侧已处理质数加 1
        else:
            # 如果当前下标是非质数
            # A 为该位置左边的质数个数，B 为右边的质数个数
            # (A*(A+1)/2 + B*(B+1)/2) * w 
            # 这一步计算了该非质数点插入到已排序质数序列中产生的逆序对期望总和
            contribution = (A * (A + 1) // 2 + B * (B + 1) // 2) % p
            ans = (ans + contribution * w) % p

    # 输出最终结果
    print(ans % p)

if __name__ == "__main__":
    main()
```

### 三、 总结

- **空间复杂度**：$O(MAX\_N)$，主要用于存储质数标记数组。
    
- **时间复杂度**：$O(N \log \log N)$，主要耗时在埃氏筛，后续的统计和遍历均为 $O(N)$。
    
- **代码精髓**：
    
    1. **回避了显式的逆元计算**：通过在阶乘循环中 `if i != 2` 和 `if i != s+1` 跳过特定项，巧妙实现了除法效果。
        
    2. **对称性应用**：利用逆序对概率为 $1/2$ 的特性简化了非质数下标间的计算。
        
    3. **动态维护状态**：通过 `A` 和 `B` 在单次遍历中处理了复杂的跨类型位置贡献。
