```python
    # --- 1. 预处理质数表 (埃氏筛) ---
    # pr[i] = 0 表示 i 是质数；pr[i] = 1 表示非质数
    pr = [0] * (1000002)
    pr[1] = 1 # 1 不是质数

    for i in range(2, 1000001):
        if pr[i] == 0:

            for j in range(i + i, 1000001, i):
                pr[j] = 1
```

这段代码实现的是著名的**埃拉托斯特尼筛法（Sieve of Eratosthenes）**，简称“埃氏筛”。它是算法竞赛中处理质数问题最常用、最经典的方法。

你可以把它想象成一个 **“排除法”** 的过程。

---

### 1. 核心逻辑拆解

我们可以把这段代码分为三个动作：**准备、发现、剔除**。

#### **第一步：准备 (Initialization)**

```Python
pr = [0] * (1000002)
pr[1] = 1
```

- **建立名单**：创建一个长度为 1,000,002 的列表。
    
- **默认假设**：初始时 `pr[i]` 全为 `0`，代码默认认为 **“全天下除了 1 以外都是质数”**。
    
- **特判 1**：显式地将 `pr[1]` 设为 `1`（非质数），因为质数必须大于 1。

#### **第二步：发现与剔除 (Discovery & Exclusion)**

```Python
for i in range(2, 1000001):
    if pr[i] == 0:
        for j in range(i + i, 1000001, i):
            pr[j] = 1
```

- **外层循环 (`i`)**：从 2 开始往后数。
    
- **判断 (`if pr[i] == 0`)**：如果数到 `i` 时，发现它的标记还是 `0`，说明**前面没有任何数能整除它**。那么 `i` 就是一个**质数**。
    
- **内层循环 (`j`)**：既然 `i` 是质数，那么它的 **2倍、3倍、4倍...** 全部都不是质数。
    
    - `range(i + i, 1000001, i)` 的意思是：从 $2i$ 开始，每次增加 $i$，直到上限。
        
    - 把这些位置的 `pr[j]` 全部改为 `1`，表示它们被“踢出”了质数名单。

---

### 2. 形象化理解：划重点

想象一张写满 1 到 100 的表格：

1. **看 2**：它是质数，拿起笔，把 **4, 6, 8, 10...** 全部划掉。
    
2. **看 3**：它是质数，拿起笔，把 **6(已划), 9, 12(已划), 15...** 全部划掉。
    
3. **看 4**：发现已经被划掉了，直接跳过。
    
4. **看 5**：它是质数，把 **10, 15, 20, 25...** 全部划掉。
    
5. **重复**：直到所有数都看一遍。

**最后剩下的没被划掉的（标记为 0 的），就是所有的质数。**

---

### 3. 这段代码的优缺点

- **优点（快）**：
    
    它的复杂度是 $O(N \log \log N)$。对于 $10^6$ 级别的数据，它几乎在瞬间（几毫秒）就能完成。这比我们一个一个判断“这个数是不是质数”要快得多。
    
- **缺点（费空间）**：
    
    它需要开辟一个和 $N$ 同样大小的列表。在内存限制极严（比如 9MB）的情况下，如果 $N$ 达到 $10^7$，这种写法就容易内存溢出。

> **小贴士**：代码里的内层循环其实可以从 `i * i` 开始，因为像 $i \times 2, i \times 3$ 这些数，在之前处理 2 和 3 的时候已经被划掉过了。
