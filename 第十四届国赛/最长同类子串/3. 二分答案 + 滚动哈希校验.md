```python
# 3. 二分答案 + 滚动哈希校验
low, high, ans = 1, min(n, m), 0
while low <= high:
    k = (low + high) // 2
    found = False
    hashes = set()
    
    # 提取 S 的所有长度为 k 的同类串哈希
    h = 0
    for i in range(k):
        v = (i - s_lst[i]) if s_lst[i] >= 0 else 0
        h = (h * BASE + v) % MOD
    hashes.add(h)
    
    for i in range(k, n):
        j = i - k
        h = (h * BASE) % MOD # 移出 j (D[j]=0) 并整体左移
        nx = s_nxt[j]
        if nx != -1 and nx < i: # 修正 nx：从距离 j 变成距离“无”
            h = (h - (nx - j) * P[i - nx]) % MOD
        v = (i - s_lst[i]) if s_lst[i] > j else 0 # 移入 i
        h = (h + v) % MOD
        hashes.add(h)
    
    # 校验 T 中是否存在相同的哈希
    h = 0
    for i in range(k):
        v = (i - t_lst[i]) if t_lst[i] >= 0 else 0
        h = (h * BASE + v) % MOD
    if h in hashes:
        found = True
    else:
        for i in range(k, m):
            j = i - k
            h = (h * BASE) % MOD
            nx = t_nxt[j]
            if nx != -1 and nx < i:
                h = (h - (nx - j) * P[i - nx]) % MOD
            v = (i - t_lst[i]) if t_lst[i] > j else 0
            h = (h + v) % MOD
            if h in hashes:
                found = True; break
                
    if found:
        ans = k
        low = k + 1
    else:
        high = k - 1
```

### 第一步：构建“搜索框架” 🔍

我们要找的是“最大长度”。如果我们知道长度 $k$ 的同构子串存在，那么长度为 $k-1$ 的一定也存在。这种**单调性**意味着我们可以通过“猜”一个长度 $k$，然后去验证它。

这就是 **二分答案 (Binary Search on Answer)**。

我们要定义搜索的范围：

- **最小值 `low`**：通常是 1（如果完全没有相同的就返回 0）。
    
- **最大值 `high`**：由于是两个字符串的公共子串，长度不可能超过其中较短的那个。

我们可以这样写逻辑：如果长度 $k$ 成立，我们就记下这个长度，然后去试更大的（`low = k + 1`）；如果不成立，就试更小的（`high = k - 1`）。

二分搜索代码的框架：

```python
# 初始化范围
low, high = 1, min(n, m)
ans = 0

while low <= high:
    k = (low + high) // 2  # 取中间值作为尝试的长度
    
    # 我们假设有一个函数 check(k)，如果能找到长度为 k 的同构子串，返回 True
    if check(k): 
        ans = k       # 记下这个可能的答案
        low = k + 1   # 既然 k 可以，那我们就去试试更大的长度
    else:
        high = k - 1  # 如果 k 不行，说明答案肯定比 k 小
```

### 第二步 如何判断重构

对于位置 $i$ 的字符，我们不关心它具体是哪个字母，而关心它**上一次出现在哪个位置**。这个距离就是：

$$
\text{当前下标 } i - \text{上一次出现的下标 } p
$$

如果是该字符第一次出现，我们通常将其记为 **0**。
